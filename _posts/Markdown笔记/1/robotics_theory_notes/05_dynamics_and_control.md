# 05. 动力学与控制理论

## 学习目标

学完本章，你应该能：

- 区分运动学和动力学。
- 理解质量、惯量、摩擦、力矩对机器人运动的影响。
- 理解开环、闭环、PID、前馈和反馈。
- 能分析振荡、超调、稳态误差、延迟和限幅。
- 知道仿真和真机差异为什么经常来自动力学与控制。

## 1. 运动学 vs 动力学

运动学问：

```text
机器人如何运动？
```

动力学问：

```text
什么力或力矩导致机器人这样运动？
```

例子：

- 运动学：机械臂关节角变化后末端在哪里。
- 动力学：为了让机械臂按这个轨迹运动，需要多大力矩。

企业项目中常见情况：

- 规划器给出的轨迹运动学上可行。
- 但真机执行时电机力矩不够、速度过快、振动太大。
- 这就是动力学和控制问题。

## 2. 基础物理量

| 概念 | 含义 | 工程影响 |
|---|---|---|
| 质量 | 物体惯性大小 | 影响加速和制动 |
| 惯量 | 抗旋转变化能力 | 影响关节控制和仿真稳定性 |
| 力 | 改变线运动 | 底盘牵引、碰撞 |
| 力矩 | 改变旋转运动 | 机械臂关节、电机输出 |
| 摩擦 | 阻碍相对运动 | 轮子打滑、关节阻尼 |
| 重力 | 向下作用力 | 机械臂负载补偿 |

## 3. 机械臂动力学方程

常见形式：

```text
M(q) * q_ddot + C(q, q_dot) * q_dot + g(q) = tau
```

直觉：

- `M(q)`：惯性项，和机械臂姿态有关。
- `C(q, q_dot)`：科氏力和离心力项，和运动速度有关。
- `g(q)`：重力项。
- `tau`：关节力矩。

零基础阶段不需要手推完整方程，但要知道：

- 同样的动作，在不同姿态下需要不同力矩。
- 高速运动时速度相关项不能忽略。
- 抓重物时重力项和负载模型很重要。

## 4. 开环控制和闭环控制

### 4.1 开环控制

只发命令，不看反馈。

例子：

```text
电机转 1 秒
```

问题：

- 电池电压变化会影响结果。
- 摩擦和负载变化会影响结果。
- 无法知道是否达到目标。

### 4.2 闭环控制

根据反馈修正命令。

```text
误差 = 目标 - 当前
控制输入 = controller(误差)
```

闭环控制是机器人稳定运行的基础。

## 5. PID 控制

PID：

```text
u = Kp * e + Ki * integral(e) + Kd * derivative(e)
```

### 5.1 P：比例项

误差越大，控制越强。

过小：

- 响应慢。

过大：

- 振荡。
- 超调。

### 5.2 I：积分项

累积长期误差，用来消除稳态误差。

风险：

- 积分 windup。
- 响应变慢。
- 超调增加。

### 5.3 D：微分项

关注误差变化趋势，起阻尼作用。

风险：

- 对噪声敏感。
- 传感器抖动会放大控制输出。

## 6. 前馈 + 反馈

企业控制中常见结构：

```text
控制输入 = 前馈项 + 反馈项
```

前馈根据模型预测需要多少控制量，反馈负责修正误差。

例子：

- 机械臂重力补偿是前馈。
- PID 根据位置误差做反馈。

## 7. 轨迹跟踪

路径只描述“从哪里到哪里”。轨迹还包含时间：

```text
位置(t), 速度(t), 加速度(t)
```

控制器要跟踪轨迹，需要考虑：

- 位置误差。
- 速度误差。
- 加速度限制。
- jerk 限制。
- 控制频率。

## 8. 延迟、采样和限幅

### 8.1 延迟

延迟来自：

- 传感器采集。
- 网络通信。
- ROS 节点调度。
- 控制器执行。

延迟会让控制器基于“过去的状态”做决策，容易导致振荡。

### 8.2 采样周期

控制频率太低：

- 响应慢。
- 误差大。
- 高速运动不稳定。

### 8.3 限幅

真实系统有：

- 最大速度。
- 最大加速度。
- 最大力矩。
- 最大电流。

规划器和控制器必须尊重这些限制。

## 9. 仿真到真机差异

常见原因：

- 仿真质量和惯量不准。
- 摩擦模型过于理想。
- 电机响应延迟未建模。
- 传感器噪声太小。
- 真机结构有间隙和弹性。
- 控制周期和通信延迟不同。

## 10. 常见工程问题

| 现象 | 可能原因 |
|---|---|
| 到目标点后还来回抖 | P 太大、D 太小、延迟大 |
| 永远差一点到不了 | 没有积分项、摩擦死区、控制量太小 |
| 启动时猛冲 | 加速度限制不足、前馈过大 |
| 高速时轨迹偏差大 | 动力学约束、延迟、控制频率 |
| 仿真稳定真机不稳定 | 摩擦、惯量、延迟、噪声模型不一致 |

## 11. 练习

1. 写一个一维小车 PID 到达目标位置。
2. 调大 `Kp` 观察超调和振荡。
3. 加入积分项观察稳态误差变化。
4. 加入控制延迟观察系统变差。
5. 在仿真中修改质量和摩擦参数，观察控制效果。
