# 光纤传输与光模块学习笔记

最后整理：2026-06-11

光纤传输不是单个互联网协议，而是一组物理层技术。它使用光作为载体，适合高速、长距离、低电磁干扰的网络连接。数据中心、城域网、骨干网、运营商网络都大量使用光纤。

## 解决的问题

- 铜缆距离和抗干扰能力有限，高速传输损耗大。
- 机房、楼宇、城市之间需要更远距离和更高带宽。
- 电磁干扰环境中需要更稳定的传输介质。

## 核心概念

| 概念 | 说明 |
|---|---|
| 单模光纤 | 纤芯细，适合长距离，常见黄色跳线 |
| 多模光纤 | 纤芯较粗，适合短距离，常见橙色或水蓝色跳线 |
| 波长 | 常见 850 nm、1310 nm、1550 nm，不同模块要求不同 |
| 光模块 | SFP、SFP+、QSFP、QSFP28 等可插拔模块 |
| 光功率 | 接收光功率过高或过低都会导致误码或链路异常 |
| LC/SC 接头 | 常见光纤连接器类型 |

## 常见组合

| 场景 | 常见方案 |
|---|---|
| 机柜内短距 | DAC 铜缆、AOC 有源光缆、短距多模光模块 |
| 机房内互联 | 10GBASE-SR、25GBASE-SR、40G/100G SR |
| 楼宇/园区 | 单模光纤加 LR 模块 |
| 城域或长距 | ER/ZR 等长距光模块，可能配合波分复用 |

## 排查思路

1. 确认模块速率和端口能力匹配，例如 10G 端口不能随意插 25G 模块。
2. 确认单模/多模、波长、距离类型匹配。
3. 查看设备上的 DOM/DDM 信息：温度、电压、发射光功率、接收光功率。
4. 检查光纤端面清洁度，灰尘会明显影响光功率。
5. 检查 TX/RX 是否接反，双芯光纤需要一收一发交叉连接。

## 常见误区

- 光纤插上不亮不一定是线断，也可能是模块不兼容或端口未启用。
- 接收光功率不是越强越好，过载同样会导致错误。
- 单模和多模不能只看接口形状，必须看光纤类型、模块波长和距离指标。
