# 01. 机器人学数学基础

## 学习目标

学完本章，你应该能：

- 看懂机器人中的向量、矩阵、坐标变换和协方差。
- 理解导数、雅可比、梯度为什么会出现在控制、运动学和优化中。
- 理解传感器噪声为什么通常用概率分布描述。
- 能把“让误差最小”写成一个简单优化问题。

## 1. 线性代数

### 1.1 向量

向量可以表示：

- 位置：`p = [x, y, z]^T`
- 速度：`v = [vx, vy, vz]^T`
- 力：`f = [fx, fy, fz]^T`
- 误差：`e = target - current`

机器人中最常见的向量是列向量。写公式时通常默认：

```text
p = [x, y, z]^T
```

### 1.2 矩阵

矩阵可以表示线性变换。机器人中最常见的是旋转矩阵：

```text
p_world = R * p_robot
```

含义是：把机器人坐标系中的点 `p_robot` 旋转到世界坐标系。

### 1.3 矩阵乘法的工程意义

矩阵乘法不是单纯计算，它代表“连续应用变换”。

```text
p_world = T_world_base * T_base_camera * p_camera
```

含义是：

1. 点先从相机坐标系变到底盘坐标系。
2. 再从底盘坐标系变到世界坐标系。

矩阵乘法顺序不能随便换。`A * B` 通常不等于 `B * A`。

### 1.4 逆矩阵

如果 `T_world_base` 表示从 `base` 到 `world` 的变换，那么：

```text
T_base_world = inverse(T_world_base)
```

工程含义：

- 已知机器人在世界坐标系的位置。
- 想把世界坐标系下的目标点转换到机器人坐标系。

### 1.5 特征值、SVD 和最小二乘

零基础阶段不需要深入证明，但要知道用途：

| 概念 | 用途 |
|---|---|
| 特征值 | 稳定性分析、协方差主方向、PCA |
| SVD | 点云配准、最小二乘、矩阵分解 |
| 最小二乘 | 标定、SLAM、曲线拟合、轨迹优化 |

## 2. 微积分

### 2.1 导数

导数描述“变化率”。

```text
速度 = 位置对时间的导数
加速度 = 速度对时间的导数
```

在机器人中：

- 位置变化快慢是速度。
- 角度变化快慢是角速度。
- 误差变化快慢影响控制器调节。

### 2.2 偏导数

一个输出可能受多个变量影响。例如机械臂末端位置受多个关节角影响：

```text
x = f(theta1, theta2, theta3)
```

偏导数表示“只改变其中一个变量时，输出怎么变”。

### 2.3 雅可比矩阵

雅可比是很多偏导数组成的矩阵。

在机械臂中：

```text
末端速度 = J(关节角) * 关节速度
```

在状态估计中：

```text
线性化后的观测变化 = H * 状态变化
```

工程直觉：

- 雅可比告诉你“输入变量的小变化会怎样影响输出”。
- 它是运动学、优化、EKF、视觉定位的核心工具。

## 3. 概率统计

### 3.1 为什么机器人离不开概率

真实世界里：

- 轮速计会打滑。
- IMU 会漂移。
- 相机会有噪声。
- 雷达会被玻璃、黑色物体、动态行人影响。

所以机器人不能说“我一定在这里”，只能说“我大概率在这里”。

### 3.2 高斯分布

很多传感器噪声近似看作高斯噪声：

```text
x ~ N(mean, variance)
```

含义：

- `mean` 是最可能的值。
- `variance` 表示不确定性大小。

### 3.3 协方差

协方差描述多个变量之间的不确定性关系。

例如二维位置：

```text
P = [[var_x, cov_xy],
     [cov_yx, var_y]]
```

工程直觉：

- 协方差大：机器人不确定自己在哪里。
- 协方差小：机器人更确定自己的状态。
- 协方差不是“误差本身”，而是“对误差范围的估计”。

### 3.4 贝叶斯公式

机器人状态估计常用贝叶斯思想：

```text
后验 = 预测 * 观测修正
```

这就是滤波器的核心：

1. 根据运动模型预测机器人在哪里。
2. 根据传感器观测修正预测。

## 4. 优化

### 4.1 误差函数

机器人里很多问题都可以写成：

```text
找到 x，让 error(x) 最小
```

例子：

- 相机标定：让投影点和真实角点误差最小。
- SLAM：让所有里程计和观测约束误差最小。
- 轨迹优化：让路径平滑、安全、满足动力学约束。

### 4.2 最小二乘

最常见目标：

```text
min sum(error_i^2)
```

为什么平方：

- 惩罚大误差。
- 数学上容易求解。
- 和高斯噪声假设匹配。

### 4.3 梯度下降

梯度表示函数上升最快的方向。要最小化函数，就朝负梯度方向走：

```text
x_next = x - learning_rate * gradient
```

机器人中经常不是手写梯度下降，而是使用成熟优化库或算法，但理解它能帮助你看懂 SLAM 和轨迹优化。

## 5. 推荐学习路线

1. 向量和矩阵：能做坐标变换。
2. 三角函数：能理解旋转和机械臂几何。
3. 导数和偏导：能理解速度、雅可比、控制。
4. 概率和协方差：能理解状态估计。
5. 最小二乘：能理解标定、SLAM、优化。

## 6. 练习

### 练习 1：二维旋转

写一个 Python 函数，把点 `(1, 0)` 旋转 90 度，检查结果是否接近 `(0, 1)`。

### 练习 2：坐标变换链

已知：

- 相机相对底盘向前 0.3m、向上 0.2m。
- 物体在相机前方 1m。

计算物体在底盘坐标系下的位置。

### 练习 3：最小二乘拟合

给一组带噪声的二维点，拟合直线 `y = ax + b`，观察噪声越大拟合越不稳定。

## 7. 常见误区

- 只背公式，不知道变量在哪个坐标系。
- 混淆行向量和列向量。
- 把协方差当成真实误差。
- 以为优化一定能找到全局最优。
- 忽略单位，比如角度和弧度混用。
