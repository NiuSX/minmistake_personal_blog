# 04. 机械臂运动学

## 学习目标

学完本章，你应该能：

- 理解机械臂自由度、关节、连杆和末端执行器。
- 区分正运动学和逆运动学。
- 理解 DH 参数、雅可比、奇异位形和工作空间。
- 知道 MoveIt 2 运动规划背后的基本理论。

## 1. 机械臂运动学解决什么问题

机械臂运动学研究：

- 已知关节角，末端在哪里？
- 已知末端目标位姿，关节角应该是多少？
- 关节速度如何影响末端速度？
- 哪些目标机械臂无法到达？

它暂时不重点研究：

- 电机力矩是否足够。
- 高速运动下振动和柔性变形。
- 控制器如何精确跟踪轨迹。

## 2. 基本概念

| 概念 | 含义 |
|---|---|
| 连杆 link | 机械臂的刚性部件 |
| 关节 joint | 两个连杆之间的运动连接 |
| 自由度 DOF | 可独立控制的运动变量数量 |
| 末端执行器 | 夹爪、吸盘、焊枪、工具等 |
| 关节空间 | 用关节角或关节位移描述的空间 |
| 笛卡尔空间 | 用末端位置和姿态描述的空间 |

常见关节：

- Revolute joint：旋转关节。
- Prismatic joint：移动关节。

## 3. 正运动学

正运动学回答：

```text
已知关节角 theta，求末端位姿 T
```

写成函数：

```text
T = f(theta)
```

例子：二维二连杆机械臂。

```text
x = l1*cos(theta1) + l2*cos(theta1 + theta2)
y = l1*sin(theta1) + l2*sin(theta1 + theta2)
```

工程用途：

- 根据编码器角度计算夹爪位置。
- 在 RViz 中显示机械臂当前姿态。
- 检查末端是否到达目标。

## 4. 逆运动学

逆运动学回答：

```text
已知末端目标位姿 T，求关节角 theta
```

写成：

```text
theta = f_inverse(T)
```

逆运动学比正运动学更难，因为：

- 可能无解：目标超出工作空间。
- 可能多解：同一个末端位姿对应多组关节角。
- 可能无限多解：冗余机械臂自由度多于任务需要。
- 可能遇到奇异位形。

MoveIt 2 中 IK 插件就是为了解这个问题。

## 5. DH 参数

DH 参数是一种给机械臂建立坐标系的方法。

每个关节通常用 4 个参数描述：

| 参数 | 含义 |
|---|---|
| `a` | 连杆长度 |
| `alpha` | 连杆扭角 |
| `d` | 连杆偏距 |
| `theta` | 关节角 |

DH 的意义：

- 把复杂机械臂拆成一段段坐标变换。
- 每段变换相乘得到末端位姿。

工程提醒：

- ROS/URDF 不强制使用 DH 参数。
- 工业机器人资料中常能看到 DH 表。
- DH 坐标系建立方式不唯一，容易写错方向。

## 6. 雅可比矩阵

机械臂雅可比描述：

```text
末端速度 = J(theta) * 关节速度
```

包括：

- 末端线速度。
- 末端角速度。

用途：

- 速度控制。
- 奇异性分析。
- 力控制。
- 逆运动学迭代求解。

直觉：

- 某个关节动一点，末端会怎么动？
- 所有关节的小变化叠加起来，就是末端速度。

## 7. 奇异位形

奇异位形指机械臂在某些姿态下失去部分运动能力。

可能表现：

- 末端某个方向动不了。
- 需要极大的关节速度才能产生很小的末端速度。
- IK 解不稳定。
- 轨迹执行突然变慢或失败。

常见原因：

- 机械臂完全伸直。
- 多个关节轴线重合。
- 末端接近工作空间边界。

## 8. 工作空间

工作空间是机械臂末端能到达的区域。

影响因素：

- 连杆长度。
- 关节角限制。
- 自碰撞。
- 外部障碍物。
- 工具长度。

工程上不要只问“能不能到”，还要问：

- 到达时姿态是否满足要求？
- 是否接近关节限位？
- 是否容易碰撞？
- 是否远离奇异位形？

## 9. MoveIt 2 背后的理论

MoveIt 2 不是只做 IK，它通常涉及：

- URDF/SRDF 机器人模型。
- 关节限制。
- 碰撞检测。
- 运动规划。
- 轨迹参数化。
- 控制器执行。

典型流程：

```text
目标位姿 -> IK -> 规划路径 -> 碰撞检测 -> 轨迹时间参数化 -> 控制器执行
```

## 10. 常见工程问题

| 现象 | 可能理论原因 |
|---|---|
| 目标点看起来很近但规划失败 | 姿态约束、关节限制或碰撞约束 |
| 机械臂突然绕很大一圈 | IK 选了另一组解 |
| 接近某个姿态时动作很怪 | 奇异位形 |
| 仿真能抓到，真机抓偏 | 标定、夹爪模型、末端工具坐标错误 |
| 末端路径不直 | 关节空间规划不等于笛卡尔直线规划 |
