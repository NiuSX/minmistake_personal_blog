# 公式速查

## 讲义导读：用泛化主线读懂这一章

这一章讲的是 **公式速查**。阅读时不要只背公式和算法名，而要把它放进机器学习的完整链路：现实问题如何变成数据，样本和标签是否可靠，模型假设空间是什么，损失函数如何定义好坏，优化算法如何找到参数，验证集如何选择模型，测试集如何估计泛化，最后错误样本如何反过来改进数据和模型。机器学习真正要解决的不是训练集上得分高，而是在未来数据上稳定可用。

### 一句话先懂

公式速查不是背符号，而是帮助你快速回到每个公式解决的问题、变量含义和适用条件。

### 通俗类比

可以把机器学习想成从错题本里总结解题方法：训练集是做过的题，模型是假设的解题套路，损失函数是扣分规则，优化是反复订正，验证集像模拟考，测试集像正式考试，泛化能力就是换一套题还能不能做对。复杂模型不是天生更聪明，它只是可选择的解题套路更多，也更容易把错题本里的偶然细节背下来。

类比只能帮助入门。回到技术上，要把每个概念对应到精确定义：样本、特征、标签、假设空间、目标函数、训练误差、泛化误差、评价指标、正则化和数据分布。只有这些对象清楚，公式才不是符号堆，实验结果才有解释意义。

### 本章学习主线

1. **问题定义**：先确定任务是分类、回归、聚类、降维、序列决策还是概率推断。
2. **数据假设**：检查样本来源、标签噪声、特征含义、训练测试分布和数据泄漏。
3. **模型偏好**：说清算法偏好什么样的规律，例如线性、大间隔、短规则、平滑函数、稀疏特征或局部相似。
4. **损失与优化**：理解模型到底在最小化什么，优化失败、局部最优或过拟合会怎样表现。
5. **评估与泛化**：用合适指标、交叉验证、学习曲线、误差分析和消融实验判断模型是否可靠。
6. **落地边界**：关注可解释性、推理成本、鲁棒性、公平性、分布漂移和维护成本。

### 概念怎么学才不容易忘

遇到新算法时，建议按 白话任务 -> 模型形式 -> 损失函数 -> 优化方法 -> 泛化控制 -> 失败模式 六步理解。比如学习决策树，不只记信息增益，还要知道树为什么会过拟合、剪枝如何改变泛化；学习 SVM，不只写对偶问题，还要理解最大间隔为什么是一种归纳偏好；学习聚类，不只跑 K-means，还要说明距离度量和簇形状假设。

### 最小实践任务

任选三个公式，逐个写出变量含义、优化目标、直觉解释和一个数值小例子。

实践时要保留失败案例：训练分数高但测试差、指标选择不匹配、类别不平衡、数据泄漏、特征尺度错误、验证集被反复调参污染、聚类结果不可解释。机器学习能力很大一部分来自能解释这些失败。

### 读完本章应该能做到

- 用自己的话解释本章算法或概念解决什么学习问题。
- 写出数据、模型、损失、优化、评估和泛化控制六个要素。
- 给出一个适合场景和一个不适合场景，并说明背后的假设。
- 设计一个最小实验，能观察关键参数变化对结果的影响。
- 能看到公式就说明它对应的模型假设、目标函数和常见误用。

## 经验风险与结构风险

给定训练集 `D = {(x_i, y_i)}_{i=1}^m`，损失函数为 `L`：

```text
经验风险: R_emp(f) = (1/m) sum_i L(f(x_i), y_i)
结构风险: R_srm(f) = R_emp(f) + lambda * Omega(f)
```

`Omega(f)` 表示模型复杂度惩罚，`lambda` 控制经验拟合与复杂度之间的权衡。

## 均方误差

```text
E(w, b) = sum_i (y_i - w^T x_i - b)^2
```

若把偏置并入参数，令 `X` 为增广样本矩阵，`w` 为增广参数：

```text
w* = (X^T X)^(-1) X^T y
```

当前提是 `X^T X` 可逆；不可逆时可用伪逆或正则化。

## 逻辑回归

二分类中：

```text
p(y=1|x) = 1 / (1 + exp(-(w^T x + b)))
```

负对数似然：

```text
L(w) = - sum_i [ y_i log p_i + (1-y_i) log(1-p_i) ]
```

梯度：

```text
grad_w L = sum_i (p_i - y_i) x_i
```

## 信息熵、信息增益、基尼指数

```text
Ent(D) = - sum_k p_k log_2 p_k
Gain(D, a) = Ent(D) - sum_v (|D_v|/|D|) Ent(D_v)
Gini(D) = 1 - sum_k p_k^2
```

信息增益偏好取值多的属性，增益率用属性固有值进行校正。

## 神经网络反向传播

对第 `l` 层：

```text
z^l = W^l a^{l-1} + b^l
a^l = sigma(z^l)
delta^l = (W^{l+1})^T delta^{l+1} ⊙ sigma'(z^l)
partial L / partial W^l = delta^l (a^{l-1})^T
```

核心是链式法则：后一层误差信号乘以前一层激活值。

## 支持向量机

硬间隔原始问题：

```text
min_{w,b} 1/2 ||w||^2
s.t. y_i(w^T x_i + b) >= 1
```

软间隔：

```text
min_{w,b,xi} 1/2 ||w||^2 + C sum_i xi_i
s.t. y_i(w^T x_i + b) >= 1 - xi_i, xi_i >= 0
```

对偶形式常写为：

```text
max_alpha sum_i alpha_i - 1/2 sum_i sum_j alpha_i alpha_j y_i y_j K(x_i, x_j)
s.t. sum_i alpha_i y_i = 0, 0 <= alpha_i <= C
```

## 朴素贝叶斯

```text
h(x) = argmax_c P(c) Π_j P(x_j | c)
```

“朴素”假设是：给定类别后，各属性条件独立。

## AdaBoost

基学习器权重：

```text
alpha_t = 1/2 ln((1 - epsilon_t) / epsilon_t)
```

样本权重更新：

```text
D_{t+1}(i) ∝ D_t(i) exp(-alpha_t y_i h_t(x_i))
```

错分样本满足 `y_i h_t(x_i) = -1`，权重被放大。

## K-means

目标函数：

```text
J = sum_i ||x_i - mu_{c_i}||^2
```

固定簇中心时，最优分配是最近中心；固定分配时，最优中心是簇内均值。

## PCA

最大方差形式：

```text
max_W tr(W^T X^T X W)
s.t. W^T W = I
```

解为协方差矩阵最大特征值对应的特征向量。

## 强化学习

状态价值函数：

```text
V^pi(s) = E_pi [ sum_{t=0}^infty gamma^t r_{t+1} | s_0=s ]
```

Bellman 期望方程：

```text
V^pi(s) = sum_a pi(a|s) sum_{s'} P(s'|s,a) [ R(s,a,s') + gamma V^pi(s') ]
```

最优动作价值函数：

```text
Q*(s,a) = sum_{s'} P(s'|s,a) [ R(s,a,s') + gamma max_{a'} Q*(s',a') ]
```

## 2026 机器学习资料与实践核对补充

机器学习的经典理论稳定，但工具库、默认参数、数据集版本和评估规范会变化。复现实验前建议记录 Python、NumPy、SciPy、scikit-learn、PyTorch 或其他库版本，固定随机种子、数据切分、预处理步骤、特征工程、评价指标和硬件环境。尤其要防止数据泄漏、验证集污染和只报告最好一次结果。

学习西瓜书和南瓜书时，建议把书中推导、南瓜书公式解释、scikit-learn 实验和自己的手算例子结合起来。官方/经典资料负责正确性，代码实验负责直觉，误差分析负责落地判断。
