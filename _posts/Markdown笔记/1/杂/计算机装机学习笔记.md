# 计算机装机学习笔记：小白硬件选购、品牌型号与装机流程

## 讲义导读：把概念落到可验证实践

这一章讲的是 **计算机装机学习笔记：小白硬件选购、品牌型号与装机流程**，属于 **计算机硬件选购与装机**。阅读时不要把它当成零散资料堆叠，而要把它当成一份讲义：先弄清它解决什么问题，再看核心概念和流程，最后做一个能复现、能观察、能排错的小练习。

### 一句话先懂

装机学习的核心，是按用途和预算选择兼容硬件，并安全完成装配、装系统、驱动和稳定性测试。

初学时先问三个问题：它的输入或前提是什么；它内部按什么规则工作；结果该用什么命令、日志、测试、图纸、波形或指标来证明。

### 通俗类比

装机像配一套工作台：每个零件单独强不代表整套好用，接口、尺寸、供电、散热和预算必须一起匹配。

类比只是入门扶手。真正掌握时，要回到准确术语、配置、接口、版本、边界条件、错误信息和验证证据上。能解释失败原因，比只会照着步骤跑通更重要。

### 本章学习主线

1. **先看场景**：这个知识点通常在什么项目、岗位或问题里出现？
2. **再看结构**：它有哪些核心对象、配置、文件、命令、接口或流程？
3. **然后看路径**：一次完整使用从哪里开始，到哪里结束，中间有哪些状态变化？
4. **接着看边界**：版本差异、平台差异、权限、性能、安全、兼容性和资源限制在哪里？
5. **最后看验证**：用最小样例、测试、日志、调试工具或实物结果证明理解是对的。

### 本章重点抓手

CPU、主板、内存、显卡、硬盘、电源、散热、机箱、显示器、兼容性、BIOS、驱动、装机流程和压力测试。

### 最小实践任务

按一个明确用途写配置单，核对主板 QVL、显卡长度、电源接口、散热限高、BIOS 支持和整机功耗。

建议把练习记录成固定格式：目标、环境版本、最小示例、执行步骤、预期结果、实际结果、错误信息、定位过程和复盘。以后遇到真实项目问题时，这些记录会比单纯收藏教程更有用。

### 常见误区

- 只看跑分，不看用途和兼容性。
- 电源、散热和机箱空间留量不足。
- 买前不查当天价格、售后和主板支持列表。

### 推荐工具与资料

官方文档、最小 demo、日志、调试器、版本管理、测试命令、性能/诊断工具和复盘记录。

## 1. 装机先理解整机由什么组成

一台台式电脑通常由以下部件组成：

| 部件 | 作用 | 小白理解 |
| --- | --- | --- |
| CPU | 负责通用计算 | 大脑，决定很多软件响应速度 |
| 主板 | 连接所有硬件 | 地基和交通枢纽 |
| 内存 RAM | 临时运行数据 | 桌面，越大越能同时摊开很多东西 |
| 硬盘 SSD/HDD | 长期存储数据 | 仓库，系统、软件、游戏、照片都放这里 |
| 显卡 GPU | 图形、游戏、AI、渲染 | 画面和并行计算的主力 |
| 电源 PSU | 给硬件供电 | 电力系统，不能省太狠 |
| 散热器 | 给 CPU/GPU 降温 | 空调和风道 |
| 机箱 | 安装和保护硬件 | 房子，决定空间和风道 |
| 显示器 | 输出画面 | 体验上限之一 |
| 键鼠音箱等 | 外设 | 影响操作和日常体验 |

装机最重要的不是单个配件最强，而是整套配置均衡、兼容、稳定、符合用途。

## 2. 先确定用途和预算

### 2.1 常见用途

**办公学习**

- 文档、网页、网课、轻度图片处理。
- 重点：稳定、安静、低功耗、够用。
- 可以选择带核显 CPU，不一定需要独立显卡。

**游戏**

- 重点：显卡、CPU、显示器分辨率和刷新率。
- 1080p 游戏更吃 CPU 和显卡平衡。
- 2K/4K 游戏更吃显卡。

**设计剪辑**

- 重点：CPU 多核、显卡加速、内存容量、SSD 容量和速度。
- 视频剪辑建议 32GB 内存起步，重度 4K/8K 可以 64GB 或更高。

**AI、深度学习、Stable Diffusion、3D 渲染**

- 重点：显卡显存、CUDA/ROCm/驱动生态、内存和 SSD。
- NVIDIA 显卡在 CUDA 生态下兼容性通常更强。

**编程开发**

- 重点：CPU、内存、SSD。
- 大型项目、虚拟机、Docker、Android Studio、IDE 多开建议 32GB 内存起步。

### 2.2 预算分配经验

游戏主机预算通常可以这样分：

| 部件 | 建议占比 |
| --- | --- |
| 显卡 | 35% - 50% |
| CPU | 15% - 25% |
| 主板 | 8% - 15% |
| 内存 | 6% - 12% |
| SSD | 6% - 12% |
| 电源 | 6% - 10% |
| 机箱散热 | 6% - 12% |

办公主机则可以减少显卡预算，把钱放到 CPU、内存、SSD、显示器和静音体验上。

## 3. CPU 处理器

### 3.1 CPU 看哪些参数

| 参数 | 含义 | 小白怎么理解 |
| --- | --- | --- |
| 核心数 | 能同时处理多少任务 | 多核适合渲染、编译、多任务 |
| 线程数 | CPU 同时调度的执行单元 | 线程多不等于游戏一定更强 |
| 频率 | 每秒运行周期 | 高频有利于游戏和单线程 |
| 缓存 | CPU 内部高速存储 | 游戏尤其吃缓存，X3D 就是典型 |
| 功耗/TDP | 发热和供电参考 | 功耗越高，散热和电源压力越大 |
| 插槽 | 和主板匹配 | 插槽不对，无法安装 |
| 内存支持 | DDR4/DDR5、频率 | 新平台多为 DDR5 |
| 核显 | CPU 自带显示能力 | 没独显也能亮机和办公 |

### 3.2 当前主流桌面 CPU 平台

| 平台 | 插槽 | 内存 | 特点 |
| --- | --- | --- | --- |
| AMD Ryzen 7000/8000/9000/9000X3D | AM5 | DDR5 | 平台寿命较长，游戏和生产力都有强型号 |
| Intel Core Ultra 200S/200S Plus | LGA1851 | DDR5 | 新桌面平台，强调能效、AI PC、创作和新芯片组 |
| Intel 12/13/14 代 Core | LGA1700 | DDR4/DDR5 看主板 | 市场存量大，价格可能有优势 |

AMD 官方资料显示 AM5 覆盖 Ryzen 7000、8000、9000 系列，并支持 DDR5、PCIe 5.0 等现代特性。Intel Core Ultra 200S 使用 Intel 800 系列芯片组，面向新一代桌面平台。

### 3.3 AMD CPU 型号怎么理解

以 `Ryzen 7 9800X3D` 为例：

- Ryzen：AMD 消费级处理器品牌。
- 7：产品等级，常见有 Ryzen 5、7、9。
- 9800：代际和定位。
- X：高性能型号。
- X3D：带 3D V-Cache 大缓存，通常游戏表现强。

AMD 常见定位：

| 系列 | 适合人群 |
| --- | --- |
| Ryzen 5 | 主流游戏、办公、性价比 |
| Ryzen 7 | 中高端游戏、多任务 |
| Ryzen 9 | 创作、渲染、编译、重度生产力 |
| Ryzen X3D | 游戏优先，尤其高帧率游戏 |
| Ryzen G/APU | 带较强核显，适合无独显小主机 |

典型理解：

- `Ryzen 5 7600/9600X`：主流游戏和办公。
- `Ryzen 7 7800X3D/9800X3D/9850X3D`：偏游戏神U路线，适合高帧率游戏。
- `Ryzen 9 7900/9900X/9950X`：多核更强，适合渲染、编译、创作。
- `Ryzen 9 X3D`：兼顾高端游戏和多核，但价格通常更高。

### 3.4 Intel CPU 型号怎么理解

Intel 新桌面命名使用 Core Ultra，例如：

- Core Ultra 5：主流。
- Core Ultra 7：中高端。
- Core Ultra 9：高端。
- K：可超频或高性能定位。

旧命名如 `i5-14600K`：

- i5/i7/i9：等级。
- 14：第 14 代。
- 600/700/900：定位。
- K：可超频。
- F：无核显。

Intel 常见定位：

| 系列 | 适合人群 |
| --- | --- |
| Core i3 / Core Ultra 3 | 入门办公 |
| Core i5 / Core Ultra 5 | 主流游戏和办公 |
| Core i7 / Core Ultra 7 | 中高端游戏、创作 |
| Core i9 / Core Ultra 9 | 高端生产力和旗舰配置 |
| F 后缀 | 无核显，必须配独显 |
| K 后缀 | 高性能、可超频，散热要求更高 |

### 3.5 AMD 和 Intel 怎么选

| 需求 | 更关注 |
| --- | --- |
| 纯游戏 | AMD X3D、Intel 高端高频型号都可比较当日价格 |
| 剪辑渲染 | 核心数、线程数、软件加速支持 |
| AI/生产力 | CPU 不是唯一，显卡和内存也很关键 |
| 预算有限 | 看板 U 套装价格，不只看 CPU 单价 |
| 长期升级 | 看平台寿命、主板扩展和 BIOS 支持 |
| 无独显办公 | 选带核显的 CPU，避开 F 后缀 |

小白建议：

- 不超频就不要为了超频买太贵主板和散热。
- 游戏优先看 CPU + 显卡整体搭配，不要 CPU 太强显卡太弱。
- 确认主板 BIOS 支持你买的 CPU。
- CPU 散热预算不能省到离谱，高端 CPU 要配好散热。

## 4. 主板

### 4.1 主板决定什么

主板不直接让游戏帧数暴涨，但决定：

- CPU 是否能装。
- 内存类型和频率。
- SSD 插槽数量。
- 显卡插槽。
- USB、网卡、声卡。
- 供电能力。
- 未来扩展。
- BIOS 功能。

### 4.2 主板尺寸

| 尺寸 | 特点 |
| --- | --- |
| ATX | 标准大板，扩展好 |
| Micro-ATX / M-ATX | 中小板，性价比高 |
| Mini-ITX | 小主机，贵且装机难度高 |

小白推荐 ATX 或 M-ATX，不建议第一次装机就做 ITX 小钢炮。

### 4.3 AMD AM5 芯片组

| 芯片组 | 定位 |
| --- | --- |
| A620 | 入门，扩展和供电较弱 |
| B650/B650E | 主流性价比 |
| B840/B850 | 新一代主流定位 |
| X670/X670E | 高端扩展 |
| X870/X870E | 新一代高端，通常 USB4、PCIe 5.0 支持更完整 |

AMD 官方 AM5 芯片组页面显示，X870 面向 Ryzen 9000/8000/7000，支持 DDR5、AMD EXPO、PCIe 5.0 NVMe 和 USB4 等特性。

### 4.4 Intel 主板芯片组

| 平台 | 常见芯片组 | 说明 |
| --- | --- | --- |
| LGA1851 Core Ultra 200S | Intel 800 系列，如 Z890、B860 等 | 新平台 |
| LGA1700 12/13/14 代 | H610、B660/B760、Z690/Z790 | 存量平台 |

Intel 官方介绍 Core Ultra 200S 搭配 Intel 800 系列芯片组，提供更多 PCIe、SATA、USB 扩展。

### 4.5 主板购买检查

- CPU 插槽是否匹配。
- 是否支持你的 CPU，必要时是否要刷 BIOS。
- 内存是 DDR4 还是 DDR5。
- M.2 SSD 插槽够不够。
- 显卡插槽位置会不会挡住其他接口。
- 供电散热是否足够。
- 后置 USB 数量是否够用。
- 是否需要 Wi-Fi 和蓝牙。
- 是否有 2.5G 网口。
- 机箱是否支持该主板尺寸。

## 5. 内存 RAM

### 5.1 内存容量怎么选

| 用途 | 建议容量 |
| --- | --- |
| 基础办公 | 16GB |
| 主流游戏 | 32GB |
| 剪辑、设计、编程多开 | 32GB - 64GB |
| 虚拟机、AI、大型工程 | 64GB 起 |

2026 年新装机通常建议 DDR5 32GB 起步，除非预算极低或只是轻办公。

### 5.2 频率和时序

内存常见写法：

```text
DDR5-6000 CL30
```

含义：

- 6000：传输速率，单位 MT/s。
- CL30：列访问延迟，数字越小越好，但不能只看 CL。

粗略真实延迟：

```text
真实延迟 ns ≈ CL / 频率 MHz * 2000
```

例如 DDR5-6000 CL30：

```text
30 / 6000 * 2000 = 10 ns
```

### 5.3 XMP 和 EXPO

- XMP：Intel 常见内存超频配置。
- EXPO：AMD 常见内存超频配置。

注意：

- 标称高频通常需要进 BIOS 开启 XMP/EXPO。
- 不开启时可能跑在 JEDEC 默认频率。
- 高频不是越高越稳，主板、CPU 内存控制器、内存颗粒都会影响稳定性。

### 5.4 内存怎么插

两根内存优先插主板说明书推荐的 A2/B2 槽位。

建议：

- 2 条比 4 条更容易高频稳定。
- 买套条，不要东拼西凑。
- 同容量、同频率、同型号更稳。

## 6. 硬盘：SSD 和 HDD

### 6.1 硬盘类型

| 类型 | 特点 | 适合 |
| --- | --- | --- |
| SATA HDD 机械硬盘 | 容量大、便宜、慢、有噪音 | 仓库盘、备份、视频素材 |
| SATA SSD | 比 HDD 快，接口老但兼容好 | 老电脑升级、普通软件盘 |
| NVMe M.2 SSD | 速度快，主流新装机首选 | 系统盘、游戏盘、工作盘 |
| PCIe 4.0 NVMe | 当前主流高性能 | 大多数用户足够 |
| PCIe 5.0 NVMe | 极高速、发热高、价格高 | 高端创作、重负载、发烧 |

### 6.2 SSD 看哪些参数

| 参数 | 含义 |
| --- | --- |
| 容量 | 1TB、2TB、4TB 等 |
| 接口 | SATA、PCIe 3.0/4.0/5.0 |
| 协议 | NVMe 或 SATA |
| 顺序读写 | 大文件传输速度 |
| 随机读写 IOPS | 系统响应、小文件性能 |
| 缓存 | DRAM 或 HMB |
| 颗粒 | TLC、QLC 等 |
| TBW | 总写入寿命参考 |
| 散热 | PCIe 5.0 SSD 尤其需要 |

### 6.3 TLC 和 QLC

| 类型 | 特点 |
| --- | --- |
| TLC | 性能和寿命更均衡，主流推荐 |
| QLC | 容量成本低，但持续写入和寿命通常弱一些 |

系统盘、工作盘优先选 TLC。仓库盘和轻度游戏盘可以考虑 QLC，但要看价格。

### 6.4 PCIe 4.0 和 PCIe 5.0 SSD 怎么选

Samsung 990 PRO 是 PCIe 4.0 旗舰级 SSD，官方标称顺序读写可到约 7450/6900 MB/s。Samsung 9100 PRO 是 PCIe 5.0 SSD，官方页面给到约 14800/13400 MB/s 的顺序读写。

选择建议：

| 用户 | 建议 |
| --- | --- |
| 普通办公 | 主流 PCIe 3.0/4.0 都够 |
| 游戏 | 1TB/2TB PCIe 4.0 更合适 |
| 剪辑大素材 | 高性能 PCIe 4.0 或 PCIe 5.0 |
| 大量 AI 数据、超大文件 | PCIe 5.0 可考虑，但注意散热 |
| 预算有限 | 容量优先于极限速度 |

游戏加载通常不会因为 PCIe 5.0 比 PCIe 4.0 快一倍而同等比例变快。小白优先买容量够、口碑好、售后稳的型号。

### 6.5 机械硬盘 HDD 品牌系列

机械硬盘主要看用途，不要只看容量。

Western Digital 颜色系列：

| 系列 | 定位 |
| --- | --- |
| WD Blue | 日常家用、普通存储 |
| WD_BLACK | 高性能、游戏、创作 |
| WD Red / Red Pro | NAS 网络存储 |
| WD Purple / Purple Pro | 监控录像 |
| WD Gold | 企业级、数据中心 |

Seagate 常见系列：

| 系列 | 定位 |
| --- | --- |
| BarraCuda | 日常桌面存储 |
| FireCuda | 游戏、高性能存储 |
| IronWolf / IronWolf Pro | NAS |
| SkyHawk | 监控 |
| Exos | 企业级 |

### 6.6 HDD 重要概念：CMR 和 SMR

- CMR：传统磁记录，随机写入和长期稳定性更好。
- SMR：叠瓦式磁记录，成本低但随机写入和重写可能变慢。

NAS、频繁写入、重要数据盘优先 CMR。购买前查清具体型号，不要只看系列名。

### 6.7 硬盘搭配方案

| 方案 | 适合 |
| --- | --- |
| 1TB NVMe SSD | 轻办公、小预算 |
| 2TB NVMe SSD | 主流游戏和日常推荐 |
| 1TB 系统盘 + 2TB 游戏盘 | 游戏多、想分类管理 |
| 2TB NVMe + 4TB/8TB HDD | 剪辑、资料、照片视频 |
| 2TB 高速 SSD + NAS | 数据安全和多设备访问 |

重要提醒：硬盘不是备份。重要资料至少 3-2-1 备份：3 份数据、2 种介质、1 份异地。

## 7. 显卡 GPU

### 7.1 显卡看哪些参数

| 参数 | 含义 |
| --- | --- |
| GPU 型号 | 决定核心性能 |
| 显存容量 | 影响高分辨率、AI、纹理、创作 |
| 显存位宽/带宽 | 影响数据吞吐 |
| 功耗 | 影响电源和散热 |
| 光追性能 | 光线追踪游戏表现 |
| AI 加速 | DLSS、生成式 AI、推理 |
| 编码器 | 直播、剪辑、录屏 |
| 尺寸 | 是否能装进机箱 |
| 供电接口 | 8pin、12VHPWR、12V-2x6 |

### 7.2 NVIDIA 显卡

当前主流产品线包括 RTX 50 系列和 RTX 40 系列。NVIDIA 官方 RTX 50 页面显示，该系列基于 Blackwell 架构，使用 GDDR7，强调 DLSS 4.5、AI 和新一代光追能力。

常见定位：

| 系列 | 适合 |
| --- | --- |
| RTX 5050/5060 | 1080p 入门游戏、轻创作 |
| RTX 5060 Ti/5070 | 1080p 高刷、2K 主流 |
| RTX 5070 Ti/5080 | 2K 高刷、4K 入门、创作 |
| RTX 5090 | 4K 旗舰、AI、重度创作 |

NVIDIA 优点：

- 游戏生态强。
- 光追和 DLSS 生态成熟。
- CUDA 对 AI、深度学习、渲染、创作软件非常重要。
- NVENC 编码器适合直播剪辑。

注意：

- 中高端显卡功耗和体积大。
- 高端卡要注意 12V-2x6/12VHPWR 线材插紧、不要硬折。
- 同一 GPU 不同品牌散热、噪音、供电、价格差异很大。

### 7.3 AMD Radeon 显卡

AMD 官方页面显示 Radeon RX 9000 系列基于 RDNA 4 架构，强调游戏、光追和 AI 加速。RX 9070 官方规格页面显示 RX 9070 拥有 56 个计算单元、16GB 显存相关产品线定位。

常见定位：

| 系列 | 适合 |
| --- | --- |
| RX 7600/7700 系列 | 1080p/2K 入门到主流 |
| RX 7800/7900 系列 | 2K/4K 游戏 |
| RX 9060/9070 系列 | 新一代主流到高性能游戏 |

AMD 优点：

- 同价位显存容量常有优势。
- 传统光栅游戏性能通常有竞争力。
- FSR 跨品牌可用。
- Linux/开源生态在部分场景友好。

注意：

- 如果主要做 CUDA 深度学习，NVIDIA 通常更省心。
- 光追和部分创作软件生态要具体看应用支持。

### 7.4 Intel Arc 显卡

Intel Arc B 系列包括 B580、B570。Intel 官方资料显示 B580 为 12GB 显存，B570 为 10GB 显存，定位现代 1440p 游戏和内容创作。

Intel Arc 优点：

- 中低价位显存配置有吸引力。
- AV1 编解码支持适合视频用户。
- 性价比可能不错。

注意：

- 驱动生态仍需要看具体游戏和软件。
- 建议主板开启 ReBAR。
- 新手如果追求最省心，优先 NVIDIA/AMD 主流型号。

### 7.5 显卡品牌怎么理解

GPU 芯片来自 NVIDIA/AMD/Intel，但显卡成品由板卡厂商生产。

常见品牌：

- ASUS 华硕。
- MSI 微星。
- GIGABYTE 技嘉。
- GALAX 影驰。
- ZOTAC 索泰。
- Colorful 七彩虹。
- Sapphire 蓝宝石，AMD 强势。
- PowerColor 撼讯，AMD 强势。
- XFX 讯景，AMD 常见。
- ASRock 华擎，AMD/Intel 常见。

同芯片不同品牌主要差异：

- 散热规模。
- 噪音。
- 供电设计。
- 频率。
- 保修。
- 尺寸。
- 外观灯效。
- 价格。

小白选显卡：

- 先确定 GPU 型号，再比较具体品牌。
- 不必盲目买最贵旗舰非公。
- 看显卡长度、厚度、电源接口。
- 看本地保修和售后。
- 避免矿卡、拆修卡、无保卡。

## 8. 电源 PSU

### 8.1 电源为什么重要

电源差会导致：

- 黑屏重启。
- 游戏崩溃。
- 硬盘掉盘。
- 显卡供电不稳。
- 高负载啸叫或保护。
- 严重时损坏硬件。

### 8.2 电源看哪些参数

| 参数 | 含义 |
| --- | --- |
| 额定功率 | 持续输出能力 |
| 80 PLUS / Cybenetics | 转换效率认证 |
| ATX 3.0/3.1 | 新显卡供电规范 |
| 12V 输出 | CPU/GPU 主要用电 |
| 模组线 | 方便走线 |
| 质保 | 好电源通常质保长 |
| 接口 | CPU 8pin、PCIe 8pin、12V-2x6 |

### 8.3 12VHPWR 和 12V-2x6

高端显卡可能使用 16pin 供电。ATX 3.1/PCIe 5.1 相关电源常见 12V-2x6 接口。Seasonic 资料显示 12V-2x6 单线可提供最高 600W，并是面向新显卡的连接方案。

使用注意：

- 插头必须完全插到底。
- 不要在接头附近急弯。
- 不要混用不同品牌模组线。
- 优先使用电源原生线材，不要乱用转接线。

### 8.4 电源功率建议

| 配置 | 建议电源 |
| --- | --- |
| 核显办公 | 300W - 450W |
| 主流 CPU + 入门独显 | 500W - 650W |
| 主流游戏显卡 | 650W - 750W |
| 中高端显卡 | 750W - 850W |
| 高端显卡 | 850W - 1000W+ |
| 旗舰显卡 + 高端 CPU | 1000W - 1200W+ |

小白建议：

- 电源不要买杂牌。
- 不要只看峰值功率，要看额定功率。
- 高端显卡优先 ATX 3.1、原生 12V-2x6。
- 机箱要能容纳电源长度和线材空间。

## 9. 散热器

### 9.1 CPU 散热类型

| 类型 | 特点 |
| --- | --- |
| 原装散热 | 便宜够用，但性能有限 |
| 单塔风冷 | 入门到主流 |
| 双塔风冷 | 性价比高，稳定耐用 |
| 240 水冷 | 中高端，美观，兼容要看机箱 |
| 360 水冷 | 高端 CPU，更强散热 |

### 9.2 风冷和水冷怎么选

风冷优点：

- 稳定。
- 维护简单。
- 不怕漏液。
- 性价比高。

水冷优点：

- 高功耗 CPU 温度表现好。
- 机箱内部更清爽。
- 外观好。

小白建议：

- 主流 CPU 用好一点双塔风冷即可。
- 高端 i9/Ryzen 9 可以考虑 360 水冷。
- 机箱必须支持冷排尺寸。
- 注意内存高度和散热器冲突。

## 10. 机箱和风道

### 10.1 机箱看什么

- 支持主板尺寸。
- 显卡长度。
- CPU 散热器高度。
- 水冷冷排位置。
- 电源长度。
- 硬盘位。
- 风扇位。
- 前面板进风。
- 防尘网。
- 走线空间。

### 10.2 风道原则

常见风道：

```text
前进风 -> 后出风
下进风 -> 上出风
```

热空气上升，上方和后方通常适合出风。

小白建议：

- 不要买完全闷罐机箱。
- 前面板网孔进风通常更好。
- 至少保证前进风、后出风。
- 显卡越高端，机箱风道越重要。

## 13. 小白装机兼容性检查表

购买前逐项确认：

- CPU 插槽和主板一致。
- 主板 BIOS 支持 CPU。
- 主板内存类型和内存一致。
- 内存高度不挡散热器。
- SSD 是 M.2 还是 SATA，主板插槽够。
- 显卡长度小于机箱限制。
- 显卡厚度不会挡住其他接口。
- 电源功率足够。
- 电源有显卡需要的接口。
- CPU 散热器支持对应平台扣具。
- CPU 散热器高度小于机箱限制。
- 水冷冷排尺寸机箱支持。
- 主板尺寸机箱支持。
- 显示器接口和显卡接口匹配。

## 14. 装机流程

### 14.1 准备工具

- 十字螺丝刀。
- 扎带。
- U 盘，安装系统。
- 硅脂，部分散热器自带。
- 防静电意识，摸金属机箱放电。
- 主板说明书。

### 14.2 推荐安装顺序

1. 主板放在盒子上。
2. 安装 CPU。
3. 安装内存。
4. 安装 M.2 SSD。
5. 安装 CPU 散热器背板和散热器。
6. 机箱安装主板挡板，如果不是一体 I/O。
7. 主板装入机箱。
8. 安装电源。
9. 接主板 24pin、电源 CPU 8pin。
10. 安装显卡。
11. 接显卡供电。
12. 接机箱前面板线。
13. 接风扇和 ARGB/RGB。
14. 检查所有线材。
15. 首次开机进 BIOS。
16. 开启 XMP/EXPO。
17. 安装系统和驱动。

### 14.3 CPU 安装注意

- AMD AM5 和 Intel LGA 都是主板针脚或触点，非常脆弱。
- CPU 有三角标记，对准主板标记。
- 不要用力压错方向。
- 不要摸触点。

### 14.4 硅脂

硅脂作用是填补 CPU 和散热器底座微小缝隙，不是越多越好。

常见方法：

- 中间一点。
- 米粒大小。
- X 形。

压力会把硅脂压开。涂太多可能溢出，太少可能覆盖不均。

### 14.5 前面板跳线

机箱前面板常见线：

- PWR SW：开机键。
- RESET SW：重启键。
- HDD LED：硬盘灯。
- POWER LED：电源灯。

看主板说明书接。开机键没有正负，LED 有正负。

## 15. BIOS 和系统安装

### 15.1 首次进 BIOS 看什么

- CPU 是否识别。
- 内存容量是否正确。
- SSD 是否识别。
- CPU 温度是否正常。
- 风扇是否转。
- BIOS 版本。
- 启动顺序。

### 15.2 XMP/EXPO

装好系统前后都可以进 BIOS 开启。开启后建议做稳定性测试。

### 15.3 Windows 安装后驱动

建议安装：

- 主板芯片组驱动。
- 显卡驱动。
- 网卡/Wi-Fi/蓝牙驱动。
- 声卡驱动，必要时。
- SSD 工具，必要时。

驱动优先从官方站点下载，不要随便用来路不明的驱动工具。

## 16. 常见故障排查

### 16.1 按开机没反应

检查：

- 电源开关是否打开。
- 24pin 是否插紧。
- CPU 8pin 是否插紧。
- PWR SW 是否接对。
- 插座和电源线是否正常。

### 16.2 风扇转但不亮屏

检查：

- 显示器线是否插在独显上，而不是主板上。
- 内存是否插紧。
- 显卡是否插紧。
- 显卡供电是否插好。
- CPU 是否带核显。
- 主板 Debug 灯显示什么。

### 16.3 内存容量不对

检查：

- 内存是否插紧。
- 插槽是否正确。
- 单条测试。
- BIOS 是否识别。

### 16.4 温度过高

检查：

- 散热器保护膜是否撕掉。
- 硅脂是否正常。
- 散热器是否压紧。
- 风扇方向是否正确。
- 机箱风道是否太差。

### 16.5 游戏闪退或黑屏

检查：

- 显卡驱动。
- 电源功率和供电线。
- 显卡温度。
- 内存 XMP/EXPO 稳定性。
- 游戏文件完整性。

## 17. 推荐配置思路

### 17.1 办公学习主机

重点：

- 带核显 CPU。
- 16GB/32GB 内存。
- 1TB SSD。
- 安静电源和机箱。

可以不买独显。

### 17.2 1080p 游戏

重点：

- 主流 6 核/8 核 CPU。
- 入门到主流显卡。
- 32GB 内存。
- 1TB/2TB SSD。

### 17.3 2K 游戏甜点

重点：

- Ryzen 5/7 或 Core Ultra 5/7。
- RTX 5070/RX 9070 级别上下的显卡，按价格调整。
- 32GB DDR5。
- 2TB PCIe 4.0 SSD。
- 750W 左右优质电源。

### 17.4 4K 高端游戏

重点：

- 高端 CPU，游戏可考虑 X3D。
- RTX 5080/5090 或对应高端显卡。
- 32GB/64GB 内存。
- 2TB - 4TB SSD。
- 850W - 1200W 优质 ATX 3.1 电源。
- 高风道机箱。

### 17.5 剪辑创作

重点：

- 多核 CPU。
- NVIDIA/AMD 按软件支持选择。
- 64GB 内存更舒服。
- 系统盘 + 素材盘 + 仓库盘。
- 显示器色准。

### 17.6 AI 学习和本地模型

重点：

- NVIDIA 显卡优先，主要因为 CUDA 生态。
- 显存越大越好。
- 32GB/64GB 系统内存。
- 2TB SSD 起步。
- 电源和散热留余量。

## 18. 购买避坑

- 不买来路不明的“全新散片显卡”。
- 不买无保矿卡给小白主力机。
- 不用杂牌电源。
- 不买容量很小的系统盘。
- 不买和 CPU 不兼容的主板。
- 不买装不进机箱的显卡。
- 不买接口不够的电源。
- 不为了跑分牺牲稳定性。
- 不只看一个平台的价格，要看整机总价。
- 不相信“i9 级”“军工级”“电竞级”这种空泛宣传。
