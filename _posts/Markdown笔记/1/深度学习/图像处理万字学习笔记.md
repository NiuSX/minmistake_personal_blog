## 一、图像处理是什么

### 1. 基本定义

图像处理是指对图像进行获取、变换、增强、恢复、分析和理解前的各种操作。它的目标可以是改善图像质量，也可以是提取有用信息，或者为后续计算机视觉模型提供更好的输入。

典型操作包括：

- 调整亮度和对比度。
- 去噪。
- 锐化。
- 边缘检测。
- 阈值分割。
- 形态学处理。
- 几何变换。
- 图像配准。
- 特征提取。
- 目标区域提取。

图像处理通常偏底层和中层视觉，而目标检测、图像分类、语义分割等属于更高层的计算机视觉任务。

### 2. 图像处理与计算机视觉的关系

图像处理关注“如何改变或分析图像本身”，计算机视觉关注“如何从图像中理解场景和对象”。

例如：

- 图像去噪是图像处理。
- 判断图片里是否有猫是计算机视觉。
- 提取边缘是图像处理。
- 检测车辆位置是计算机视觉。

二者关系密切。很多传统视觉系统先做图像处理，再做特征提取和分类。深度学习兴起后，很多步骤由神经网络自动学习，但图像处理仍然在数据清洗、预处理、增强、后处理和工业应用中非常重要。

### 3. 图像处理的典型应用

- 医学影像增强。
- 工业缺陷检测。
- 文档扫描矫正。
- OCR 前处理。
- 遥感图像分析。
- 安防监控。
- 人脸图像预处理。
- 自动驾驶视觉。
- 图像修复。
- 摄影后期。
- 机器人视觉。

### 4. 学图像处理的价值

即使现在深度学习非常强，图像处理仍然值得学习。原因是：

1. 它帮助理解图像数据本质。
2. 它能快速解决很多工程问题。
3. 它是调试视觉模型输入输出的重要工具。
4. 它在小数据、规则明确场景中仍很有效。
5. 它能提升对 CNN、检测、分割模型的理解。

## 二、数字图像基础

### 1. 图像的矩阵表示

数字图像本质上是一个矩阵。

灰度图：

```text
H x W
```

彩色图：

```text
H x W x C
```

其中：

- H 是高度。
- W 是宽度。
- C 是通道数。

灰度图每个像素只有一个亮度值。RGB 图像每个像素有红、绿、蓝三个值。

### 2. 像素值

常见 8 位图像像素范围：

```text
0 到 255
```

其中：

- 0 表示黑。
- 255 表示白。

彩色 RGB 图像中，每个通道都有 0 到 255 的值。

例如：

```text
红色:   (255, 0, 0)
绿色:   (0, 255, 0)
蓝色:   (0, 0, 255)
白色:   (255, 255, 255)
黑色:   (0, 0, 0)
```

### 3. 图像坐标系

图像坐标通常以左上角为原点：

```text
x 向右增加
y 向下增加
```

像素坐标常写为：

```text
(x, y)
```

数组索引常写为：

```text
image[y, x]
```

这是初学者常犯错的地方：数组第一维是行，也就是 y；第二维是列，也就是 x。

### 4. 通道顺序

OpenCV 默认使用 BGR：

```text
B, G, R
```

PIL、matplotlib 通常使用 RGB：

```text
R, G, B
```

如果通道顺序弄错，图像颜色会明显异常。

### 5. 图像数据类型

常见类型：

- `uint8`：0 到 255。
- `float32`：可表示 0 到 1 或其他范围。
- `uint16`：医学和遥感图像常见。

运算时要注意溢出。例如 `uint8` 中：

```text
250 + 10
```

可能溢出，不一定得到 260。实际处理中常转换为 float 再计算。

## 三、像素、分辨率与采样量化

### 1. 采样

采样是把连续空间图像离散成像素网格。

采样越密，分辨率越高，能保留更多细节。

### 2. 量化

量化是把连续亮度值离散成有限级别。

8 位图像有：

```text
2^8 = 256
```

个灰度级。

量化级别越高，灰度变化越细腻，但文件更大。

### 3. 分辨率

分辨率通常表示像素数量：

```text
1920 x 1080
```

分辨率越高，图像细节越多，但计算量和存储也更大。

### 4. 空间分辨率与灰度分辨率

空间分辨率：像素网格密度。

灰度分辨率：像素值可表示的等级数量。

两者都影响图像质量。

### 5. 混叠

采样频率不足会产生混叠。图像中表现为锯齿、摩尔纹等。

下采样前常使用低通滤波，减少高频信息造成的混叠。

## 四、颜色空间

### 1. RGB

RGB 是最常见颜色空间，用红、绿、蓝三通道表示颜色。

适合显示设备和深度学习模型输入。

缺点是亮度和颜色信息混合在一起，不总适合颜色分析。

### 2. BGR

BGR 与 RGB 通道顺序相反。OpenCV 默认 BGR。

在 OpenCV 和 matplotlib 混用时要特别注意转换。

### 3. 灰度图

灰度图只有亮度信息。常见转换公式：

```text
Gray = 0.299R + 0.587G + 0.114B
```

人眼对绿色更敏感，因此 G 权重更高。

### 4. HSV

HSV 包括：

- H：Hue，色相。
- S：Saturation，饱和度。
- V：Value，亮度。

HSV 适合按颜色分割目标，例如提取红色、蓝色、绿色物体。

### 5. HSL

HSL 与 HSV 类似，包括：

- Hue。
- Saturation。
- Lightness。

常用于图像编辑和颜色调整。

### 6. YUV / YCbCr

Y 表示亮度，U/V 或 Cb/Cr 表示色差信息。

常用于视频编码和压缩，因为人眼对亮度更敏感，对色度不那么敏感。

### 7. Lab

Lab 颜色空间试图更符合人眼感知。

包括：

- L：亮度。
- a：绿到红。
- b：蓝到黄。

适合颜色差异计算、颜色校正等。

### 8. 颜色空间选择

不同任务适合不同颜色空间：

- 显示和深度学习输入：RGB。
- OpenCV 处理：BGR。
- 颜色阈值分割：HSV。
- 亮度处理：YUV/Lab。
- 灰度结构分析：Gray。

## 五、图像直方图与灰度变换

### 1. 图像直方图

直方图统计每个灰度值出现的次数。

灰度图直方图横轴是灰度值，纵轴是像素数量。

直方图可以反映：

- 图像偏暗还是偏亮。
- 对比度是否低。
- 灰度分布是否集中。
- 是否存在过曝或欠曝。

### 2. 亮度调整

最简单亮度调整：

```text
g(x,y) = f(x,y) + b
```

`b > 0` 变亮，`b < 0` 变暗。

需要注意像素值裁剪到合法范围。

### 3. 对比度调整

线性对比度调整：

```text
g(x,y) = a * f(x,y) + b
```

`a > 1` 提高对比度，`0 < a < 1` 降低对比度。

### 4. Gamma 变换

Gamma 变换：

```text
s = c * r^gamma
```

其中 r 通常归一化到 0 到 1。

效果：

- gamma < 1：提升暗部。
- gamma > 1：压暗图像。

Gamma 变换常用于亮度校正。

### 5. 直方图均衡化

直方图均衡化通过重新分布灰度值，增强对比度。

适合低对比度灰度图。

缺点：

- 可能过度增强噪声。
- 彩色图像直接对 RGB 分别均衡可能颜色失真。

### 6. CLAHE

CLAHE 是限制对比度自适应直方图均衡化。

它在局部区域做均衡，并限制增强幅度，减少噪声放大。

常用于医学影像、低光照增强等。

## 六、卷积与空间滤波

### 1. 空间滤波

空间滤波是在图像空间中对邻域像素进行计算。典型方式是卷积。

输出像素由输入像素邻域加权求和得到。

### 2. 卷积核

卷积核是一个小矩阵，例如：

```text
1 1 1
1 1 1
1 1 1
```

不同卷积核产生不同效果：

- 平滑。
- 锐化。
- 边缘检测。
- 模糊。

### 3. 均值滤波核

`3x3` 均值滤波：

```text
1/9 1/9 1/9
1/9 1/9 1/9
1/9 1/9 1/9
```

它用邻域平均值替代中心像素，可以平滑噪声。

### 4. 边界处理

卷积时图像边缘邻域不完整，需要边界处理：

- zero padding。
- replicate。
- reflect。
- wrap。

不同边界处理会影响边缘结果。

### 5. 相关与卷积

严格卷积会翻转卷积核。很多图像处理库实现的是相关运算，但习惯上也称卷积。

对于对称卷积核，卷积和相关结果相同。

## 七、图像平滑与去噪

### 1. 噪声类型

常见噪声：

- 高斯噪声。
- 椒盐噪声。
- 泊松噪声。
- 斑点噪声。
- 压缩噪声。

不同噪声适合不同去噪方法。

### 2. 均值滤波

均值滤波用邻域平均值替代中心像素。

优点：

- 简单。
- 速度快。

缺点：

- 会模糊边缘。
- 对椒盐噪声效果一般。

### 3. 高斯滤波

高斯滤波使用高斯权重，中心像素权重大，远处权重小。

公式：

```text
G(x,y) = 1/(2πσ^2) * exp(-(x^2+y^2)/(2σ^2))
```

`sigma` 越大，模糊越强。

高斯滤波常用于边缘检测前的降噪。

### 4. 中值滤波

中值滤波取邻域像素中位数。

特别适合椒盐噪声。

优点：

- 保边能力比均值滤波好。
- 对孤立异常点鲁棒。

### 5. 双边滤波

双边滤波同时考虑空间距离和像素值相似度。

它可以在平滑噪声的同时保留边缘。

缺点是计算较慢。

### 6. 非局部均值去噪

非局部均值利用图像中相似 patch 的重复性进行去噪。

适合自然图像，但计算成本较高。

## 八、图像锐化与边缘检测

### 1. 锐化

锐化增强图像边缘和细节。

常用方法：

- 拉普拉斯锐化。
- Unsharp Masking。
- 高频增强。

### 2. 拉普拉斯算子

常见拉普拉斯核：

```text
0 -1 0
-1 4 -1
0 -1 0
```

或：

```text
-1 -1 -1
-1 8 -1
-1 -1 -1
```

拉普拉斯是二阶导数，对噪声敏感，通常先平滑。

### 3. Sobel 算子

Sobel 用于检测水平和垂直方向梯度。

水平方向：

```text
-1 0 1
-2 0 2
-1 0 1
```

垂直方向：

```text
-1 -2 -1
0 0 0
1 2 1
```

梯度幅值：

```text
G = sqrt(Gx^2 + Gy^2)
```

### 4. Scharr 算子

Scharr 是 Sobel 的改进版本，在小核情况下旋转对称性更好。

### 5. Canny 边缘检测

Canny 是经典边缘检测算法，步骤：

1. 高斯滤波降噪。
2. 计算梯度幅值和方向。
3. 非极大值抑制。
4. 双阈值检测。
5. 边缘连接。

Canny 效果通常比简单 Sobel 更干净。

### 6. 边缘检测的应用

- 文档边界检测。
- 工业缺陷检测。
- 目标轮廓提取。
- 图像配准。
- 传统特征提取。

## 九、阈值分割

### 1. 全局阈值

最简单分割：

```text
if pixel > T:
    foreground
else:
    background
```

适合前景和背景灰度差异明显的图像。

### 2. Otsu 阈值

Otsu 方法自动选择阈值，使类间方差最大。

适合双峰直方图场景。

### 3. 自适应阈值

自适应阈值根据局部区域计算阈值，适合光照不均的图像。

常用于文档二值化。

### 4. 多阈值分割

如果图像包含多个灰度区域，可以使用多个阈值分成多个类别。

### 5. 阈值分割局限

阈值方法依赖灰度或颜色差异，对复杂背景、阴影、噪声敏感。

## 十、形态学处理

### 1. 基本概念

形态学处理主要用于二值图像，也可扩展到灰度图像。它通过结构元素处理图像形状。

常见操作：

- 腐蚀。
- 膨胀。
- 开运算。
- 闭运算。
- 梯度。
- 顶帽。
- 黑帽。

### 2. 腐蚀

腐蚀会让前景区域变小。

作用：

- 去除小白点。
- 分离粘连物体。
- 缩小边界。

### 3. 膨胀

膨胀会让前景区域变大。

作用：

- 填补小空洞。
- 连接断裂区域。
- 扩大边界。

### 4. 开运算

开运算：

```text
先腐蚀，后膨胀
```

作用：

- 去除小噪声。
- 保持主体形状。

### 5. 闭运算

闭运算：

```text
先膨胀，后腐蚀
```

作用：

- 填补小孔洞。
- 连接小裂缝。

### 6. 形态学梯度

形态学梯度：

```text
膨胀图 - 腐蚀图
```

可以提取边界。

### 7. 顶帽与黑帽

顶帽：

```text
原图 - 开运算
```

用于提取亮小目标。

黑帽：

```text
闭运算 - 原图
```

用于提取暗小结构。

## 十一、几何变换

### 1. 平移

平移将图像整体移动。

矩阵：

```text
[1 0 tx]
[0 1 ty]
```

### 2. 缩放

缩放改变图像尺寸。

需要选择插值方法：

- 最近邻。
- 双线性。
- 双三次。
- Area。

### 3. 旋转

旋转通常围绕图像中心。

旋转后可能出现黑边或裁剪，需要调整输出画布。

### 4. 仿射变换

仿射变换保持直线和平行关系。包括：

- 平移。
- 旋转。
- 缩放。
- 剪切。

需要 3 对点确定二维仿射变换。

### 5. 透视变换

透视变换可以处理拍摄角度造成的梯形畸变。

需要 4 对点确定。

应用：

- 文档矫正。
- 车道线鸟瞰图。
- 平面目标校正。

### 6. 图像配准

图像配准是将多张图像对齐。

常用于：

- 医学影像。
- 遥感。
- 全景拼接。
- 多时相图像分析。

## 十二、频域图像处理

### 1. 空间域与频域

空间域直接处理像素。

频域把图像表示为不同频率成分的组合。

低频表示：

- 平滑区域。
- 缓慢变化。

高频表示：

- 边缘。
- 细节。
- 噪声。

### 2. 傅里叶变换

傅里叶变换把图像从空间域转换到频域。

二维图像频谱可以显示不同频率成分强度。

### 3. 低通滤波

保留低频，抑制高频。

效果：

- 平滑。
- 去噪。
- 模糊。

### 4. 高通滤波

保留高频，抑制低频。

效果：

- 边缘增强。
- 细节增强。
- 锐化。

### 5. 带通与陷波滤波

带通保留某一频率范围。

陷波滤波用于去除特定频率噪声，例如周期条纹噪声。

### 6. 频域处理应用

- 周期噪声去除。
- 图像压缩。
- 纹理分析。
- 锐化和平滑。

## 十三、图像增强与图像复原

### 1. 图像增强

图像增强目标是让图像更适合人看或机器处理。

方法：

- 亮度调整。
- 对比度增强。
- 直方图均衡。
- 去雾。
- 去噪。
- 锐化。

增强不一定追求真实恢复，只要服务任务。

### 2. 图像复原

图像复原目标是根据退化模型恢复原始图像。

退化可能来自：

- 模糊。
- 噪声。
- 运动抖动。
- 失焦。
- 压缩损伤。

复原比增强更强调物理模型。

### 3. 图像去模糊

去模糊需要估计模糊核或使用学习方法。

传统方法：

- 维纳滤波。
- 反卷积。

### 4. 图像去雾

传统去雾常用暗通道先验。现代方法也大量使用深度学习。

### 5. 超分辨率

超分辨率把低分辨率图像恢复为高分辨率图像。

传统插值包括：

- 最近邻。
- 双线性。
- 双三次。

深度学习方法可以生成更细节的结果，但可能产生幻觉。

## 十四、传统特征提取

### 1. 为什么需要特征

深度学习之前，图像识别依赖人工设计特征。

传统流程：

```text
预处理 -> 特征提取 -> 分类器
```

### 2. Harris 角点

Harris 用于检测角点。角点在多个方向上灰度变化都明显。

应用：

- 图像匹配。
- 跟踪。
- 配准。

### 3. SIFT

SIFT 是经典局部特征，具有尺度和旋转不变性。

步骤包括：

- 尺度空间极值检测。
- 关键点定位。
- 方向分配。
- 描述子生成。

### 4. SURF

SURF 是 SIFT 的加速版本。

### 5. ORB

ORB 是一种高效特征，常用于实时场景。

优点：

- 快。
- 免费可用。
- 适合匹配。

### 6. HOG

HOG 是方向梯度直方图，常用于行人检测。

它统计局部区域梯度方向分布，描述形状结构。

### 7. LBP

LBP 是局部二值模式，常用于纹理分析和人脸识别早期方法。

## 十五、图像分割基础

### 1. 分割是什么

图像分割是把图像划分为有意义区域。

传统分割方法包括：

- 阈值分割。
- 边缘分割。
- 区域生长。
- 分水岭。
- 聚类。
- 图割。

### 2. 区域生长

从种子点开始，将相似邻域加入区域。

需要定义相似性和停止条件。

### 3. 分水岭算法

把图像看成地形，灰度值表示高度，从低处开始灌水，不同水域交界形成分割边界。

常用于粘连物体分割。

### 4. GrabCut

GrabCut 根据用户给定粗略框，使用图割方法分离前景背景。

### 5. 聚类分割

例如 K-means 根据颜色或特征把像素聚成不同类别。

### 6. 传统分割与深度分割

传统分割依赖颜色、边缘和灰度规则。深度分割模型如 U-Net、DeepLab、Mask R-CNN 能利用语义信息，适合复杂场景。

## 十八、常见误区

### 1. 图像处理已经过时

深度学习很强，但图像处理仍然是数据预处理、增强、后处理和工程排查的重要基础。

### 2. 滤波越强越好

强滤波会去掉噪声，也会损失细节和边缘。

### 3. 直方图均衡化一定提升效果

它可能增强对比度，也可能放大噪声或造成颜色失真。

### 4. 边缘检测可以直接完成识别

边缘只是低级特征，复杂识别还需要形状、纹理、语义和上下文。

### 5. RGB 和 BGR 无所谓

通道顺序错误会导致颜色严重偏差，影响可视化和模型输入。

### 6. 二值化阈值可以固定不变

光照变化时固定阈值可能失效，需要自适应阈值或光照归一化。

### 7. Mask resize 可以用双线性

类别 mask 必须用最近邻插值，否则会产生非法类别值。

## 十九、面试高频问题

### 1. 什么是数字图像？

数字图像是对连续图像经过采样和量化后得到的二维或三维矩阵。灰度图是二维矩阵，彩色图通常是三通道矩阵。

### 2. RGB 和 HSV 有什么区别？

RGB 用红绿蓝表示颜色，适合显示；HSV 用色相、饱和度、亮度表示，更适合基于颜色的分割和分析。

### 3. 什么是直方图均衡化？

直方图均衡化通过重新分布灰度值，扩大灰度动态范围，从而提升图像对比度。

### 4. 均值滤波和中值滤波区别？

均值滤波取邻域平均，适合平滑高斯噪声但会模糊边缘；中值滤波取邻域中位数，对椒盐噪声效果好且较能保留边缘。

### 5. 高斯滤波的 sigma 有什么作用？

sigma 控制高斯核宽度。sigma 越大，平滑越强，图像越模糊。

### 6. Sobel 和 Canny 区别？

Sobel 计算图像梯度，得到边缘响应；Canny 是完整边缘检测流程，包括平滑、梯度、非极大值抑制、双阈值和边缘连接，结果通常更细更干净。

### 7. 腐蚀和膨胀有什么作用？

腐蚀缩小前景，可去除小噪声；膨胀扩大前景，可填补小孔和连接断裂区域。

### 8. 开运算和闭运算区别？

开运算先腐蚀后膨胀，常用于去除小噪声；闭运算先膨胀后腐蚀，常用于填补小孔洞。

### 9. 仿射变换和透视变换区别？

仿射变换保持平行关系，需要 3 对点确定；透视变换不一定保持平行关系，可处理视角变化，需要 4 对点确定。

### 10. 低通滤波和高通滤波区别？

低通滤波保留低频，效果是平滑和去噪；高通滤波保留高频，效果是增强边缘和细节。

### 11. 为什么图像下采样前要低通滤波？

为了减少高频成分造成的混叠，避免下采样后出现锯齿和伪影。

### 12. 图像处理和计算机视觉有什么区别？

图像处理侧重对图像进行变换、增强和分析；计算机视觉侧重从图像中理解对象、场景和语义。

5. 是否需要颜色信息？
6. 后续任务是给人看还是给模型用？

理解图像和任务，再选择合适方法，通常比堆叠复杂算法更有效。
