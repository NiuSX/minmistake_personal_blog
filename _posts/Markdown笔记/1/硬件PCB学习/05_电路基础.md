# 05 电路基础

## 学习目标

学完本章，你应该能：

- 理解电压、电流、电阻、功率的关系。
- 看懂串联、并联、分压、限流等基础电路。
- 理解电容、电感、二极管、三极管、MOSFET 的基础作用。
- 能搭建并分析简单低压电路。

电路基础是硬件学习的根。PCB 只是电路的物理实现。如果不理解电路，PCB 很容易变成“照着连线”，遇到问题也无法排查。

## 1. 电路的基本条件

一个电路要工作，通常需要：

- 电源
- 负载
- 闭合回路
- 合适的电压和电流

例如 LED 电路：

```text
5V -> 限流电阻 -> LED -> GND
```

电流从电源正极经过电阻和 LED，最后回到电源负极，形成闭合回路。

## 2. 电压

电压可以理解为推动电流流动的压力。

常见电压：

- 1.8V：低功耗数字芯片
- 3.3V：MCU、传感器常用
- 5V：USB、Arduino、部分模块
- 12V：电机、灯带、工业小系统
- 24V：工业控制常用

设计时要确认：

- 芯片供电范围
- IO 电平范围
- 接口电平是否兼容
- 保护器件工作电压

## 3. 电流

电流表示电荷流动大小。

设计时关注：

- 芯片典型电流
- 最大电流
- 启动电流
- 峰值电流
- 睡眠电流

例如 Wi-Fi 模块可能平均电流不高，但发射瞬间峰值很大。如果电源设计只看平均值，模块可能复位。

## 4. 电阻

电阻用于限制电流和分配电压。

常见用途：

- LED 限流
- 分压
- 上拉
- 下拉
- 电流采样
- 信号匹配

### LED 限流

公式：

```text
R = (Vcc - Vf) / I
```

其中：

- Vcc 是电源电压。
- Vf 是 LED 正向压降。
- I 是目标电流。

例子：

```text
Vcc = 5V
Vf = 2V
I = 10mA
R = 300Ω
```

实际可选 330Ω。

## 5. 串联电路

串联特点：

- 电流相同
- 电压分配
- 总电阻相加

公式：

```text
Rtotal = R1 + R2 + R3
```

应用：

- LED 限流
- 电阻分压
- 电池串联提高电压

## 6. 并联电路

并联特点：

- 电压相同
- 电流分配
- 总电阻变小

两个电阻并联：

```text
Rtotal = R1 x R2 / (R1 + R2)
```

应用：

- 电容并联增加容量
- 电源去耦多个电容并联
- 多负载并接电源

## 7. 分压电路

分压公式：

```text
Vout = Vin x R2 / (R1 + R2)
```

用途：

- 电池电压检测
- ADC 输入缩放
- 参考电压生成

注意：

分压电路输出阻抗不能太高，否则容易受 ADC 输入、噪声和漏电影响。

## 8. 电容基础

电容特点：

- 隔直流，通交流的趋势。
- 抵抗电压突变。
- 可以储能。

常见用途：

- 去耦
- 滤波
- 储能
- 耦合
- 延时

### 去耦电容

芯片电源脚附近常放 100nF 电容。

作用：

- 提供瞬态电流。
- 降低电源噪声。
- 让芯片供电更稳定。

## 9. 电感基础

电感特点：

- 抵抗电流突变。
- 储存磁场能量。

用途：

- DC-DC 电源
- LC 滤波
- EMI 抑制

注意：

电感有饱和电流。超过饱和电流后，电感量会下降，电路可能失控。

## 10. 二极管基础

二极管单向导通。

常见用途：

- 整流
- 防反接
- 续流
- 钳位
- ESD 保护

常见类型：

- 普通二极管
- 肖特基二极管
- 稳压二极管
- TVS
- LED

### 续流二极管

感性负载如继电器、电机断电时会产生反向高压。续流二极管给电流提供释放路径，保护驱动器件。

## 11. 三极管基础

三极管类型：

- NPN
- PNP

常见用途：

- 小信号放大
- 开关控制
- 电平转换

新手常用 NPN 低端开关：

```text
MCU GPIO -> 基极电阻 -> NPN
负载 -> NPN -> GND
```

注意：

基极需要限流电阻。

## 12. MOSFET 基础

MOSFET 常用于开关。

类型：

- NMOS
- PMOS

常见参数：

- Vds：漏源耐压
- Id：最大电流
- Rds(on)：导通电阻
- Vgs(th)：阈值电压
- Qg：栅极电荷

注意：

`Vgs(th)` 不是完全导通电压。用 3.3V MCU 控制负载时，要选逻辑电平 MOSFET。

## 13. 开关和按键

机械按键会有抖动。

处理方法：

- 软件消抖
- RC 硬件消抖
- 施密特触发器

按键输入常见：

```text
GPIO -> 按键 -> GND
GPIO 通过电阻上拉到 VCC
```

按下为低，松开为高。

## 14. 基础电路模块

入门需要掌握这些小模块：

- LED 限流
- 按键输入
- 分压测电压
- NPN 控制蜂鸣器
- MOSFET 控制负载
- LDO 稳压
- RC 低通
- TVS 接口保护

## 实操练习

1. 计算并搭建一个 5V LED 限流电路。
2. 用两个电阻搭分压，把 5V 分到约 3.3V。
3. 用按键和上拉电阻做一个输入电路。
4. 用 NPN 或 MOSFET 控制一个小蜂鸣器或 LED 条。
5. 用电容观察按键消抖或 RC 延时效果。

## 检查清单

- 我能否解释电流必须形成闭合回路？
- 我能否计算 LED 限流电阻？
- 我能否区分串联和并联？
- 我能否解释上拉和下拉？
- 我是否知道感性负载需要续流保护？
- 我是否知道 MOSFET 阈值电压不等于完全导通？

## 常见误区

- 误区：LED 可以直接接电源。
  纠正：需要限流，否则可能烧毁。

- 误区：GPIO 可以直接驱动任何负载。
  纠正：GPIO 电流有限，大负载需要驱动器件。

- 误区：电容越大越好。
  纠正：电容要按频率、稳定性和电源需求选择。

## 本章总结

电路基础看似简单，但直接决定硬件设计质量。先熟练掌握电阻、电容、二极管、三极管、MOSFET 和基础模块，再进入复杂原理图和 PCB，会稳很多。
