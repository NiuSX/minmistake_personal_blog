# 08 常用工具与仪器

## 学习目标

学完本章，你应该能：

- 知道硬件学习需要哪些软件、工具和仪器。
- 理解万用表、限流电源、示波器、逻辑分析仪分别解决什么问题。
- 建立基础实验台配置。
- 知道工具使用中的安全注意事项。

工具不是越贵越好。新手最重要的是会用基础工具，并建立正确测量习惯。

## 1. EDA 软件

EDA 是 Electronic Design Automation，用于原理图和 PCB 设计。

| 软件 | 特点 | 适合 |
| :--- | :--- | :--- |
| KiCad | 开源、跨平台、功能完整 | 新手和开源项目 |
| 立创 EDA | 在线、器件库和打样结合方便 | 国内快速打样 |
| Altium Designer | 商业专业工具 | 企业和复杂项目 |
| EasyEDA | 在线工具 | 入门和简单项目 |
| Eagle | 历史较久 | 一些开源项目 |

新手建议：

- 想系统学习：KiCad。
- 想快速打样贴片：立创 EDA。
- 工作要求 Altium：再学 Altium。

## 2. 仿真软件

仿真不是必须，但对理解电路很有帮助。

| 工具 | 用途 |
| :--- | :--- |
| Falstad | 在线直观电路仿真，适合入门 |
| LTspice | 模拟电路、电源仿真 |
| TINA-TI | TI 电路仿真 |
| QSPICE | 电路仿真 |

适合仿真的内容：

- RC 滤波
- 运放电路
- 电源瞬态
- MOSFET 开关
- 简单波形

注意：

仿真结果依赖模型和设置。仿真通过不代表实物一定没问题。

## 3. 万用表

万用表是硬件必备工具。

常用功能：

- 测直流电压
- 测交流电压
- 测电阻
- 通断蜂鸣
- 测电流
- 测二极管压降

### 使用注意

测电压：

- 表笔并联在被测点两端。
- 黑表笔接 GND，红表笔接信号或电源。

测电流：

- 万用表要串联进电路。
- 表笔插孔要换到电流档。
- 不要用电流档直接并联电源，否则相当于短路。

通断档：

- 只能在断电状态下使用。
- 用于检查短路和连接。

## 4. 可调限流电源

作用：

- 设置输出电压。
- 设置最大电流。
- 观察实时电流。
- 安全第一次上电。

关键概念：

- CV：恒压模式。
- CC：恒流模式。

如果电路短路，电源会进入 CC 限流模式，电压下降，电流被限制。

新手建议：

- 第一次上电使用限流。
- 电流异常立即断电。
- 不要直接用大电流适配器首上电。

## 5. 烙铁和焊接工具

基础焊接工具：

- 温控烙铁
- 烙铁架
- 焊锡丝
- 助焊剂
- 吸锡带
- 吸锡器
- 镊子
- 放大镜或显微镜
- 洗板水或酒精

温度建议：

- 有铅焊锡：约 300-330°C
- 无铅焊锡：约 330-370°C

具体温度和焊盘大小、烙铁头、焊锡有关。

## 6. 示波器

示波器用来看电压随时间变化。

适合观察：

- 电源纹波
- 时钟
- PWM
- 复位信号
- UART 波形
- 开关电源 SW 节点
- 按键抖动

关键参数：

- 带宽
- 采样率
- 通道数
- 存储深度
- 探头倍率

新手常用设置：

- 探头 10x。
- 先夹好地线。
- 根据波形调整时间档和电压档。
- 使用触发稳定波形。

注意：

普通台式示波器地夹通常接大地。测非隔离市电电路非常危险，新手不要做。

## 7. 逻辑分析仪

逻辑分析仪用于看数字信号高低电平和通信协议。

适合：

- UART
- I2C
- SPI
- 1-Wire
- GPIO 时序

优点：

- 便宜。
- 多通道。
- 可以协议解码。

缺点：

- 看不到模拟电压细节。
- 只能判断高低电平。

新手调 I2C、SPI、UART 时非常有用。

## 8. 函数发生器

函数发生器可以输出测试信号：

- 正弦波
- 方波
- 三角波
- 脉冲

用途：

- 测滤波器
- 测运放
- 测 ADC 输入
- 模拟传感器信号

入门阶段不是必需，有示波器后再考虑。

## 9. 电子负载

电子负载用于模拟负载消耗电流。

用途：

- 测电源输出能力
- 测 DC-DC 稳定性
- 测电池放电
- 测温升

如果学习电源设计，电子负载很有用。

## 10. 热工具

常见：

- 热风枪
- 加热台
- 热像仪
- 温度计

用途：

- 拆焊贴片 IC。
- 焊接 QFN。
- 观察发热点。
- 评估散热。

热风枪使用要小心，容易吹飞小元件或烤坏塑料连接器。

## 11. 新手基础实验台

最低配置：

- 万用表
- 可调限流电源
- 温控烙铁
- 镊子
- 助焊剂
- 吸锡带
- 面包板和杜邦线
- 常用电阻电容包
- LED、按键、三极管、MOSFET

推荐进阶：

- 入门示波器
- 简易逻辑分析仪
- 放大镜或显微镜
- 热风枪

## 实操练习

1. 用万用表测 5V 电源输出。
2. 用通断档检查一根杜邦线。
3. 设置限流电源 5V / 100mA 点亮 LED。
4. 用示波器观察 Arduino 或 MCU 输出的 PWM。
5. 用逻辑分析仪抓 UART 输出。

## 检查清单

- 我是否会正确测电压？
- 我是否知道测电流要串联？
- 我是否会设置限流电源？
- 我是否知道示波器地夹的风险？
- 我是否能用逻辑分析仪看 UART/I2C/SPI？

## 常见误区

- 误区：万用表电流档和电压档一样用。
  纠正：电流档要串联，错误并联会短路。

- 误区：没有示波器就不能学硬件。
  纠正：入门可以先用万用表和逻辑分析仪，但示波器迟早要学。

- 误区：贵工具自动解决问题。
  纠正：关键是知道测哪里、为什么测。

## 本章总结

工具是硬件学习的延伸感官。万用表看静态值，示波器看波形，逻辑分析仪看数字协议，限流电源保护上电安全。先把基础工具用熟，比买很多高级设备更重要。
