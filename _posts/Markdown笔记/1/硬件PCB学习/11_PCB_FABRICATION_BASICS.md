# 11 PCB 制造工艺基础

## 学习目标

学完本章，你应该能：

- 理解 PCB 是如何被制造出来的。
- 看懂板厂下单页面上的常见参数。
- 知道板厚、铜厚、层数、阻焊、表面处理、线宽线距的意义。
- 在设计前设置合理工艺规则。

PCB 设计不能脱离制造工艺。你在软件里能画出来，不代表板厂能生产；板厂能生产，也不代表成本合理。

## 1. PCB 制造的基本过程

简化流程：

```text
基材准备
  -> 内外层图形转移
  -> 蚀刻铜箔
  -> 钻孔
  -> 孔金属化
  -> 阻焊
  -> 丝印
  -> 表面处理
  -> 成型
  -> 电测
  -> 包装
```

你不需要一开始掌握所有工艺细节，但要知道设计规则和这些工艺直接相关。

## 2. 板材

常见板材：

| 板材 | 特点 | 使用场景 |
| :--- | :--- | :--- |
| FR-4 | 最常见玻纤板，成本低 | 普通电子产品 |
| 铝基板 | 散热好 | LED、大功率 |
| Rogers 等高频板 | 高频性能好 | 射频、高速 |
| FPC | 柔性可弯折 | 排线、可穿戴 |

新手默认使用 FR-4。

## 3. 板厚

常见板厚：

- 0.6mm
- 0.8mm
- 1.0mm
- 1.2mm
- 1.6mm
- 2.0mm

默认常用 1.6mm。

选择板厚考虑：

- 机械强度
- 外壳空间
- 连接器要求
- 插拔强度
- 阻抗控制
- 成本

例如 USB 金手指、板边连接器可能对板厚有明确要求。

## 4. 铜厚

铜厚通常用 oz 表示。

| 铜厚 | 近似厚度 | 特点 |
| :--- | :--- | :--- |
| 0.5oz | 约 17um | 细线容易，载流较小 |
| 1oz | 约 35um | 最常见 |
| 2oz | 约 70um | 大电流更好，但加工成本更高 |

大电流板可以选择厚铜，但线宽、间距、焊接、成本都要考虑。

## 5. 层数

常见：

- 单层板
- 双层板
- 四层板
- 六层板
- 八层及以上

新手建议：

- 从双层板开始。
- 有较复杂 MCU、USB、无线、电源时考虑四层。
- 不要一开始做多层高速板。

## 6. 最小线宽线距

线宽：铜线宽度。

线距：铜线之间距离。

板厂常规工艺通常比极限工艺便宜可靠。

新手建议：

- 不要用板厂最小线宽线距作为默认设计。
- 留足余量，例如普通信号使用 0.20mm 或 0.25mm 以上。
- 电源线按电流加宽。

## 7. 孔径和孔环

孔径：

- 钻孔大小。

孔环：

- 孔周围保留的铜环。

注意：

- 过孔孔径太小会增加成本。
- 通孔元件孔径要适合元件引脚。
- 孔环太小可能制造偏差后断环。

## 8. 阻焊层

阻焊是覆盖铜皮的绝缘油墨。

作用：

- 防止焊锡乱流。
- 防止铜氧化。
- 降低短路风险。
- 提供板子颜色。

常见颜色：

- 绿色
- 蓝色
- 红色
- 黑色
- 白色
- 紫色

注意：

颜色主要是外观，某些颜色丝印可读性较差。黑色板不容易目检细节，白色板看助焊剂残留不明显。

## 9. 丝印层

丝印用于文字和标记。

应该标：

- 位号
- 接口名称
- 电源极性
- 按键功能
- LED 功能
- 测试点名称
- 版本号

注意：

- 丝印不要压焊盘。
- 字太小会看不清或生产不出来。
- 极性器件方向必须清楚。

## 10. 表面处理

常见表面处理：

| 工艺 | 特点 | 使用场景 |
| :--- | :--- | :--- |
| HASL 喷锡 | 便宜，表面不够平整 | 普通板 |
| 无铅喷锡 | 环保，常规选择 | 普通产品 |
| ENIG 沉金 | 表面平整，适合细间距 | QFN、细间距、较好品质 |
| OSP | 成本低，平整 | 批量产品 |

新手如果有细间距芯片，可以考虑沉金。

## 11. 电测

电测用于检查 PCB 制造后的开短路。

它能发现：

- 断线
- 短路

不能发现：

- 原理图设计错误
- 封装错误
- 布局问题
- 元件焊接问题

## 12. 拼板

拼板是把多个小 PCB 合在一张大板中生产和贴片。

常见方式：

- V-Cut
- 邮票孔
- 工艺边

新手单板打样时可以先不拼板。需要贴片或批量时再学习拼板。

## 13. 下单参数如何选

新手常规选择：

| 参数 | 建议 |
| :--- | :--- |
| 板材 | FR-4 |
| 层数 | 2 层 |
| 板厚 | 1.6mm |
| 铜厚 | 1oz |
| 阻焊 | 绿色 |
| 丝印 | 白色 |
| 表面处理 | 无铅喷锡或沉金 |
| 最小线宽线距 | 使用常规工艺 |

## 实操练习

1. 打开一个 PCB 打样网站，查看下单参数。
2. 对照自己的 PCB，确认线宽线距是否满足常规工艺。
3. 导出 Gerber 后用 Gerber Viewer 检查每一层。
4. 找一个连接器，确认板厚是否有要求。

## 检查清单

- 设计规则是否符合板厂工艺？
- 板厚是否适合外壳和连接器？
- 铜厚是否满足电流？
- 线宽线距是否留余量？
- 孔径是否适合元件？
- 丝印是否清楚且不压焊盘？
- 表面处理是否适合封装？

## 常见误区

- 误区：能画出来就能生产。
  纠正：必须符合板厂工艺规则。

- 误区：线越细越专业。
  纠正：线越细越难制造，可靠性和成本都受影响。

- 误区：PCB 颜色影响电气性能很大。
  纠正：普通低速板中颜色主要影响外观和可读性。

## 本章总结

PCB 制造工艺决定你的设计边界。新手用常规 FR-4、双层、1.6mm、1oz、常规线宽线距即可。不要追求极限工艺，先把板子做得可靠、好焊、好测。
