# 12 PCB 层叠结构

## 学习目标

学完本章，你应该能：

- 理解 PCB 层叠结构的意义。
- 区分双层板、四层板和多层板的适用场景。
- 知道地平面和电源平面对信号、电源、EMI 的影响。
- 能为简单项目选择合适层数。

层叠结构不是“板子有几层”这么简单。它直接影响回流路径、电源阻抗、信号完整性、EMI 和制造成本。

## 1. 什么是层叠

层叠 Stackup 指 PCB 中铜层和介质层的排列方式。

例如四层板：

```text
Top Signal
Dielectric
GND Plane
Dielectric
Power Plane
Dielectric
Bottom Signal
```

层叠决定：

- 哪些层走信号。
- 哪些层做地平面。
- 哪些层做电源平面。
- 信号到参考平面的距离。
- 阻抗控制能力。
- EMI 表现。

## 2. 单层板

单层板只有一层铜。

优点：

- 成本低。
- 制造简单。

缺点：

- 布线困难。
- 回流路径差。
- 不适合复杂电路。

适合：

- 极简单 LED 板。
- 低成本玩具电路。

新手可以了解，但不建议作为主要学习对象。双层板更实用。

## 3. 双层板

双层板有顶层和底层。

常见做法：

- 顶层走主要信号和元件。
- 底层尽量铺 GND。
- 必要时底层也走线。

优点：

- 成本低。
- 打样方便。
- 适合入门。

缺点：

- 地平面容易被走线切碎。
- 电源分配不如四层。
- 高速和 EMI 表现有限。

适合：

- LED 板
- 简单 MCU 板
- 传感器板
- 低速接口板

## 4. 双层板设计建议

推荐：

- 底层尽量保持完整 GND。
- 顶层走信号和电源。
- 必须在底层走线时，尽量短，避免切断关键回流。
- 多打 GND 过孔连接顶底层地。
- 关键芯片下方保持地参考。

错误：

- 底层被信号线切得支离破碎。
- 高速信号跨越地缝。
- GND 只用细线连接。
- 大电流地和敏感地混乱穿过。

## 5. 四层板

四层板是很多中等复杂项目的理想选择。

常见层叠：

```text
L1 Top Signal + Components
L2 GND Plane
L3 Power Plane / Signals
L4 Bottom Signal
```

或：

```text
L1 Top Signal
L2 GND Plane
L3 GND / Power
L4 Bottom Signal
```

优点：

- 有连续地平面。
- 回流路径好。
- 电源阻抗低。
- EMI 更容易控制。
- 布线更容易。

缺点：

- 成本高于双层。
- 需要理解层叠和参考平面。

适合：

- 带 USB 的 MCU 板。
- Wi-Fi / BLE 模块板。
- 较复杂传感器板。
- 小型产品板。
- 对稳定性要求更高的板。

## 6. 为什么 GND 平面重要

GND 平面提供：

- 低阻抗回流路径。
- 信号参考。
- 屏蔽。
- 散热。
- 降低 EMI。

高频信号的返回电流会倾向于沿信号线下方最近路径返回。如果地平面连续，回路面积小；如果地平面断裂，返回电流被迫绕路，噪声和辐射增加。

## 7. 电源平面

电源平面用于分配电源。

优点：

- 阻抗低。
- 压降低。
- 可以和 GND 平面形成分布电容。
- 适合多芯片供电。

但电源平面不是必须完整一整层。小项目中可以局部铺电源铜皮。

## 8. 多层板

六层、八层及以上用于：

- 高速接口
- DDR
- BGA
- 高密度设计
- 多电源复杂系统
- 射频

多层板考虑：

- 信号参考平面
- 阻抗控制
- 电源平面分割
- 过孔结构
- 层间耦合
- 成本

新手先不要从多层高速板开始。

## 9. 层叠选择方法

| 项目类型 | 推荐层数 |
| :--- | :--- |
| LED、按键、简单传感器 | 2 层 |
| 普通 MCU 小板 | 2 层或 4 层 |
| USB、无线模块、较高稳定性 | 4 层 |
| DDR、PCIe、BGA | 6 层以上 |
| 射频产品 | 4 层以上，按射频要求 |

判断标准：

- 是否需要完整地平面？
- 是否有高速接口？
- 是否有无线或敏感模拟？
- 是否布线密度很高？
- 成本是否允许？

## 10. 常见层叠错误

- 双层板底层大量乱走线，地平面破碎。
- 四层板没有完整 GND 层。
- 高速信号参考电源平面但电源平面被切割。
- 电源平面分割导致回流路径断开。
- 差分线跨越参考平面变化。
- 不和板厂确认阻抗层叠。

## 实操练习

1. 打开一个双层 PCB，观察 GND 是否连续。
2. 在 KiCad 中尝试设置双层板和四层板。
3. 找一块开发板，观察它可能是几层板。
4. 画一个简单四层层叠方案，并说明每层用途。

## 检查清单

- 我是否知道为什么需要 GND 平面？
- 双层板是否尽量保持底层 GND 连续？
- 四层板是否有完整 GND 层？
- 高速信号是否有连续参考平面？
- 电源平面是否被合理规划？
- 层数选择是否符合项目需求和成本？

## 常见误区

- 误区：层数越多越专业。
  纠正：层数应由需求决定，简单项目双层足够。

- 误区：GND 铺铜只为了好看。
  纠正：GND 平面对回流、EMI 和稳定性很重要。

- 误区：四层板随便定义内层。
  纠正：至少要保证可靠参考平面。

## 本章总结

层叠结构决定 PCB 的电气基础。新手先用双层板学习，再在 USB、无线、复杂 MCU 项目中学习四层板。记住：完整连续的 GND 平面往往比多绕几根线更重要。
