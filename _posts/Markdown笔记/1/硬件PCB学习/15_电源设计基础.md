# 15 电源设计基础

## 学习目标

学完本章，你应该能：

- 理解电源树、电源预算和电源保护。
- 区分 LDO、Buck、Boost、Buck-Boost 的用途。
- 计算 LDO 功耗和发热风险。
- 知道 DC-DC 布局为什么重要。

电源是硬件系统的地基。很多看似通信、程序、芯片的问题，本质是电源不稳、纹波过大、瞬态电流不足或上电顺序错误。

## 1. 电源设计要解决什么

电源设计要保证：

- 电压正确。
- 电流足够。
- 纹波可接受。
- 瞬态响应足够。
- 发热可控。
- 上电顺序正确。
- 异常情况下有保护。

设计前要明确：

- 输入电源是什么。
- 系统需要几路电压。
- 每路电压给哪些模块供电。
- 每路电流典型值和最大值。

## 2. 电源树

电源树描述电源从输入到各模块的路径。

例子：

```text
USB 5V
  -> 5V_SYS
    -> LDO 3.3V
      -> MCU
      -> Sensor
      -> Pull-up
    -> LED Driver
```

电源树应包含：

- 电压名称。
- 转换芯片。
- 最大电流。
- 保护器件。
- 负载模块。

## 3. 电源预算

电源预算用于确认电源芯片是否够用。

示例：

| 模块 | 电压 | 典型电流 | 最大电流 |
| :--- | :--- | :--- | :--- |
| MCU | 3.3V | 30mA | 100mA |
| 传感器 | 3.3V | 5mA | 20mA |
| Wi-Fi | 3.3V | 120mA | 500mA |
| LED | 5V | 60mA | 300mA |

选择电源芯片时按最大电流并留余量。

## 4. LDO

LDO 是低压差线性稳压器。

优点：

- 电路简单。
- 外围少。
- 噪声低。
- 适合小电流。

缺点：

- 效率低。
- 压差和电流越大，发热越大。

功耗：

```text
P = (Vin - Vout) x Iout
```

例子：

```text
Vin = 5V
Vout = 3.3V
Iout = 300mA
P = 0.51W
```

0.51W 对小封装可能已经很热。

## 5. LDO 选型

看这些参数：

- 输入电压范围。
- 输出电压。
- 最大输出电流。
- Dropout 压差。
- 静态电流。
- 输出噪声。
- PSRR。
- 稳定所需输出电容。
- 封装热阻。

注意：

有些 LDO 对输出电容 ESR 有要求。必须按数据手册选电容。

## 6. Buck 降压

Buck 用于从高电压降到低电压。

例子：

- 12V -> 5V
- 5V -> 3.3V

优点：

- 效率高。
- 适合较大电流。

缺点：

- 有开关噪声。
- 外围复杂。
- PCB 布局敏感。

关键器件：

- 电感
- 输入电容
- 输出电容
- 二极管或同步 MOSFET
- 反馈电阻

## 7. Boost 升压

Boost 用于升压。

例子：

- 3.7V 电池 -> 5V

注意：

升压时输入电流可能很大。不能只看输出电流。

大致：

```text
Pin ≈ Pout / 效率
```

如果 5V 输出 1A，输出功率 5W，效率 85%，输入 3.7V：

```text
输入电流 ≈ 5W / 0.85 / 3.7V ≈ 1.59A
```

电池和走线都要承受这个电流。

## 8. Buck-Boost

输入可能高于或低于输出时使用。

例子：

- 锂电池 4.2V 到 3.0V，全程输出 3.3V。

电路更复杂，初学可先用成熟模块或参考设计。

## 9. 电源保护

常见保护：

- 保险丝
- PTC 自恢复保险丝
- 防反接二极管
- MOSFET 防反接
- TVS
- 输入滤波
- 过压保护
- 欠压保护
- 软启动

### 防反接

简单方法：

- 串联二极管，简单但有压降。
- PMOS 防反接，压降低但设计稍复杂。

## 10. 输入输出电容

输入电容：

- 提供瞬态电流。
- 降低输入纹波。
- 靠近电源芯片输入脚。

输出电容：

- 降低输出纹波。
- 改善负载瞬态。
- 保证稳压器稳定。

具体容值和类型按数据手册。

## 11. DC-DC 布局要点

最关键：

- 高频回路面积小。
- 输入电容非常靠近芯片。
- SW 节点面积小。
- 电感位置合理。
- 反馈线远离 SW 节点。
- 地回路短。
- 散热铜皮足够。

新手做 DC-DC，必须参考官方 Layout Example。

## 12. 电源测试

测试内容：

- 空载输出电压。
- 轻载输出电压。
- 满载输出电压。
- 纹波。
- 启动波形。
- 负载瞬态。
- 温升。

用到：

- 万用表。
- 示波器。
- 电子负载。
- 热像仪。

## 实操练习

1. 设计一个 USB 5V 转 3.3V LDO 电源。
2. 计算输出 100mA、300mA、500mA 时 LDO 功耗。
3. 查一个 Buck 芯片数据手册，画出典型应用。
4. 找出它推荐的电感、电容和布局图。
5. 在 PCB 中按推荐布局摆放。

## 检查清单

- 是否画了电源树？
- 是否计算最大电流？
- LDO 功耗是否可接受？
- DC-DC 是否按数据手册布局？
- 输入输出电容是否靠近？
- 是否有防反接和过流保护？
- 是否留了电源测试点？

## 常见误区

- 误区：LDO 输出电流够就行。
  纠正：还要看功耗和散热。

- 误区：DC-DC 原理图对了就行。
  纠正：布局不对可能纹波大、发热、EMI 差。

- 误区：电源适配器标 5V 就一定稳定。
  纠正：负载、线损、瞬态都会影响实际电压。

## 本章总结

电源设计先做电源树和预算，再选 LDO 或 DC-DC，并按数据手册设计外围和布局。电源问题会影响整个系统，必须优先验证。
