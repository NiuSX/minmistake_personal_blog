# 16 接地、回流路径与去耦

## 学习目标

学完本章，你应该能：

- 理解 GND 不是理想 0V。
- 理解信号回流路径和回路面积。
- 知道去耦电容的作用和放置原则。
- 避免错误分割地平面。

接地和去耦是硬件稳定性的核心。很多板子原理图看起来正确，但实际复位、噪声大、通信不稳，往往和地、回流、去耦有关。

## 1. GND 是什么

GND 是电路中的参考点和返回路径。

理想情况下 GND 是 0V，但真实 PCB 中：

- 铜线有电阻。
- 铜线有电感。
- 大电流会产生压降。
- 高频电流会产生噪声。

因此不同位置的 GND 可能并不完全相同。

## 2. 电流必须回流

任何信号电流都有返回电流。

例如 GPIO 输出高低电平：

```text
MCU 输出 -> 信号线 -> 负载
负载 -> GND 返回 -> MCU GND
```

如果返回路径很远，回路面积变大，噪声和辐射增加。

## 3. 回路面积

回路面积越大：

- 更容易产生 EMI。
- 更容易受外界干扰。
- 高频信号质量更差。

降低回路面积：

- 信号线靠近连续地平面。
- 电源和地走近。
- 大电流回路短。
- DC-DC 高频回路紧凑。

## 4. 高频回流路径

高频返回电流倾向于走信号线下方最近的参考平面。

因此：

- 信号线下方要有连续 GND。
- 不要跨越地平面裂缝。
- 换层时附近加 GND 过孔。

如果信号跨过地缝，回流被迫绕远，容易产生 EMI 和波形问题。

## 5. 地平面

地平面好处：

- 低阻抗。
- 回流路径短。
- 屏蔽。
- 散热。
- 降低噪声。

双层板：

- 尽量一层作为完整 GND。

四层板：

- 至少一层完整 GND。

## 6. 模拟地和数字地

很多新手听说要分 AGND 和 DGND，然后把地割得很碎。这通常会带来新问题。

正确理解：

- 模拟电路怕数字噪声。
- 大电流回流不能穿过模拟敏感区域。
- 地分区要让电流路径合理，而不是简单切割。

对于多数低速 MCU 板：

- 保持完整地平面。
- 通过布局分区控制电流路径。
- 模拟输入远离数字噪声。
- 必要时在单点连接或磁珠隔离电源。

## 7. 去耦电容的作用

芯片工作时会瞬间需要电流。电源路径有电感，远处电源来不及响应。

去耦电容提供本地瞬态电流，并降低电源噪声。

常见：

- 100nF 陶瓷电容靠近每个 IC 电源脚。
- 1uF / 10uF 放在电源分支。
- 更大电容放电源入口。

## 8. 去耦电容布局

原则：

- 靠近芯片电源脚。
- 电源脚到电容路径短。
- 电容到地路径短。
- 接地过孔靠近电容地端。
- 小电容更靠近芯片。

错误：

- 去耦电容远离芯片。
- 多个去耦电容排成一排放在板边。
- 电容接地绕长线。
- 电源先过芯片再到电容。

## 9. 多个电容并联

常见组合：

```text
100nF + 1uF + 10uF
```

不同电容覆盖不同频段和瞬态需求。

但不要机械堆电容。某些电源芯片对输出电容和 ESR 有稳定性要求，必须按数据手册。

## 10. 磁珠隔离

磁珠可用于隔离高频噪声。

例子：

```text
3V3 -> 磁珠 -> 3V3_A -> 模拟电路
```

注意：

- 磁珠后面需要本地电容。
- 不能让磁珠承受超过额定电流。
- 不要盲目在每个电源分支都加磁珠。

## 11. 大电流地

大电流负载如电机、LED、继电器：

- 地回流电流大。
- 会造成地电位波动。
- 不应穿过 MCU 或 ADC 的敏感地路径。

布局策略：

- 大电流回路靠近电源入口闭合。
- 敏感电路远离大电流路径。
- 必要时使用星形供电思路。

## 12. 接口 ESD 回流

接口 TVS 泄放静电需要低阻抗到地路径。

原则：

- TVS 靠近接口。
- TVS 到 GND 路径短。
- GND 过孔靠近 TVS。
- 不让 ESD 电流穿过板内敏感区域。

## 实操练习

1. 在一个 MCU 板 PCB 中标出去耦电容位置。
2. 检查每个去耦电容到 GND 的路径。
3. 找一根高速或时钟线，看它下方是否有连续 GND。
4. 找一个电机或继电器驱动板，画出大电流回路。
5. 修改双层板，让底层 GND 更连续。

## 检查清单

- GND 是否连续？
- 关键信号是否跨越地缝？
- 去耦电容是否靠近电源脚？
- 电容接地过孔是否足够近？
- 大电流地是否避开敏感区域？
- TVS 到地路径是否短？
- 是否错误地把地割得很碎？

## 常见误区

- 误区：GND 就是 0V，怎么接都一样。
  纠正：真实 GND 有阻抗和噪声。

- 误区：模拟地和数字地必须大面积切开。
  纠正：错误切割可能破坏回流路径。

- 误区：去耦电容只要放在原理图上即可。
  纠正：PCB 位置决定效果。

## 本章总结

接地、回流和去耦决定电路稳定性。保持连续地平面、缩小回路面积、正确放置去耦电容，比盲目加器件更有效。
