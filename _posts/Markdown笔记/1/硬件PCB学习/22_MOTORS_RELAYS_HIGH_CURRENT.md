# 22 电机、继电器和大电流负载

## 学习目标

学完本章，你应该能：

- 理解电机、继电器、加热片、灯带等负载的电气特点。
- 知道感性负载为什么需要续流保护。
- 掌握 MOSFET 驱动大电流负载的基础方法。
- 在 PCB 中规划大电流路径和散热。

大电流负载是新手烧板高发区域。它们不仅耗电大，还经常带来电压跌落、噪声、发热和 EMI。

## 1. 大电流负载的特点

常见负载：

- 电机
- 继电器
- LED 灯带
- 加热片
- 电磁阀
- 舵机
- 扬声器

特点：

- 电流大。
- 启动或动作瞬间电流更大。
- 可能是感性负载。
- 会产生噪声和反向电压。
- 走线和器件会发热。

## 2. GPIO 不能直接驱动大负载

MCU GPIO 通常只能输出几 mA 到几十 mA，具体看数据手册。

负载如果需要：

- 100mA
- 500mA
- 1A
- 更高

必须使用驱动器件，例如三极管、MOSFET、驱动芯片。

## 3. 继电器驱动

继电器线圈是感性负载。

典型低端驱动：

```text
VCC -> Relay Coil -> MOSFET / NPN -> GND
```

必须加续流二极管：

```text
二极管反向并联在线圈两端
```

作用：

- 线圈断电时释放能量。
- 保护 MOSFET 或三极管。

## 4. 继电器选型

看两部分：

线圈参数：

- 线圈电压。
- 线圈电流。
- 吸合电压。
- 释放电压。

触点参数：

- 最大电压。
- 最大电流。
- AC / DC 能力。
- 负载类型。

注意：

触点能切 AC 10A，不代表能安全切 DC 10A。直流拉弧更难熄灭。

## 5. MOSFET 驱动

常见低端 NMOS 开关：

```text
负载 -> NMOS Drain
NMOS Source -> GND
Gate -> MCU GPIO
```

常加：

- 栅极串联电阻。
- 栅极下拉电阻。
- 续流二极管或 TVS。

## 6. MOSFET 选型

看：

- Vds 耐压。
- Id 电流。
- Rds(on)。
- Vgs 驱动电压。
- 栅极电荷 Qg。
- 封装热阻。

关键提醒：

`Vgs(th)` 不是完全导通电压。3.3V MCU 要选择在 Vgs=2.5V 或 3.3V 下 Rds(on) 明确给出的逻辑电平 MOSFET。

## 7. MOSFET 发热计算

导通损耗：

```text
P = I^2 x Rds(on)
```

例子：

```text
I = 2A
Rds(on) = 50mΩ = 0.05Ω
P = 2 x 2 x 0.05 = 0.2W
```

还要考虑封装散热和环境温度。

## 8. 电机驱动

电机类型：

- 直流有刷电机
- 无刷电机
- 步进电机
- 舵机

电机问题：

- 启动电流大。
- 堵转电流大。
- 产生反电动势。
- 电刷噪声。
- 电源跌落。

设计要点：

- 电源容量足够。
- 驱动芯片电流足够。
- 电源入口加大电容。
- 电机线远离信号线。
- 驱动回路短。
- 必要时加 TVS、RC、磁珠。

## 9. H 桥

H 桥用于控制直流电机正反转。

可用：

- 分立 MOSFET。
- 专用 H 桥驱动芯片。

新手建议用成熟驱动芯片。

注意：

- 防止上下桥臂直通。
- 关注死区时间。
- 关注散热。
- 关注电机电流峰值。

## 10. LED 灯带和加热片

这些通常是电阻性大电流负载。

设计要点：

- 电源电流足够。
- MOSFET 电流足够。
- 走线宽。
- 接口额定电流足够。
- 端子和线材足够。
- 注意发热。

LED 灯带可能有较大浪涌电流，电源入口要有余量。

## 11. PCB 大电流设计

原则：

- 走线宽或铺铜。
- 回路面积小。
- 多个过孔并联换层。
- 远离模拟和时钟。
- 连接器额定电流足够。
- 加散热铜皮。
- 必要时开窗上锡增加载流。

不要只加宽正极，回流 GND 也要能承受同样电流。

## 12. 噪声隔离

大负载可能导致 MCU 复位。

解决：

- 负载电源和 MCU 电源分支。
- 负载附近加大电容。
- MCU 电源加稳压和去耦。
- 大电流回流不穿过 MCU 地。
- 驱动输入加电阻。
- 必要时光耦或隔离。

## 实操练习

1. 设计一个 MCU 控制继电器电路，包含续流二极管。
2. 设计一个 NMOS 控制 12V LED 灯带电路。
3. 计算 2A 电流下 MOSFET 导通损耗。
4. 在 PCB 中画出负载电流回路。
5. 给负载电源入口添加保险丝和 TVS。

## 检查清单

- 负载电流是否超过 GPIO 能力？
- MOSFET 是否能被 3.3V 或 5V 完全驱动？
- 感性负载是否有续流或吸收？
- 走线和连接器是否能承受电流？
- GND 回流是否足够宽？
- 电源是否有足够瞬态能力？
- 大电流是否远离敏感信号？

## 常见误区

- 误区：继电器只要能吸合就行。
  纠正：还要看线圈驱动、触点容量和续流保护。

- 误区：MOSFET 标称电流很大就一定能用。
  纠正：实际受驱动电压、散热和 PCB 影响。

- 误区：只加粗正极线。
  纠正：回流路径同样承载电流。

## 本章总结

电机、继电器和大电流负载要重点处理驱动、保护、走线、散热和噪声。不要让 MCU 直接带负载，不要忽略感性负载的反向能量。
