# 24 电源完整性基础

## 学习目标

学完本章，你应该能：

- 理解电源完整性关注什么。
- 知道电源纹波、瞬态掉压、地弹、去耦不足的原因。
- 掌握基础电源网络设计和测试方法。
- 能判断芯片复位或通信异常是否可能由电源问题引起。

电源完整性 Power Integrity，简称 PI，关注芯片在各种工作状态下是否得到稳定、低噪声、低阻抗的电源。

## 1. 电源完整性是什么

电源完整性要保证：

- 电压在允许范围内。
- 纹波足够小。
- 负载突变时不掉压过多。
- 电源噪声不影响模拟和时钟。
- 地参考稳定。
- 每个芯片瞬态电流有本地供应。

## 2. 常见 PI 问题

| 问题 | 表现 |
| :--- | :--- |
| 电源纹波大 | ADC 噪声、通信异常 |
| 瞬态掉压 | MCU 复位、无线模块掉线 |
| 去耦不足 | 芯片不稳定、随机错误 |
| 地弹 | GPIO 误触发、接口异常 |
| 电源路径阻抗高 | 负载一大电压下降 |
| DC-DC 布局差 | 发热、噪声、EMI |

## 3. 电源阻抗

真实电源路径有：

- 电阻
- 电感
- 电容

当芯片瞬间需要电流时，路径电感会阻碍电流变化，导致本地电压波动。

解决：

- 就近去耦。
- 使用低阻抗电源平面。
- 缩短电源路径。
- 增加储能电容。

## 4. 去耦网络

去耦电容按位置分：

- 芯片引脚旁：高频去耦。
- 模块附近：局部储能。
- 电源入口：大容量储能。

典型：

```text
每个 IC: 100nF
每个模块: 1uF / 4.7uF / 10uF
电源入口: 47uF / 100uF 或按负载
```

具体以芯片手册和负载需求为准。

## 5. 电容不是理想器件

真实电容有：

- ESR
- ESL
- 直流偏压效应
- 温度特性

高频下电容表现会受封装和布局影响。

所以去耦不只是“放一个 100nF”，还要放对位置。

## 6. 瞬态负载

这些模块会有瞬态大电流：

- Wi-Fi / BLE 发射。
- 电机启动。
- LED 同时点亮。
- MCU 从睡眠唤醒。
- 大量 GPIO 同时翻转。

处理：

- 电源芯片有足够峰值能力。
- 模块附近加储能电容。
- 电源路径短而宽。
- 敏感模块单独滤波。

## 7. 电源纹波

纹波来源：

- DC-DC 开关。
- 负载变化。
- 布局寄生参数。
- 输入电源质量差。

测量：

- 用示波器 AC 耦合或合适量程。
- 探头地线要短。
- 在负载附近测。

注意：

长地线会把噪声测得比真实更夸张。

## 8. 地弹

地弹是地电位因瞬态电流而变化。

常见原因：

- 大电流回流经过细地线。
- 多个输出同时翻转。
- 电机和 MCU 共用不良地路径。

处理：

- 地平面低阻抗。
- 大电流路径独立规划。
- 去耦靠近。
- 减小回路面积。

## 9. 电源平面和铺铜

四层板中，电源平面可以降低电源阻抗。

双层板中：

- 电源可用宽线或局部铺铜。
- GND 尽量保持完整。
- 关键负载附近加电容。

## 10. 电源测试方法

测试点：

- 电源输入。
- 稳压器输出。
- 负载芯片电源脚附近。
- 模拟电源。
- 参考电压。

测试内容：

- 空载电压。
- 满载电压。
- 纹波。
- 启动波形。
- 负载突变。
- 温升。

## 实操练习

1. 用示波器测 LDO 输出纹波。
2. 在 MCU 板上测芯片电源脚附近电压。
3. 给 Wi-Fi 或电机负载加大电容，观察复位问题是否改善。
4. 比较长地线探头和短地弹簧测纹波的差别。
5. 画出一块板的电源路径和回流路径。

## 检查清单

- 是否有电源预算？
- 电源路径是否短而宽？
- 每个芯片是否就近去耦？
- 瞬态负载附近是否有储能？
- DC-DC 布局是否按推荐？
- 纹波是否在可接受范围？
- 负载突变是否会导致复位？

## 常见误区

- 误区：万用表测电压正常，电源就没问题。
  纠正：万用表看不到瞬态和纹波。

- 误区：多放大电容一定更好。
  纠正：电容位置、ESR、稳定性都重要。

- 误区：电源问题只会影响电源模块。
  纠正：电源问题会表现为复位、通信错误、ADC 噪声等。

## 本章总结

电源完整性关注动态稳定性。正确的电源树、低阻抗路径、合理去耦、储能电容和真实波形测量，是解决电源问题的基础。
