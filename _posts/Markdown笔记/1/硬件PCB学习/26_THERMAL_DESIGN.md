# 26 热设计基础

## 学习目标

学完本章，你应该能：

- 理解器件为什么发热。
- 会做基础功耗和温升估算。
- 知道 PCB 散热铜皮、热过孔、封装选择的作用。
- 在布局时避免热源影响敏感器件。

热设计不是大功率产品才需要。LDO、MOSFET、DC-DC、电机驱动、LED 都可能让小板明显发热。

## 1. 热从哪里来

发热来自功耗。

常见功耗：

- LDO 压差损耗。
- MOSFET 导通损耗。
- 电阻功耗。
- DC-DC 开关和导通损耗。
- LED 发热。
- MCU 和无线模块工作功耗。

功耗越大，散热越差，温度越高。

## 2. 功耗计算

基本：

```text
P = U x I
P = I^2 x R
```

LDO：

```text
P = (Vin - Vout) x Iout
```

MOSFET：

```text
P = I^2 x Rds(on)
```

电阻：

```text
P = I^2 x R
```

## 3. 热阻

热阻表示热传导困难程度。

常见：

- θJA：结到环境。
- θJC：结到壳。

温升估算：

```text
温升 = 功耗 x 热阻
```

例子：

```text
功耗 = 0.5W
θJA = 100°C/W
温升 = 50°C
```

如果环境 25°C，芯片结温约 75°C。

## 4. 封装对散热的影响

同一个芯片不同封装，散热能力差别很大。

一般：

- 小 SOT-23 散热有限。
- SOT-223 散热更好。
- QFN 底部散热焊盘需要连接铜皮。
- DPAK 等功率封装适合更大功率。

选封装时不能只看体积，要看功耗。

## 5. PCB 散热方式

常用：

- 增大铜皮面积。
- 顶层和底层用过孔连接。
- 使用热过孔。
- 连接到 GND 或电源平面。
- 选择更厚铜。
- 加散热片。
- 加风道。

铜皮越大，散热越好，但也要考虑电气连接和布局。

## 6. 热过孔

热过孔用于把热从顶层传到底层或内层平面。

常见于：

- QFN 底部焊盘。
- 电源芯片。
- MOSFET。
- LED。

注意：

- 过孔数量和孔径按封装建议。
- 有些工艺需要塞孔或盖油，避免焊锡流失。

## 7. 热源布局

热源包括：

- LDO
- DC-DC
- MOSFET
- 电机驱动
- 大功率电阻
- LED
- 无线模块

布局原则：

- 热源远离温度传感器。
- 热源周围留散热空间。
- 多个热源不要过度集中。
- 外壳中考虑空气流动。
- 用户可触摸区域控制温度。

## 8. 温度对器件影响

温度升高会导致：

- 寿命降低。
- 参数漂移。
- 漏电增大。
- 稳压器限流或热关断。
- 电容寿命下降。
- 传感器误差。

电解电容对温度尤其敏感。

## 9. 热测试

方法：

- 手感初步判断。
- 热电偶。
- 红外测温。
- 热像仪。
- 芯片内部温度寄存器。

注意：

手摸只能粗略判断，不要摸高压或可能烫伤部位。

测试条件：

- 空载。
- 典型负载。
- 最大负载。
- 高环境温度。
- 长时间运行。

## 实操练习

1. 计算 5V 转 3.3V LDO 在 200mA、500mA 时功耗。
2. 查一个 LDO 的 θJA，估算温升。
3. 在 PCB 中给 LDO 增加散热铜皮。
4. 用热像仪或温度计测电源芯片温度。
5. 把温度传感器布局远离热源。

## 检查清单

- 是否计算主要器件功耗？
- 封装散热是否足够？
- 是否需要散热铜皮？
- 是否需要热过孔？
- 热源是否远离传感器？
- 最大负载下温度是否可接受？
- 外壳是否影响散热？

## 常见误区

- 误区：芯片没烧就没问题。
  纠正：长期高温会降低寿命和可靠性。

- 误区：LDO 简单所以不会热。
  纠正：压差和电流越大，LDO 越热。

- 误区：小板功率小不用热设计。
  纠正：小板散热面积更小，更容易局部高温。

## 本章总结

热设计从功耗计算开始。选合适封装，留散热铜皮和热过孔，避免热源影响敏感器件，并在最大负载下实际测试温度。
