# 28 物料、封装和供应链

## 学习目标

学完本章，你应该能：

- 理解 BOM 的作用和基本格式。
- 知道封装、符号、实物之间的关系。
- 学会基础器件选型和替代料判断。
- 建立供应链意识，避免选到买不到或快停产的器件。

硬件设计最终要买元件、焊元件。物料和封装管理不好，会导致板子无法装配，甚至需要重新打样。

## 1. BOM 是什么

BOM 是 Bill of Materials，物料清单。

典型字段：

| 字段 | 说明 |
| :--- | :--- |
| 位号 | R1、C3、U2 |
| 数量 | 同类器件数量 |
| 名称 | 电阻、电容、芯片 |
| 参数 | 10k、100nF、3.3V |
| 封装 | 0603、SOT-23 |
| 厂商 | Manufacturer |
| 厂商料号 | MPN |
| 供应商料号 | LCSC、DigiKey 等 |
| 替代料 | 可替换型号 |
| 备注 | 精度、耐压、特殊要求 |

## 2. BOM 为什么重要

BOM 用于：

- 采购。
- 贴片。
- 成本估算。
- 替代料管理。
- 版本追踪。

如果 BOM 不完整：

- 采购可能买错。
- 贴片厂无法确认。
- 返修难以定位。
- 改版后无法追踪差异。

## 3. 符号、封装和实物

三者关系：

- Symbol：原理图里的逻辑符号。
- Footprint：PCB 上的焊盘。
- Physical Part：真实元件。

必须一致：

- 引脚编号一致。
- 封装尺寸一致。
- 极性方向一致。
- 焊盘适合焊接。

常见错误：

- 原理图引脚编号和封装焊盘不一致。
- 连接器方向反。
- IC Pin 1 错。
- 电容封装和实物尺寸不匹配。

## 4. 常见封装

电阻电容：

- 0402
- 0603
- 0805
- 1206

三极管 / MOSFET：

- SOT-23
- SOT-223
- TO-252

IC：

- SOIC
- TSSOP
- QFP
- QFN
- BGA

新手建议：

- 电阻电容选 0603 或 0805。
- IC 选引脚可见封装。
- 少用 QFN 和 BGA 作为第一版。

## 5. 封装核对方法

核对：

- 数据手册封装尺寸。
- 焊盘推荐图。
- 引脚间距。
- Pin 1 位置。
- 元件高度。
- 3D 模型。
- 实物方向。

实用方法：

- 打印 1:1 PCB 图，把元件放上去比对。
- 买样品后再最终确认封装。
- 对连接器特别谨慎。

## 6. 器件选型

选型看：

- 电气参数。
- 封装。
- 温度范围。
- 供货。
- 价格。
- 生命周期。
- 替代料。
- 数据手册完整性。
- 是否有参考设计。

不要只选最便宜，也不要选过度复杂器件。

## 7. 替代料

替代料要确认：

- 参数相同或更优。
- 封装兼容。
- 引脚兼容。
- 温度范围满足。
- 精度满足。
- 供应稳定。

电阻电容替代较容易。

芯片替代更复杂，可能涉及：

- 引脚不同。
- 软件驱动不同。
- 时序不同。
- 外围电路不同。

## 8. 供应链风险

风险：

- 缺货。
- 涨价。
- 停产。
- 交期长。
- 假货。
- 单一供应商。

降低风险：

- 选常用料。
- 准备替代料。
- 不轻易选冷门芯片。
- 量产前确认库存。
- 关键器件找正规渠道。

## 9. 成本意识

成本包括：

- 元件价格。
- PCB 成本。
- 贴片成本。
- 测试成本。
- 返修成本。
- 库存成本。

有时候一个便宜器件如果难焊、难测、故障率高，整体成本反而更高。

## 10. 物料版本管理

每次改版记录：

- 新增器件。
- 删除器件。
- 替换器件。
- 参数变化。
- 封装变化。
- 替代料变化。

BOM 要和硬件版本对应。

## 实操练习

1. 为一个 LED 小板整理 BOM。
2. 查每个元件是否能购买。
3. 核对每个元件封装。
4. 给电阻电容找替代料。
5. 买一个连接器样品，对照 PCB 封装检查方向。

## 检查清单

- BOM 是否包含位号、参数、封装、型号？
- 封装是否和实物匹配？
- Pin 1 是否正确？
- 极性是否清楚？
- 关键器件是否有库存？
- 是否准备替代料？
- BOM 是否和 PCB 版本一致？

## 常见误区

- 误区：EDA 库里的封装一定正确。
  纠正：必须核对数据手册和实物。

- 误区：替代料只要参数差不多。
  纠正：还要看封装、引脚、温度和应用条件。

- 误区：BOM 是量产才需要。
  纠正：打样阶段也需要 BOM 管理。

## 本章总结

物料和封装是硬件落地的关键。会画原理图不够，还要确保元件买得到、焊得上、参数对、版本清楚。连接器和封装尤其要反复核对。
