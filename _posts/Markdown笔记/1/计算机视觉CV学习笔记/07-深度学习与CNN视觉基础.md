# 07-深度学习与 CNN 视觉基础

## 1. 为什么深度学习适合视觉

传统视觉依赖人工设计特征，而深度学习可以从数据中自动学习特征。CNN 能从低层边缘、纹理逐步学习到高层语义。

典型层级：

```text
边缘 -> 纹理 -> 局部形状 -> 目标部件 -> 物体类别
```

## 2. 神经网络基本概念

核心组成：

- 输入。
- 参数。
- 前向传播。
- 损失函数。
- 反向传播。
- 优化器。
- 训练集、验证集、测试集。

训练目标是让模型预测结果接近真实标签。

## 3. 卷积

卷积层用卷积核在图像上滑动，提取局部特征。

参数：

- kernel size：卷积核大小。
- stride：步长。
- padding：填充。
- channels：通道数。

卷积优点：

- 局部连接。
- 参数共享。
- 平移等变。

## 4. 池化

池化用于降低特征图尺寸。

常见：

- Max Pooling。
- Average Pooling。

作用：

- 降低计算量。
- 增大感受野。
- 提供一定平移鲁棒性。

## 5. 激活函数

常见激活：

- ReLU。
- Leaky ReLU。
- GELU。
- SiLU/Swish。

激活函数提供非线性，否则多层线性变换等价于一层线性变换。

## 6. 归一化

常见：

- BatchNorm。
- LayerNorm。
- GroupNorm。

作用：

- 稳定训练。
- 加快收敛。
- 缓解梯度问题。

## 7. 经典 CNN

### 7.1 LeNet

早期 CNN，用于手写数字识别。

### 7.2 AlexNet

推动深度学习视觉爆发，引入 ReLU、Dropout、大规模 GPU 训练。

### 7.3 VGG

使用大量 3x3 卷积，结构简单，但参数量大。

### 7.4 ResNet

引入残差连接：

```text
y = F(x) + x
```

解决深层网络退化问题，是视觉模型重要基础。

### 7.5 MobileNet

使用深度可分离卷积，适合移动端和边缘设备。

## 8. Vision Transformer

ViT 把图像分成 patch，当作 token 输入 Transformer。

特点：

- 全局建模能力强。
- 大数据下效果好。
- 计算成本较高。

常见模型：

- ViT。
- Swin Transformer。
- DeiT。

## 9. 迁移学习

迁移学习使用预训练模型作为基础，在自定义数据上微调。

好处：

- 降低数据需求。
- 加快收敛。
- 提高性能。

常见做法：

- 冻结 backbone，只训练分类头。
- 解冻部分层微调。
- 全量微调。

## 10. 损失函数

分类：

- Cross Entropy。
- Focal Loss。

回归：

- L1 Loss。
- MSE。
- Smooth L1。

分割：

- Dice Loss。
- BCE。
- Cross Entropy。
- Lovasz Loss。

检测：

- 分类损失。
- 边框回归损失。
- IoU/GIoU/DIoU/CIoU。

## 11. 优化器

常见：

- SGD。
- Adam。
- AdamW。

学习率非常关键。过大可能发散，过小收敛慢。

常见学习率策略：

- StepLR。
- Cosine Annealing。
- Warmup。
- OneCycle。

## 12. 过拟合和欠拟合

过拟合：

- 训练集好，验证集差。
- 模型记住训练样本，不泛化。

解决：

- 增加数据。
- 数据增强。
- 正则化。
- Dropout。
- 降低模型复杂度。
- 早停。

欠拟合：

- 训练集和验证集都差。

解决：

- 增大模型。
- 训练更久。
- 提高输入质量。
- 调整学习率。
- 检查标签。

## 13. 深度学习视觉训练流程

```text
收集数据 -> 清洗数据 -> 划分数据集 -> 数据增强 -> 选择模型 -> 训练 -> 验证 -> 误差分析 -> 调参 -> 导出部署
```

训练前优先检查：

- 标签是否正确。
- 类别是否平衡。
- 输入尺寸是否合适。
- 训练和推理预处理是否一致。
- 指标是否符合业务目标。
