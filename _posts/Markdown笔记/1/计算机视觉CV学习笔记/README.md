# 计算机视觉 CV 学习笔记目录

这套文档用于系统学习计算机视觉 Computer Vision，覆盖从图像基础、OpenCV、传统图像处理、经典算法、特征匹配、几何视觉，到深度学习视觉模型、目标检测、图像分割、视频理解、OCR、训练评估和工程部署。

计算机视觉不是只会调用模型接口，也不是只会 OpenCV 函数。完整的 CV 能力包括：

- 理解图像数据本质。
- 掌握常用预处理和增强方法。
- 会使用传统图像处理算法定位问题。
- 理解特征、匹配、相机模型和几何关系。
- 掌握 CNN、Transformer、检测、分割、关键点等深度学习方法。
- 会构建数据集、训练模型、评估指标和调参。
- 会把模型部署到真实业务、工业、机器人、移动端或边缘设备中。

## 学习建议

- 先学图像基础和 OpenCV，再进入深度学习。
- 传统算法不要跳过。很多工业视觉、检测定位、预处理、缺陷分析仍然依赖传统方法。
- 每个算法都要理解“输入、输出、参数、适用场景、失效原因”。
- 训练模型前，先检查数据质量、标注质量、类别分布和评价指标。
- 工程落地时，准确率不是唯一目标，还要考虑速度、稳定性、误报漏报、部署成本和维护成本。
