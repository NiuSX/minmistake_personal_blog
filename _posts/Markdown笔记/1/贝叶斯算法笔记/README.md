## 建议阅读顺序

1. 概率基础和静态模型：朴素贝叶斯、贝叶斯线性回归、贝叶斯逻辑回归。
2. 结构化概率模型：贝叶斯网络、HMM。
3. 递归贝叶斯滤波：卡尔曼滤波、扩展/无迹卡尔曼滤波、粒子滤波。
4. 通用后验推断：MCMC、变分推断。
5. 应用型贝叶斯模型：LDA、高斯过程、贝叶斯神经网络。
6. 后验驱动决策：贝叶斯优化、Thompson Sampling、Beta-Binomial 贝叶斯 A/B 测试。
