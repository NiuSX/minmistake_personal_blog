# 第 3 章 线性模型

## 1. 本章目标

线性模型是机器学习中最基础的一类模型。它假设输出可以由输入特征的线性组合决定。

基本形式：

```text
f(x) = w1*x1 + w2*x2 + ... + wd*xd + b
```

其中 w 是权重，b 是偏置。

## 2. 为什么线性模型重要

线性模型看起来简单，但非常重要：

- 容易理解。
- 训练效率高。
- 可解释性强。
- 是许多复杂模型的基础。
- 在高维稀疏数据上常常表现很好。

很多复杂模型本质上也是在某个变换后的空间里做线性模型。

## 3. 线性回归

线性回归用于预测连续值。

目标是找到一组参数，使预测值尽量接近真实值。

常用平方损失：

```text
loss = (预测值 - 真实值)^2
```

整体目标是最小化所有样本的平方损失和。

### 直觉

在二维中，线性回归就是找一条直线，让它尽量贴近所有数据点。

在高维中，就是找一个超平面。

### 优点

- 简单。
- 可解释。
- 训练快。

### 缺点

- 只能表达线性关系。
- 对异常值敏感。

## 4. 对数几率回归

虽然名字里有“回归”，但逻辑回归主要用于分类。

它先计算线性组合：

```text
z = w^T x + b
```

再通过 sigmoid 函数把 z 映射到 0 到 1：

```text
p = 1 / (1 + e^(-z))
```

p 可以理解为样本属于正类的概率。

### 分类规则

如果 p 大于阈值，例如 0.5，就判为正类；否则判为负类。

### 为什么不用线性回归直接分类

线性回归输出没有概率意义，可能小于 0 或大于 1。sigmoid 可以把输出压到 0 到 1，更适合二分类。

### 优点

- 输出概率。
- 可解释。
- 可加正则化。
- 工业界常用。

## 5. 线性判别分析 LDA

LDA 的思想是把样本投影到一条直线上，使：

- 同类样本投影后尽量接近。
- 异类样本投影后尽量远离。

一句话：

```text
类内紧凑，类间分离。
```

LDA 不只是分类方法，也可用于降维。

## 6. 多分类学习

许多线性模型天然处理二分类，多分类可通过拆分实现。

常见策略：

### 一对其余 OvR

每次把一个类别作为正类，其他类别作为负类。

若有 N 个类别，训练 N 个分类器。

### 一对一 OvO

每次选两个类别训练一个分类器。

若有 N 个类别，需要训练 N(N-1)/2 个分类器。

### 多对多

把类别划分成多个组合，用编码方式进行分类。

## 7. 类别不平衡

类别不平衡指不同类别样本数量差异很大。

例如欺诈检测中，正常交易远多于欺诈交易。

常见处理：

- 调整分类阈值。
- 过采样少数类。
- 欠采样多数类。
- 使用类别权重。
- 使用 Precision、Recall、F1、AUC 等指标。

## 8. 新手易错点

1. 线性模型不等于只能处理简单问题。如果特征工程或核映射合适，线性模型也很强。
2. 逻辑回归不是回归任务，而是分类模型。
3. 参数大小代表特征影响，但前提是特征尺度可比。
4. 准确率高不代表类别不平衡问题解决。

## 10. 精讲：线性模型到底“线性”在哪里

线性模型指模型对参数是线性的，常见形式是：

```text
f(x) = w^T x + b
```

如果只有一个特征，模型是一条直线；如果有两个特征，模型是一个平面上的分割线；如果有更多特征，模型是高维空间中的超平面。

线性模型的本质是：每个特征对结果做一个加权贡献，然后把这些贡献加起来。

例如预测房价：

```text
房价 = 8000 * 面积 + 50000 * 是否学区房 - 3000 * 距离地铁距离 + b
```

这不是实际公式，只是帮助理解。权重表示特征影响方向和强度：

- 权重为正：特征越大，预测越大。
- 权重为负：特征越大，预测越小。
- 权重接近 0：特征影响较弱。

## 11. 精讲：线性回归从几何上怎么理解

一维输入时，线性回归是在散点图中找一条直线，让点到直线的整体距离尽量小。

为什么常用平方误差？

因为平方误差有几个优点：

- 正负误差不会抵消。
- 大误差会被更重惩罚。
- 数学上可导，方便优化。
- 在高斯噪声假设下有概率解释。

平方误差也有缺点：对异常值敏感。

如果某个房价数据录错，多了一个 0，平方误差会非常大，模型可能被这个异常点拉偏。

## 12. 精讲：逻辑回归为什么是分类模型

逻辑回归名字中有“回归”，但常用于分类。

它先计算线性分数：

```text
z = w^T x + b
```

然后用 sigmoid 把分数转成概率：

```text
p = 1 / (1 + e^(-z))
```

sigmoid 的特点：

- z 很大时，p 接近 1。
- z 很小时，p 接近 0。
- z = 0 时，p = 0.5。

因此逻辑回归可以解释为：

```text
先用线性模型打分，再把分数转成属于正类的概率。
```

## 13. 精讲：逻辑回归的决策边界

如果阈值取 0.5，则：

```text
p >= 0.5 判为正类
p < 0.5 判为负类
```

因为 sigmoid(z) >= 0.5 等价于 z >= 0，所以决策边界是：

```text
w^T x + b = 0
```

这说明逻辑回归的分类边界仍然是线性的。

如果数据本身不是线性可分，可以通过特征变换构造非线性特征，例如：

```text
x1, x2 -> x1, x2, x1^2, x2^2, x1*x2
```

这样模型对新特征仍是线性的，但对原始输入可以形成非线性边界。

## 14. 精讲：线性判别分析 LDA 的直觉

LDA 的目标不是直接拟合概率，而是找一个投影方向。

好投影方向应该满足：

- 同类样本投影后聚在一起。
- 不同类样本投影后分得很开。

可以把它想象成用手电筒照三维物体，在墙上形成二维影子。不同照射方向得到的影子不同。LDA 要找一个方向，让不同类别在影子上最好区分。

LDA 的思想非常重要，因为很多降维和表示学习方法都在追求类似目标：让有用差异保留，无用变化压缩。

## 15. 精讲：多分类拆分策略怎么选

### OvR

如果有猫、狗、车三类，OvR 会训练：

- 猫 vs 非猫。
- 狗 vs 非狗。
- 车 vs 非车。

预测时看哪个分类器置信度最高。

优点：模型数量少。

缺点：每个分类器面对类别不平衡，正类少、负类多。

### OvO

三类会训练：

- 猫 vs 狗。
- 猫 vs 车。
- 狗 vs 车。

预测时多个分类器投票。

优点：每个分类器任务更简单。

缺点：类别多时分类器数量快速增加。

## 16. 精讲：类别不平衡的真实危害

假设诈骗交易只占 1%。一个模型永远预测“非诈骗”，准确率也有 99%。

但这个模型完全没用，因为它一个诈骗都抓不到。

因此类别不平衡时要关注：

- 少数类召回率。
- Precision。
- F1。
- PR 曲线。
- AUC。
- 业务成本。

处理方法：

- 给少数类更高权重。
- 欠采样多数类。
- 过采样少数类。
- 调整分类阈值。
- 使用异常检测思路。

## 17. 本章实践清单

训练线性模型时建议检查：

1. 特征是否需要标准化。
2. 是否存在强共线性。
3. 是否需要正则化。
4. 类别是否不平衡。
5. 权重解释是否建立在同一尺度上。
6. 线性边界是否足够表达数据规律。
