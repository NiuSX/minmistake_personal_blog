# 第 5 章 神经网络

## 1. 本章目标

本章介绍神经网络的基本结构和训练方法。

重点是理解：

- 神经元模型。
- 感知机。
- 多层网络。
- 误差逆传播。
- 局部极小与全局最小。
- 深度学习的基本思想。

## 2. 神经元模型

人工神经元接收多个输入，每个输入乘以权重，求和后经过激活函数输出。

形式：

```text
y = f(w1*x1 + w2*x2 + ... + wd*xd + b)
```

其中：

- x 是输入。
- w 是权重。
- b 是偏置。
- f 是激活函数。

激活函数提供非线性能力。没有激活函数，多层线性网络仍然等价于一层线性模型。

## 3. 感知机

感知机是最早的神经网络模型之一。

它可以学习线性可分问题。

学习过程：

1. 初始化权重。
2. 输入样本，得到预测。
3. 如果预测错误，调整权重。
4. 重复直到分类正确或达到迭代上限。

局限：无法解决 XOR 等线性不可分问题。

## 4. 多层网络

多层网络包含：

- 输入层。
- 隐藏层。
- 输出层。

隐藏层让模型可以学习复杂非线性映射。

直觉：

- 低层学习简单特征。
- 高层组合简单特征，形成复杂表示。

例如图像任务中，低层可能识别边缘，高层识别形状，更高层识别物体。

## 5. 误差逆传播算法

误差逆传播，也叫反向传播，是训练多层神经网络的核心算法。

步骤：

1. 前向传播：输入经过网络得到输出。
2. 计算损失：比较预测和真实标签。
3. 反向传播：从输出层向前计算每个权重对损失的影响。
4. 参数更新：用梯度下降调整权重。

核心数学工具是链式法则。

## 6. 损失函数

常见损失：

- 回归：均方误差。
- 二分类：交叉熵。
- 多分类：softmax 交叉熵。

损失函数决定模型“错得多严重”，也决定优化方向。

## 7. 全局最小与局部极小

神经网络的损失曲面通常很复杂，可能存在很多局部极小、鞍点和平坦区域。

### 全局最小

所有可能参数中损失最低的位置。

### 局部极小

附近区域内最低，但不一定是全局最低。

实际深度学习中，不一定非要找到全局最小；找到泛化性能好的低损失区域更重要。

## 8. 常见神经网络

### RBF 网络

径向基函数网络使用距离度量进行局部响应。

### ART 网络

自适应共振理论网络强调稳定性与可塑性的平衡。

### SOM 网络

自组织映射用于无监督聚类和可视化。

### 级联相关网络

通过逐步增加隐藏单元构建网络。

## 9. 深度学习

深度学习指使用多层神经网络学习层级表示。

深度学习成功依赖：

- 大数据。
- GPU 算力。
- 反向传播。
- 更好的激活函数。
- 正则化。
- 网络结构设计。

## 10. 新手易错点

1. 神经网络不是模拟完整人脑，只是受神经系统启发的计算模型。
2. 层数越多不一定越好，训练难度和过拟合风险也会上升。
3. 激活函数非常关键，否则多层线性网络仍是线性模型。
4. 损失低不等于泛化好。
5. 神经网络需要特征缩放和合理初始化。

## 12. 精讲：神经网络从哪里开始理解

神经网络不要一开始就想成人脑。更实用的理解是：

```text
神经网络是一种由很多可调函数单元组成的复杂函数。
```

它接收输入，经过多层变换，输出预测结果。训练神经网络就是调整里面大量权重，让预测越来越接近真实答案。

例如识别手写数字：

- 输入：图片像素。
- 输出：0 到 9 的概率。
- 训练：如果把 5 预测成 3，就根据错误调整权重。

## 13. 精讲：单个神经元做了什么

一个神经元做三件事：

1. 接收输入。
2. 加权求和。
3. 经过激活函数输出。

公式：

```text
z = w1*x1 + w2*x2 + ... + wd*xd + b
y = f(z)
```

权重 w 表示输入特征的重要性。偏置 b 可以理解为整体门槛。激活函数 f 决定神经元如何响应。

如果没有激活函数，多层网络只是多个线性变换叠加，仍然等价于一个线性模型。激活函数让网络能表达弯曲、复杂、非线性的决策边界。

## 14. 精讲：感知机为什么只能处理线性可分

感知机输出由一个线性分数决定：

```text
w^T x + b
```

它相当于在空间中画一条线或一个超平面。

如果两类样本可以被一条线分开，感知机能学到。若样本像 XOR 那样交叉分布，无论怎么画一条直线都分不开，感知机就失败。

这说明单层模型表达能力有限。

## 15. 精讲：隐藏层的意义

隐藏层可以把原始输入变换到新的表示空间。

XOR 在原始二维空间中线性不可分，但经过隐藏层变换后，可能在新的空间中变得线性可分。

这就是神经网络强大的关键：

```text
不是直接在原始特征上分类，而是先学习一组更有用的中间表示。
```

深度学习中的“深”就来自多层表示逐级变换。

## 16. 精讲：反向传播为什么重要

多层网络最大的问题是：输出错了，隐藏层该怎么改？

反向传播解决“责任分配”问题。

它从输出层的误差开始，沿网络连接反向计算每个参数对误差的贡献。

可以类比公司项目失败：

- 客户不满意是最终误差。
- 不能只责怪最后交付的人。
- 要沿流程追溯每个环节对失败的贡献。
- 每个环节根据责任大小调整。

反向传播用链式法则完成这个追责过程。

## 17. 精讲：学习率和训练稳定性

梯度告诉我们参数应该往哪个方向改，学习率决定每次改多少。

学习率太大：

- 可能越过最优点。
- 损失震荡。
- 甚至发散。

学习率太小：

- 收敛很慢。
- 容易卡住。

实际训练中常用学习率衰减、Adam 等优化器改善训练。

## 18. 精讲：过拟合在神经网络中为什么常见

神经网络参数很多，表达能力很强。

如果数据少，网络可能记住训练样本细节。

缓解方法：

- 增加数据。
- 数据增强。
- Dropout。
- L2 正则化。
- 早停。
- 减小网络规模。
- Batch Normalization。

## 19. 精讲：深度学习为什么后来才成功

神经网络思想很早就有，但长期受限于：

- 数据不够。
- 算力不够。
- 网络太深时梯度消失。
- 初始化和优化方法不成熟。

2010 年代后，条件成熟：

- 互联网带来大量数据。
- GPU 适合矩阵并行计算。
- ReLU、Dropout、BatchNorm、残差连接等技巧成熟。
- 开源框架降低门槛。

所以深度学习不是突然发明，而是旧思想在新条件下爆发。

## 20. 本章实践清单

训练神经网络时检查：

1. 输入是否归一化。
2. 激活函数是否合适。
3. 损失函数是否匹配任务。
4. 学习率是否过大或过小。
5. 训练集和验证集曲线是否显示过拟合。
6. 是否需要正则化或早停。
7. 数据量是否支撑模型规模。
