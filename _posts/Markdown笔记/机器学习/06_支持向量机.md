# 第 6 章 支持向量机

## 1. 本章目标

支持向量机 SVM 是经典监督学习方法，尤其适合中小规模、高维数据分类。

核心思想：

```text
在能分开样本的分类面中，选择间隔最大的那个。
```

## 2. 间隔与支持向量

在二分类中，线性分类器可以用超平面表示：

```text
w^T x + b = 0
```

样本到分类面的距离越大，分类越有信心。

SVM 希望找到一个分类面，使离分类面最近的样本也尽量远。

这些最近的关键样本叫支持向量。

## 3. 最大间隔的直觉

如果有很多条线都能分开训练样本，哪条最好？

SVM 认为：边界离两类样本都更远的分类线更可靠。

原因：

- 对噪声更稳。
- 泛化能力更好。
- 不容易因为一点扰动就分类错误。

## 4. 对偶问题

SVM 原始优化问题可转化为对偶问题。

对偶形式的重要意义：

1. 计算中只涉及样本之间的内积。
2. 引出核函数。
3. 支持向量对应非零拉格朗日乘子。

新手不必一开始死磕推导，但要理解：对偶问题让 SVM 能用核技巧处理非线性。

## 5. 核函数

现实数据往往线性不可分。

一种思路是把数据映射到高维空间，在高维空间中线性可分。

核函数的妙处是：不显式计算高维映射，只直接计算高维空间中的内积。

常见核函数：

- 线性核。
- 多项式核。
- 高斯核 / RBF 核。
- Sigmoid 核。

### 核技巧直觉

原空间中画不出直线分开样本，但把数据升到更高维后，也许可以用平面分开。

核函数就是在不真正展开高维坐标的情况下完成这种计算。

## 6. 软间隔

硬间隔要求所有训练样本都被正确分类。但现实数据有噪声，强行完全分开可能过拟合。

软间隔允许少量样本违反间隔约束。

通过惩罚参数 C 控制：

- C 大：更重视训练集正确，可能过拟合。
- C 小：更容忍错误，间隔更宽，可能欠拟合。

## 7. 正则化视角

SVM 的目标可以理解为：

```text
模型复杂度 + 训练误差惩罚
```

这与正则化思想一致：既要拟合数据，也要控制模型复杂度。

## 8. 支持向量回归

SVR 用于回归任务。

它允许预测值在真实值附近一个范围内不受惩罚，只有超出范围才计算损失。

直觉：不追求每个点完全贴合，而是允许一个误差管道。

## 9. SVM 优缺点

优点：

- 理论基础强。
- 最大间隔带来较好泛化。
- 核函数可处理非线性。
- 高维小样本场景表现好。

缺点：

- 大规模数据训练成本高。
- 核函数和参数选择敏感。
- 输出概率不如逻辑回归自然。
- 对噪声和重叠数据需要调 C。

2. 为什么最大间隔有助于泛化？
3. 对偶问题为什么重要？

## 11. 精讲：SVM 的核心直觉

SVM 不只是找一条能分开两类样本的线，而是找一条“最稳”的线。

假设平面上有两类点，很多直线都能把它们分开。SVM 会选择离两类最近样本都尽量远的那条线。

原因是：边界越靠近样本，未来稍微有点扰动就可能分错；边界离样本越远，模型越稳。

这就是最大间隔思想。

## 12. 精讲：支持向量为什么关键

分类边界并不由所有样本决定，而主要由离边界最近的少数样本决定。

这些样本就是支持向量。

如果移动远离边界的样本，分类面可能几乎不变；如果移动支持向量，分类面可能变化很大。

这说明 SVM 学到的是由关键边界样本支撑的分类器。

## 13. 精讲：硬间隔与软间隔

### 硬间隔

硬间隔要求训练样本全部正确分类，并且间隔最大。

适合数据完全线性可分且噪声很少的情况。

问题：现实数据通常有噪声，硬间隔可能过拟合，甚至不可行。

### 软间隔

软间隔允许少数样本分错或落入间隔内。

它在两个目标之间权衡：

1. 间隔尽量大。
2. 训练错误尽量少。

参数 C 控制对错误的惩罚。

- C 大：不能容忍错误，边界更贴合训练集。
- C 小：更容忍错误，边界更平滑。

## 14. 精讲：核函数为什么强大

有些数据在原空间中不能用直线分开。

例如一类点围成圆，另一类点在圆外。二维平面中直线分不开，但如果增加一个新特征：

```text
z = x1^2 + x2^2
```

数据在三维空间中可能就能用平面分开。

核函数的作用是：不用显式计算高维特征，也能得到高维空间中的内积结果。

这叫核技巧。

## 15. 精讲：常见核函数怎么选

### 线性核

适合：

- 特征维度高。
- 样本较多。
- 数据近似线性可分。

文本分类中线性核经常有效。

### 多项式核

能表达特征之间的多项式组合关系。

参数较敏感，实际使用要谨慎。

### RBF 核

也叫高斯核，常用且强大。

它可以形成复杂非线性边界。

但参数选择很重要：

- gamma 太大：边界过于弯曲，容易过拟合。
- gamma 太小：边界太平滑，容易欠拟合。

## 16. 精讲：SVM 与逻辑回归的区别

逻辑回归关注概率输出，优化交叉熵损失。

SVM 关注分类间隔，优化最大间隔目标。

区别：

| 维度 | 逻辑回归 | SVM |
| --- | --- | --- |
| 输出 | 概率更自然 | 默认输出分类分数 |
| 核心思想 | 最大似然/概率建模 | 最大间隔 |
| 对边界样本 | 所有样本影响参数 | 支持向量最关键 |
| 非线性 | 需特征变换 | 可用核函数 |

## 17. 精讲：SVM 实战注意事项

1. 特征通常需要标准化。
2. 大数据集上训练可能很慢。
3. RBF 核要调 C 和 gamma。
4. 类别不平衡时要设置类别权重。
5. 若需要概率输出，可能要额外校准。

## 18. 本章理解检查

如果你能解释下面这句话，就基本掌握 SVM：

```text
SVM 不是只追求把训练样本分对，而是追求用尽量宽的安全带把两类样本分开。
```
