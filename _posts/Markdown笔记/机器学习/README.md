## 每章怎么学

每章建议按以下顺序读：

1. 先读“本章目标”，知道这一章解决什么问题。
2. 再读核心概念和算法流程。
3. 遇到公式先看直觉解释，再回到公式。
4. 最后用“复习问题”自测。

如果复习问题答不上来，不要急着进入下一章。

## 当前版本内容特点

每个章节文件已经按“精讲版”补充：

- 用通俗例子解释概念。
- 说明算法为什么这样设计。
- 补充公式背后的直觉。
- 总结优点、缺点和适用场景。
- 标出新手容易误解的地方。
- 给出实践检查清单。

建议不要只看标题跳读。每章后半部分的“精讲”内容才是帮助入门的重点。

## 学习建议

机器学习最重要的是形成统一框架：

```text
数据 -> 特征 -> 模型 -> 损失/目标 -> 优化 -> 评估 -> 泛化
```

后面每个算法都可以放进这个框架里理解。
