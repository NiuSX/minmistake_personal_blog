# 聚类与降维实验

无监督学习没有标签，因此不能像分类那样直接计算准确率。它更依赖数据预处理、可视化、内部指标和解释。

## 一、聚类前必须标准化

K-means、层次聚类、DBSCAN 都常依赖距离。如果特征尺度不同，距离会被数值范围大的特征支配。

```python
from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)
```

## 二、K-means 基础实验

```python
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score

for k in range(2, 8):
    km = KMeans(n_clusters=k, random_state=42, n_init="auto")
    labels = km.fit_predict(X_scaled)
    score = silhouette_score(X_scaled, labels)
    print(k, score)
```

轮廓系数越高，通常说明簇内更紧凑、簇间更分离。但指标不能代替解释。

## 三、肘部法

K-means 的惯性表示样本到簇中心的平方距离总和。

```python
inertias = []

for k in range(1, 10):
    km = KMeans(n_clusters=k, random_state=42, n_init="auto")
    km.fit(X_scaled)
    inertias.append(km.inertia_)
```

如果随着 K 增加，惯性下降速度在某一点明显变慢，这个点可能是合理 K 值。但肘部法并不总是清晰。

## 四、PCA 可视化

```python
from sklearn.decomposition import PCA

pca = PCA(n_components=2)
X_2d = pca.fit_transform(X_scaled)

print(pca.explained_variance_ratio_)
```

`explained_variance_ratio_` 表示每个主成分解释了多少方差。二维可视化能帮助观察结构，但不能当作严格证明。

## 五、聚类结果解释

聚类完成后，要给每个簇做画像：

```python
df_result = X.copy()
df_result["cluster"] = labels
profile = df_result.groupby("cluster").mean()
print(profile)
```

对每个簇回答：

1. 这个簇有什么显著特征？
2. 和其他簇差异在哪里？
3. 是否能命名？
4. 是否能指导后续动作？

## 六、DBSCAN 适用场景

DBSCAN 适合发现不规则形状簇和离群点。

```python
from sklearn.cluster import DBSCAN

db = DBSCAN(eps=0.5, min_samples=5)
labels = db.fit_predict(X_scaled)
```

标签 `-1` 通常表示噪声点。DBSCAN 对 `eps` 很敏感，调参时可以观察 k 距离图。

## 七、无监督实验总结

无监督学习要特别谨慎：

1. 先明确相似度含义。
2. 做标准化。
3. 尝试不同算法和参数。
4. 用指标辅助判断。
5. 用可视化辅助理解。
6. 最终必须回到可解释性和任务目标。
