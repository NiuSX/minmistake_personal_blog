# 模型评估与调参实验

模型调参的目标不是让训练分数最高，而是找到泛化能力更好的模型。正确调参必须结合交叉验证和独立测试集。

## 一、交叉验证

```python
from sklearn.model_selection import cross_val_score
from sklearn.linear_model import LogisticRegression

clf = LogisticRegression(max_iter=1000)
scores = cross_val_score(clf, X, y, cv=5, scoring="f1")

print(scores)
print(scores.mean(), scores.std())
```

平均值反映整体性能，标准差反映稳定性。如果标准差很大，说明模型对数据划分敏感。

## 二、网格搜索

```python
from sklearn.model_selection import GridSearchCV
from sklearn.svm import SVC

param_grid = {
    "C": [0.1, 1, 10],
    "kernel": ["linear", "rbf"],
    "gamma": ["scale", "auto"],
}

grid = GridSearchCV(
    SVC(),
    param_grid=param_grid,
    cv=5,
    scoring="f1",
    n_jobs=-1,
)

grid.fit(X_train, y_train)

print(grid.best_params_)
print(grid.best_score_)
```

网格搜索适合参数范围较小的情况。如果参数很多，可以用随机搜索。

## 三、随机搜索

```python
from sklearn.model_selection import RandomizedSearchCV
from sklearn.ensemble import RandomForestClassifier
from scipy.stats import randint

param_dist = {
    "n_estimators": randint(100, 500),
    "max_depth": randint(3, 20),
    "min_samples_split": randint(2, 20),
}

search = RandomizedSearchCV(
    RandomForestClassifier(random_state=42),
    param_distributions=param_dist,
    n_iter=30,
    cv=5,
    scoring="f1",
    random_state=42,
    n_jobs=-1,
)

search.fit(X_train, y_train)
```

## 四、学习曲线

学习曲线用来判断增加数据是否可能有帮助。

如果训练分数和验证分数都低，通常是欠拟合，需要更强模型或更好特征。

如果训练分数高、验证分数低，通常是过拟合，需要正则化、降复杂度或更多数据。

## 五、验证曲线

验证曲线观察某个超参数变化对训练和验证性能的影响。

例如决策树深度：

```text
深度太小：训练差，验证也差，欠拟合。
深度适中：训练好，验证好。
深度太大：训练很好，验证下降，过拟合。
```

## 六、调参纪律

1. 先建立基线，再调复杂模型。
2. 一次只重点调少数关键参数。
3. 用验证集或交叉验证调参。
4. 测试集只在最后使用。
5. 记录每次实验配置。
6. 不要为了测试集分数反复修改模型。

## 七、调参不是万能的

如果数据质量差、特征不包含有效信息、标签定义混乱，调参无法根本解决问题。很多时候，改数据和改特征比换模型更有效。
