# 机器学习代码实践索引

这个目录的目标是把前面的理论笔记落到实际项目中。学习机器学习不能只看概念，必须知道一个任务从数据到模型到底怎么做。

推荐顺序：

1. 先读 `01_sklearn项目基本模板.md`，掌握通用项目骨架。
2. 做分类任务时读 `02_分类任务完整流程.md`。
3. 做回归任务时读 `03_回归任务完整流程.md`。
4. 学无监督时读 `04_聚类与降维实验.md`。
5. 调模型时读 `05_模型评估与调参实验.md`。
6. 特征处理时读 `06_特征工程实战清单.md`。
7. 出问题时查 `07_常见问题排查手册.md`。
8. 看不懂公式和代码关系时读 `08_从公式到代码的理解方法.md`。

这些文档使用通用 sklearn 思路描述，不依赖某一个固定数据集。真正练习时，可以用鸢尾花、乳腺癌、波士顿房价替代数据、手写数字、垃圾短信、客户分群等公开数据集。
