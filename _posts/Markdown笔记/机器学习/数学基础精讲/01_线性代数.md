# 线性代数

线性代数是机器学习的“数据语言”。样本、特征、参数、模型输入输出，大多数都可以写成向量或矩阵。

## 一、向量：一个样本的表示

一个样本如果有多个特征，就可以写成向量：

```text
x = [x1, x2, x3, ..., xd]
```

例如一个房子可以用面积、房龄、楼层、距离地铁距离表示：

```text
x = [89, 12, 15, 0.8]
```

向量的每一维是一个特征。机器学习模型处理的不是“房子”这个自然对象，而是这个向量表示。

## 二、矩阵：一批样本的表示

如果有 n 个样本，每个样本 d 个特征，可以组成矩阵：

```text
X = n 行 d 列
```

每一行是一个样本，每一列是一个特征。

线性模型常写成：

```text
y_hat = Xw + b
```

其中 `X` 是数据矩阵，`w` 是权重向量，`b` 是偏置，`y_hat` 是预测结果。

## 三、内积：相似度和打分

两个向量的内积：

```text
w^T x = w1*x1 + w2*x2 + ... + wd*xd
```

在线性模型中，内积就是给样本打分。每个特征乘以对应权重，再加起来。

如果两个向量方向接近，内积通常较大；如果方向相反，内积可能较小甚至为负。因此内积也常被用来衡量相似度。

## 四、范数：长度和复杂度

范数衡量向量大小。

L2 范数：

```text
||w||_2 = sqrt(w1^2 + w2^2 + ... + wd^2)
```

L1 范数：

```text
||w||_1 = |w1| + |w2| + ... + |wd|
```

机器学习中，范数常用于正则化。L2 正则惩罚权重整体过大，L1 正则倾向于让部分权重变成 0。

## 五、矩阵乘法的机器学习意义

矩阵乘法不是抽象运算，它常表示“批量计算”。

一个样本预测：

```text
y_hat = w^T x
```

一批样本预测：

```text
y_hat = Xw
```

这就是为什么 numpy、pandas、sklearn、深度学习框架大量使用矩阵运算：它能把很多样本一起计算，效率高且表达简洁。

## 六、特征值与特征向量的直觉

如果一个矩阵作用在某个向量上，只改变这个向量的长度，不改变方向，那么这个向量就是特征向量，长度变化倍数是特征值。

PCA 中，协方差矩阵的特征向量对应数据主要变化方向，特征值对应该方向上的方差大小。保留最大特征值对应的方向，就能保留最多变化信息。

## 七、PCA 和线性代数

PCA 的核心可以理解为：

1. 找到数据方差最大的方向。
2. 把数据投影到这些方向上。
3. 用少数方向近似原始高维数据。

这些方向本质上来自协方差矩阵的特征向量。

## 八、学习重点

新手不需要一开始证明所有定理，但必须掌握：

1. 样本可以表示为向量。
2. 数据集可以表示为矩阵。
3. 内积可以表示线性打分和相似度。
4. 范数可以表示大小和复杂度。
5. PCA 与特征向量、方差方向有关。
