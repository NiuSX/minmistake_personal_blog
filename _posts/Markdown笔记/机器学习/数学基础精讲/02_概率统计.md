# 概率统计

概率统计是机器学习处理不确定性的语言。分类概率、噪声、泛化误差、贝叶斯模型、最大似然估计，都离不开概率思想。

## 一、概率是什么

概率描述事件发生的不确定性。

```text
P(A) = 事件 A 发生的概率
```

概率在 0 到 1 之间。0 表示不可能，1 表示必然发生。

在机器学习中，模型常输出概率。例如：

```text
P(流失=是 | 用户特征) = 0.82
```

这不是说用户一定会流失，而是模型认为流失风险较高。

## 二、条件概率

条件概率：

```text
P(A|B)
```

表示在 B 已经发生的条件下，A 发生的概率。

机器学习中最常见的是：

```text
P(y|x)
```

也就是给定样本特征 x，标签 y 的概率。

## 三、贝叶斯公式

贝叶斯公式：

```text
P(c|x) = P(x|c)P(c) / P(x)
```

含义：

1. `P(c)` 是先验概率，还没看特征前类别的概率。
2. `P(x|c)` 是似然，类别为 c 时看到特征 x 的概率。
3. `P(c|x)` 是后验概率，看完特征后类别为 c 的概率。

贝叶斯分类器就是选择后验概率最大的类别。

## 四、期望

期望是随机变量的平均水平。

如果一个随机变量可能取不同值，每个值有不同概率，期望就是加权平均。

机器学习中的损失函数经常写成期望形式：

```text
泛化误差 = 在未知数据分布上的期望损失
```

训练集上的平均损失是对这个期望的近似。

## 五、方差

方差衡量随机变量围绕均值波动的程度。

方差大，说明数据分散；方差小，说明数据集中。

偏差-方差分解中的“方差”描述模型对训练集变化的敏感程度。模型太复杂时，训练数据稍微变一点，模型就变化很大，方差高。

## 六、极大似然估计

极大似然估计的思想是：选择一组参数，让已经观察到的数据出现概率最大。

如果参数是 `theta`，数据是 `D`，似然是：

```text
P(D|theta)
```

极大似然就是找：

```text
argmax_theta P(D|theta)
```

很多机器学习算法都可以从极大似然角度理解。比如线性回归在高斯噪声假设下等价于最小二乘，逻辑回归常通过最大化条件似然训练。

## 七、统计估计为什么会有误差

训练数据只是总体的一部分。用样本估计总体规律必然有随机误差。

样本越少，估计越不稳定；样本越多，估计通常越可靠。但如果样本有偏，即使数量很多，也可能学到偏差规律。

## 八、新手重点

1. 理解条件概率和贝叶斯公式。
2. 知道期望是平均水平，方差是波动程度。
3. 明白最大似然是在找最能解释数据的参数。
4. 知道训练集估计不等于真实总体。
5. 理解概率输出不是绝对判断，而是不确定性表达。
