# 微积分与梯度

微积分在机器学习中主要用于优化。模型训练的核心常常是：损失函数太大，要调整参数让损失下降。梯度告诉我们应该往哪个方向调整。

## 一、导数的直觉

导数表示函数在某一点的变化率。

如果导数大于 0，说明往右走函数值上升。

如果导数小于 0，说明往右走函数值下降。

如果导数接近 0，说明函数在这一点附近变化很小，可能接近极小值、极大值或鞍点。

## 二、偏导数

机器学习模型通常有很多参数。损失函数不是只依赖一个变量，而是依赖很多权重。

偏导数表示：只改变其中一个参数，其他参数固定时，损失如何变化。

例如：

```text
Loss(w1, w2, b)
```

对 `w1` 的偏导表示只调整 `w1` 时损失变化方向。

## 三、梯度

梯度是所有偏导数组成的向量：

```text
grad = [dL/dw1, dL/dw2, ..., dL/dwd]
```

梯度方向是函数上升最快的方向。因此梯度下降使用：

```text
参数 = 参数 - 学习率 * 梯度
```

减去梯度，就是朝损失下降最快的局部方向走。

## 四、学习率

学习率决定每次更新走多远。

学习率太大，可能越过最优点，甚至发散。

学习率太小，训练会很慢，可能卡在不理想区域。

调学习率是训练神经网络和梯度模型的重要工作。

## 五、链式法则

链式法则用于复合函数求导。

神经网络是多层函数嵌套：

```text
输入 -> 第一层 -> 激活 -> 第二层 -> 激活 -> 输出 -> 损失
```

反向传播就是用链式法则，从损失开始一层层往回计算每个参数对损失的影响。

## 六、反向传播直觉

反向传播不是一种神秘算法，它做两件事：

1. 前向传播：根据当前参数计算预测和损失。
2. 反向传播：计算损失对每个参数的梯度。

有了梯度，优化器就能更新参数。

## 七、梯度为零不一定是最优

梯度为零可能是局部最小值、局部最大值，也可能是鞍点。深度学习中的损失面复杂，优化不一定找到全局最优，但通常能找到足够好的解。

## 八、新手重点

1. 导数表示变化率。
2. 偏导表示某个参数对损失的影响。
3. 梯度是所有偏导的集合。
4. 梯度下降沿着负梯度方向更新参数。
5. 反向传播是链式法则的高效实现。
