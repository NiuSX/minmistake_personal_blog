# 最优化基础

机器学习训练通常可以理解为一个最优化问题：给定数据和模型，找到一组参数，使损失函数尽可能小。

## 一、目标函数

目标函数也叫优化目标。常见形式：

```text
目标 = 数据损失 + 正则项
```

数据损失衡量模型对训练样本拟合得好不好。

正则项限制模型复杂度，防止为了训练集表现过度调整参数。

## 二、损失函数

损失函数衡量单个样本或一批样本预测得多差。

回归常用平方损失：

```text
(y - y_hat)^2
```

分类常用交叉熵或对数损失。

损失函数选择会影响模型行为。平方损失会放大大误差，交叉熵会强烈惩罚自信但错误的预测。

## 三、梯度下降

梯度下降是最常见的优化方法之一：

```text
theta = theta - learning_rate * gradient
```

其中 `theta` 是参数，`gradient` 是损失对参数的梯度。

直觉：如果梯度告诉你往某个方向损失会上升，那么往反方向走就能降低损失。

## 四、批量、小批量和随机梯度下降

批量梯度下降每次使用全部训练样本计算梯度，稳定但慢。

随机梯度下降每次使用一个样本，速度快但波动大。

小批量梯度下降每次使用一小批样本，是深度学习中最常用的折中。

## 五、凸优化和非凸优化

凸问题像一个碗，局部最小值就是全局最小值，优化相对容易。

非凸问题地形复杂，可能有很多局部极小值和鞍点。神经网络通常是非凸优化问题。

线性回归、逻辑回归等许多经典模型在常见设置下有较好的优化性质；神经网络更依赖初始化、学习率、优化器和数据规模。

## 六、正则化

正则化是在目标函数中加入复杂度惩罚。

L2 正则：

```text
Loss + lambda ||w||_2^2
```

L1 正则：

```text
Loss + lambda ||w||_1
```

`lambda` 越大，惩罚越强，模型越简单，但太大可能欠拟合。

## 七、早停

早停也是一种正则化思想。训练过程中，如果验证集性能不再提升，就停止训练，避免模型继续记忆训练集噪声。

## 八、优化与泛化不是一回事

优化好表示训练目标被更好地最小化。泛化好表示未知样本表现好。训练损失很低并不一定泛化好。

因此机器学习不是只追求损失最低，还要通过验证集、正则化、模型选择和误差分析控制泛化。

## 九、新手重点

1. 训练模型就是优化目标函数。
2. 损失函数决定模型惩罚什么错误。
3. 梯度下降根据梯度更新参数。
4. 正则化限制模型复杂度。
5. 优化能力和泛化能力要分开看。
