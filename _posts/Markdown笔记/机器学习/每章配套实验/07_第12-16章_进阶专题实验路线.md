## 一、计算学习理论实验：样本量与泛化

目标：观察样本量增加时，训练误差和测试误差如何变化。

实验思路：

1. 固定一个模型，例如逻辑回归或决策树。
2. 使用不同训练样本数量。
3. 记录训练分数和测试分数。
4. 画出学习曲线。

```python
from sklearn.model_selection import learning_curve
from sklearn.linear_model import LogisticRegression

train_sizes, train_scores, test_scores = learning_curve(
    LogisticRegression(max_iter=1000),
    X,
    y,
    cv=5,
    train_sizes=[0.1, 0.3, 0.5, 0.7, 1.0],
    scoring="accuracy"
)
```

理解重点：

1. 样本少时方差是否更大？
2. 增加数据是否能缩小训练和验证差距？
3. 模型复杂度如何影响曲线形状？

## 二、半监督学习实验：伪标签

目标：模拟少量有标签数据和大量无标签数据。

实验思路：

1. 只保留少量标签样本训练基线模型。
2. 用模型给无标签样本打伪标签。
3. 只选择高置信样本加入训练。
4. 比较加入前后验证集表现。

需要记录：

1. 伪标签阈值。
2. 加入样本数量。
3. 验证集指标变化。
4. 是否出现错误放大。

## 五、强化学习实验：网格世界

目标：实现最小版 Q-learning。

环境设定：

1. 4×4 网格。
2. 起点在左上角。
3. 终点在右下角。
4. 每走一步奖励 -1。
5. 到达终点奖励 0。

核心伪代码：

```text
初始化 Q(s,a)=0
重复多个 episode：
    从起点开始
    按 epsilon-greedy 选动作
    执行动作，得到奖励和下一状态
    用 Q-learning 更新 Q(s,a)
    直到终点
```

理解重点：

1. 什么是状态？
2. 什么是动作？
3. 奖励如何影响策略？
4. epsilon 如何控制探索？
5. Q 表如何逐渐学到路径价值？
