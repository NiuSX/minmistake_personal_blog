# 机器学习英文术语对照

## 基础概念

| 中文 | 英文 | 说明 |
|---|---|---|
| 机器学习 | Machine Learning | 从数据中学习规律 |
| 数据集 | Dataset | 样本集合 |
| 样本 | Sample / Instance | 一个训练或预测对象 |
| 特征 | Feature | 描述样本的变量 |
| 标签 | Label / Target | 监督学习中的真实输出 |
| 模型 | Model | 从输入到输出的映射 |
| 参数 | Parameter | 模型从数据中学习的量 |
| 超参数 | Hyperparameter | 训练前设置或通过验证选择的量 |
| 训练 | Training / Fitting | 用数据学习模型 |
| 预测 | Prediction / Inference | 用模型输出结果 |
| 泛化 | Generalization | 在未知数据上的表现 |
| 过拟合 | Overfitting | 训练好、测试差 |
| 欠拟合 | Underfitting | 训练和测试都差 |

## 评估指标

| 中文 | 英文 | 说明 |
|---|---|---|
| 准确率 | Accuracy | 总体预测正确比例 |
| 查准率 | Precision | 预测为正中有多少真为正 |
| 查全率 | Recall | 真实为正中有多少被找出 |
| F1 值 | F1-score | 查准率和查全率的调和平均 |
| 混淆矩阵 | Confusion Matrix | 分类结果统计表 |
| ROC 曲线 | ROC Curve | TPR 与 FPR 的关系 |
| AUC | Area Under Curve | ROC 曲线下面积 |
| 平均绝对误差 | MAE | 回归平均绝对误差 |
| 均方误差 | MSE | 回归平方误差平均 |
| 均方根误差 | RMSE | MSE 开根号 |
| 决定系数 | R-squared / R2 | 回归解释程度 |

## 算法名称

| 中文 | 英文 | 说明 |
|---|---|---|
| 线性回归 | Linear Regression | 线性回归模型 |
| 逻辑回归 | Logistic Regression | 常用于分类 |
| 决策树 | Decision Tree | 树形条件判断模型 |
| 随机森林 | Random Forest | 多棵随机树集成 |
| 支持向量机 | Support Vector Machine / SVM | 最大间隔分类 |
| 朴素贝叶斯 | Naive Bayes | 条件独立假设的贝叶斯分类 |
| 神经网络 | Neural Network | 多层非线性模型 |
| K 均值 | K-means | 原型聚类 |
| 主成分分析 | Principal Component Analysis / PCA | 线性降维 |
| 强化学习 | Reinforcement Learning | 交互式序列决策学习 |

## 优化与正则

| 中文 | 英文 | 说明 |
|---|---|---|
| 损失函数 | Loss Function | 衡量预测错误 |
| 目标函数 | Objective Function | 要优化的总体目标 |
| 梯度下降 | Gradient Descent | 沿负梯度更新参数 |
| 学习率 | Learning Rate | 每次更新步长 |
| 正则化 | Regularization | 限制模型复杂度 |
| L1 正则 | L1 Regularization | 倾向稀疏 |
| L2 正则 | L2 Regularization | 缩小权重 |
| 早停 | Early Stopping | 验证集不提升时停止训练 |

## 概率与统计

| 中文 | 英文 | 说明 |
|---|---|---|
| 先验概率 | Prior Probability | 观察数据前的概率 |
| 似然 | Likelihood | 参数下观察到数据的可能性 |
| 后验概率 | Posterior Probability | 观察数据后的概率 |
| 极大似然估计 | Maximum Likelihood Estimation / MLE | 最大化数据似然 |
| 期望 | Expectation | 随机变量平均水平 |
| 方差 | Variance | 波动程度 |
| 条件独立 | Conditional Independence | 给定条件后变量独立 |
