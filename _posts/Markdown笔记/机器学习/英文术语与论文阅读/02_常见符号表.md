# 常见符号表

机器学习公式中符号很多，但常见符号是固定的。理解符号后，公式阅读会轻松很多。

## 数据相关

| 符号 | 含义 |
|---|---|
| `D` | 数据集 |
| `m` 或 `n` | 样本数量 |
| `d` | 特征维度 |
| `x` | 一个样本的特征向量 |
| `x_i` | 第 i 个样本 |
| `x_j` | 第 j 个特征，有时也表示第 j 个样本，需看上下文 |
| `X` | 数据矩阵 |
| `y` | 标签或输出 |
| `y_i` | 第 i 个样本的标签 |
| `C` 或 `c` | 类别 |

## 模型相关

| 符号 | 含义 |
|---|---|
| `f` | 模型或预测函数 |
| `h` | 假设函数 |
| `H` | 假设空间 |
| `w` | 权重向量 |
| `b` | 偏置 |
| `theta` | 参数集合 |
| `y_hat` | 预测值 |
| `p` | 概率 |

## 损失与优化

| 符号 | 含义 |
|---|---|
| `L` | 损失函数 |
| `J` | 目标函数 |
| `E` | 误差，有时也表示期望 |
| `lambda` | 正则化强度 |
| `alpha` | 学习率或显著性水平，需看上下文 |
| `grad` | 梯度 |
| `||w||` | 向量范数 |

## 概率统计

| 符号 | 含义 |
|---|---|
| `P(A)` | 事件 A 的概率 |
| `P(A|B)` | B 发生条件下 A 的概率 |
| `E[X]` | 随机变量 X 的期望 |
| `Var(X)` | 随机变量 X 的方差 |
| `mu` | 均值 |
| `sigma` | 标准差 |
| `Sigma` | 协方差矩阵 |

## 聚类与降维

| 符号 | 含义 |
|---|---|
| `K` | 聚类数量或近邻数量 |
| `mu_k` | 第 k 个簇中心 |
| `c_i` | 第 i 个样本所属簇 |
| `z` | 降维后的表示或隐变量 |
| `W` | 投影矩阵或权重矩阵 |

## 强化学习

| 符号 | 含义 |
|---|---|
| `s` | 状态 |
| `a` | 动作 |
| `r` | 奖励 |
| `pi` | 策略 |
| `V(s)` | 状态价值函数 |
| `Q(s,a)` | 动作价值函数 |
| `gamma` | 折扣因子 |
| `epsilon` | 探索概率或误差容忍，需看上下文 |

## 阅读公式的建议

1. 先找数据符号和参数符号。
2. 再找目标函数。
3. 看求和是在对样本求和、对特征求和，还是对类别求和。
4. 看最大化或最小化对象是什么。
5. 不要脱离上下文死记符号，同一个符号在不同章节可能含义不同。
