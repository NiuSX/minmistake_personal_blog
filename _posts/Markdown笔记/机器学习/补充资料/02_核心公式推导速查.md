# 核心公式推导速查

这份文档不是为了让你机械背公式，而是帮助你看懂公式在表达什么。学习机器学习公式时，先问三件事：

1. 这个公式衡量什么？
2. 哪些量是已知的，哪些量是要学习的？
3. 优化这个公式会让模型产生什么行为？

## 一、经验误差与泛化误差

经验误差：

```text
E_train = 模型在训练集上的平均损失
```

泛化误差：

```text
E_general = 模型在未知数据分布上的期望损失
```

训练时我们只能直接看到经验误差，但真正关心泛化误差。过拟合就是经验误差低、泛化误差高。

## 二、平方损失与线性回归

线性回归假设：

```text
f(x) = w^T x + b
```

平方损失：

```text
L = sum_i (y_i - f(x_i))^2
```

直觉：预测值和真实值差得越远，惩罚越大，而且大误差会被平方放大。

为什么最小二乘合理？如果假设噪声服从均值为 0 的高斯分布，那么最小化平方损失等价于最大化似然。

## 三、逻辑回归与对数几率

逻辑回归建模的是概率：

```text
P(y=1|x) = 1 / (1 + exp(-(w^T x + b)))
```

其中 `w^T x + b` 是对数几率：

```text
log(P(y=1|x) / P(y=0|x)) = w^T x + b
```

直觉：线性模型先给样本打一个分，再用 sigmoid 把分数压到 0 到 1 之间，作为属于正类的概率。

逻辑回归常用对数损失。预测越自信但越错，惩罚越大。

## 四、信息熵

信息熵衡量不确定性：

```text
H(D) = - sum_k p_k log p_k
```

如果一个集合中类别非常混杂，熵高；如果几乎都是同一类，熵低。

决策树用信息增益选择划分属性：

```text
Gain(D, a) = H(D) - sum_v (|D_v| / |D|) H(D_v)
```

直觉：选择一个属性划分后，如果子集合更纯，不确定性下降多，信息增益就大。

## 五、基尼指数

基尼指数：

```text
Gini(D) = 1 - sum_k p_k^2
```

直觉：从集合中随机抽两个样本，它们类别不一致的概率越高，基尼指数越大。CART 决策树常用它选择划分。

## 六、贝叶斯公式

贝叶斯公式：

```text
P(c|x) = P(x|c) P(c) / P(x)
```

分类时比较不同类别的后验概率：

```text
argmax_c P(c|x)
```

因为对所有类别来说 `P(x)` 相同，所以可以比较：

```text
argmax_c P(x|c) P(c)
```

朴素贝叶斯进一步假设特征条件独立：

```text
P(x|c) = product_j P(x_j|c)
```

这个假设很强，但能大幅降低参数估计难度。

## 七、SVM 最大间隔

线性分类边界：

```text
w^T x + b = 0
```

几何间隔与 `||w||` 相关。最大化间隔等价于最小化：

```text
1/2 ||w||^2
```

并满足分类约束：

```text
y_i(w^T x_i + b) >= 1
```

软间隔加入松弛变量，允许部分样本违反间隔：

```text
min 1/2 ||w||^2 + C sum_i xi_i
```

直觉：第一项希望边界简单、间隔大；第二项惩罚训练错误或间隔违反；`C` 控制二者权衡。

## 八、核函数

核函数：

```text
K(x_i, x_j) = phi(x_i)^T phi(x_j)
```

直觉：不显式计算高维映射 `phi(x)`，而是直接计算高维空间中的内积。核技巧让 SVM 能处理非线性边界。

## 九、K-means 目标函数

K-means 最小化簇内平方误差：

```text
min sum_i ||x_i - mu_{c_i}||^2
```

其中 `mu_{c_i}` 是样本所属簇的中心。

直觉：每个样本尽量靠近自己的簇中心。算法在“分配样本到最近中心”和“重新计算中心”之间交替。

## 十、PCA 主成分分析

PCA 寻找方差最大的投影方向。第一主成分是让投影后数据方差最大的单位向量。

直觉：如果数据在某个方向上变化最大，这个方向保留的信息通常最多。PCA 用少数主成分近似原始数据，达到降维和去噪目的。

## 十一、L1 与 L2 正则化

L2 正则：

```text
Loss + lambda ||w||_2^2
```

倾向于让权重变小，但通常不变成 0。

L1 正则：

```text
Loss + lambda ||w||_1
```

倾向于让部分权重变成 0，因此能产生稀疏解。

直觉：正则化是在告诉模型，不要为了训练集上的一点收益使用过大的复杂度。

## 十二、贝尔曼方程

状态价值：

```text
V(s) = E[ R_{t+1} + gamma V(S_{t+1}) | S_t = s ]
```

动作价值：

```text
Q(s,a) = E[ R_{t+1} + gamma max_a' Q(S_{t+1}, a') ]
```

直觉：一个状态的长期价值，等于眼前奖励加上下一个状态的折扣价值。强化学习中的许多算法都围绕这个递推关系展开。

## 十三、公式学习方法

每个公式按下面方式复习：

1. 用一句话说它在衡量什么。
2. 标出哪些变量是数据，哪些变量是参数。
3. 说明最大化或最小化它会产生什么效果。
4. 找一个小例子手算一遍。
5. 说出它的局限。
