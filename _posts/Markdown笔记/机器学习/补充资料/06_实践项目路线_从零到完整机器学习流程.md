# 实践项目路线：从零到完整机器学习流程

学机器学习不能只读笔记。真正掌握至少要完整做过一个项目，从问题定义到数据处理、建模、评估、误差分析和复盘。

## 一、完整流程总览

```text
问题定义
  -> 数据收集
  -> 数据理解
  -> 数据清洗
  -> 特征工程
  -> 数据划分
  -> 建立基线
  -> 训练模型
  -> 调参验证
  -> 误差分析
  -> 最终测试
  -> 总结复盘
```

任何跳过评估和误差分析的项目，都很难真正学到东西。

## 二、第一步：问题定义

先写清楚：

1. 这是分类、回归、聚类、降维还是强化学习问题？
2. 样本单位是什么？
3. 标签如何定义？
4. 错误代价是什么？
5. 成功标准是什么？

例如做“房价预测”，不能只说目标是预测房价。要明确样本是一套房，标签是成交价还是挂牌价，指标用 MAE 还是 RMSE，是否更关注高价房误差。

## 三、第二步：数据理解

拿到数据后不要直接训练模型。先做探索：

1. 每列含义是什么？
2. 缺失值比例是多少？
3. 数值特征分布如何？
4. 类别特征有哪些取值？
5. 标签是否不平衡？
6. 是否有异常值？
7. 是否存在时间泄漏或目标泄漏？

很多模型效果问题不是算法问题，而是数据理解不充分。

## 四、第三步：数据清洗

常见清洗动作：

1. 处理缺失值。
2. 处理异常值。
3. 删除重复样本。
4. 修正非法取值。
5. 统一单位和格式。
6. 处理类别特征中的低频类别。

清洗要记录原因。不要为了让模型分数变高随意删除数据。

## 五、第四步：特征工程

特征工程的目标是让模型更容易发现规律。

常见方式：

1. 数值特征标准化或归一化。
2. 偏态特征做对数变换。
3. 类别特征做 one-hot 编码。
4. 时间特征拆成月份、星期、小时。
5. 构造比例、差值、频率、历史统计特征。
6. 对高维特征做选择或降维。

特征工程必须避免使用未来信息。例如预测未来流失时，不能使用未来才知道的行为。

## 六、第五步：数据划分

标准划分：

1. 训练集：训练模型。
2. 验证集：选择模型和调参。
3. 测试集：最终评估。

如果数据有时间顺序，应按时间划分，不能随机打乱。否则会让未来数据泄漏到训练中。

## 七、第六步：建立基线

基线模型非常重要。没有基线，就不知道复杂模型是否真的有价值。

分类任务可以用：

1. 多数类预测。
2. 逻辑回归。
3. 朴素贝叶斯。
4. 简单决策树。

回归任务可以用：

1. 预测均值或中位数。
2. 线性回归。
3. 岭回归。

聚类任务可以先用 K-means 或层次聚类做初步结构探索。

## 八、第七步：模型训练与调参

调参前要明确哪些超参数影响什么。

例如：

1. 正则强度越大，模型越简单。
2. 决策树深度越大，越容易过拟合。
3. 随机森林树越多，通常更稳定但更慢。
4. SVM 的 C 越大，对训练错误惩罚越强。
5. K-means 的 K 决定簇数量。

调参不要在测试集上做。测试集只用于最后一次评估。

## 九、第八步：误差分析

误差分析是提升模型最有效的步骤。

分类任务可以看：

1. 混淆矩阵。
2. 哪些类别最容易混淆。
3. 错误样本是否集中在某些人群、时间段、特征范围。
4. 高置信错误样本有什么共同点。

回归任务可以看：

1. 残差分布。
2. 大误差样本。
3. 误差是否随真实值变大而变大。
4. 是否存在系统性低估或高估。

聚类任务可以看：

1. 簇内样本是否相似。
2. 簇间是否有清晰差异。
3. 聚类结果是否稳定。
4. 是否能给每个簇命名和解释。

## 十、第九步：最终测试和复盘

最终测试只能做一次或尽量少做。测试结果出来后，要记录：

1. 使用了哪些数据。
2. 做了哪些预处理。
3. 尝试了哪些模型。
4. 最终指标是多少。
5. 模型主要错误来自哪里。
6. 如果继续改进，优先做什么。

## 十一、推荐练手项目

### 项目 1：鸢尾花分类

目标：理解分类、训练集测试集、准确率、决策边界。

推荐算法：逻辑回归、决策树、SVM。

### 项目 2：房价预测

目标：理解回归、特征工程、均方误差、残差分析。

推荐算法：线性回归、岭回归、随机森林回归。

### 项目 3：垃圾短信分类

目标：理解文本特征、高维稀疏数据、查准率和查全率。

推荐算法：朴素贝叶斯、逻辑回归。

### 项目 4：客户分群

目标：理解无监督学习、标准化、聚类解释。

推荐算法：K-means、层次聚类、DBSCAN。

### 项目 5：降维可视化

目标：理解 PCA、t-SNE、可视化解释边界。

推荐数据：手写数字、文本向量、客户特征。

## 十二、项目报告模板

```text
1. 问题定义
2. 数据来源与字段说明
3. 数据探索发现
4. 数据清洗与特征工程
5. 数据划分方式
6. 基线模型结果
7. 候选模型对比
8. 最终模型与指标
9. 误差分析
10. 局限与后续改进
```

按这个模板写完一个项目，比只看十遍算法介绍更有效。
