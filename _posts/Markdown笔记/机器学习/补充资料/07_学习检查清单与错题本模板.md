# 学习检查清单与错题本模板

机器学习学习中最大的错觉是“看懂了”。真正掌握至少要能做到四件事：

1. 能用自己的话解释概念。
2. 能手推核心公式的直觉。
3. 能判断算法适用场景。
4. 能在项目中发现问题并改进。

## 一、每章学习检查清单

每学完一章，用下面清单自测。

### 1. 核心问题

我能不能用一句话说清楚本章解决什么问题？

例如：

```text
模型评估与选择：解决如何判断模型在未知数据上好不好，以及如何在多个模型中做选择。
```

如果只能复述标题，说明还没真正理解。

### 2. 核心概念

我能不能列出本章 5 到 10 个关键词，并解释它们之间的关系？

例如模型评估章节：

```text
经验误差 -> 泛化误差 -> 过拟合 -> 验证集 -> 性能度量 -> 模型比较
```

### 3. 核心公式

我能不能说明本章公式中每个量的含义？

不要只背公式，要能回答：

1. 这个公式输入什么？
2. 输出什么？
3. 为什么这样定义？
4. 优化它会带来什么行为？

### 4. 算法流程

我能不能不用看书，写出算法大致步骤？

例如 K-means：

1. 初始化 K 个中心。
2. 把每个样本分配给最近中心。
3. 根据分配结果重新计算中心。
4. 重复直到中心变化很小或达到迭代次数。

### 5. 适用场景

我能不能说出这个算法适合什么数据、不适合什么数据？

只知道优点不够，还要知道限制。限制往往决定项目中是否能用。

### 6. 常见误区

我能不能列出本章至少 3 个新手误区？

例如聚类：

1. 把聚类结果当真实标签。
2. 不做标准化就计算距离。
3. 只看轮廓系数，不看业务解释。

### 7. 实践验证

我有没有用一个小数据集跑过本章算法？

理论理解和代码实践必须互相验证。哪怕只是鸢尾花、手写数字、房价这样的小数据集，也比纯阅读有效。

## 二、错题本模板

建议每个错题按下面格式记录。

```text

正确答案：

相关知识点：

复习日期：
第 1 次：
第 2 次：
第 3 次：
```

## 三、常见错因分类

### 1. 概念边界不清

例如混淆查准率和查全率、混淆生成式和判别式、混淆参数和超参数。

解决方法：整理对照表，用自己的例子解释。

### 2. 只背结论，不懂条件

例如背下“随机森林不容易过拟合”，但不知道这是相对单棵深树而言，也不知道噪声极大、特征泄漏或数据分布变化时仍然会出问题。

解决方法：每个结论后面补一句“在什么条件下成立”。

### 3. 公式符号恐惧

看到公式直接跳过，导致后面算法只剩口号。

解决方法：不急着推导，先标注每个符号含义，再看公式整体在衡量什么。

### 4. 缺少实践反馈

只读文字会觉得每个算法都懂，真正跑数据时才发现不会处理缺失值、不会选指标、不会看错误样本。

解决方法：每学完一类算法，至少做一个小实验。

## 四、周复习安排

每周建议安排一次复盘：

1. 回顾本周学了哪几章。
2. 每章写一句话总结。
3. 每章列 3 个关键词。
4. 选 2 道题重新做。
5. 记录一个仍然不懂的问题。
6. 找一个小实验验证一个算法。

## 五、阶段性验收标准

### 入门阶段

能解释样本、特征、标签、模型、损失、泛化、过拟合。

### 监督学习阶段

能比较逻辑回归、决策树、朴素贝叶斯、SVM、神经网络、集成学习的适用场景。

### 无监督阶段

能解释聚类、降维、特征选择的区别，并知道它们如何帮助建模。

### 进阶阶段

能理解学习理论、半监督、概率图、规则学习、强化学习分别扩展了机器学习的哪一部分能力。

### 实践阶段

能独立完成一个小项目，并写出包含误差分析的项目报告。
