# 机器学习补充资料索引

这个文件夹不是按章节重讲，而是补齐学习闭环：

1. 把全书知识点串成一张地图。
2. 把核心公式整理成可复习的速查表。
3. 把常见算法按任务、数据特点和优缺点放到一起比较。
4. 把新手最容易混淆的概念做成对照表。
5. 给每章配复习题和参考答案。
6. 给出从零完成机器学习项目的实践路线。
7. 提供学习检查清单和错题本模板。

推荐读法：

1. 学完 1-2 章后，读 `01_全书知识地图_从数据到泛化.md`。
2. 学每个算法时，同步查 `02_核心公式推导速查.md`。
3. 学完监督学习后，读 `03_算法选型手册.md`。
4. 每学完三章，读一次 `04_易混概念对照表.md`。
5. 章节复习时使用 `05_章节复习题与参考答案.md`。
6. 准备动手实践时读 `06_实践项目路线_从零到完整机器学习流程.md`。
7. 每周复盘时使用 `07_学习检查清单与错题本模板.md`。
