## 为什么要拆成“超级精讲”

原来的 `01_绪论.md` 到 `16_强化学习.md` 是按章整理的详细笔记，适合建立框架。但如果要做到“每个知识点万字精讲级别”，就不能继续把所有内容塞进一个章节文件里。

更合理的结构是：

```text
机器学习/
  超级精讲/
    第01章_绪论/
      01_本章总览.md
      02_机器学习到底是什么.md
      03_基本术语全拆解.md
      04_假设空间.md
      05_归纳偏好与没有免费的午餐.md
      06_本章复习与练习.md
```

后续每章也会按小节拆分。这样每个知识点都能独立扩写到足够深。

## 使用建议

1. 先读根目录下 `机器学习/01_绪论.md` 到 `16_强化学习.md`，建立完整框架。
2. 再进入本目录按章节阅读“超级精讲”，重点看概念拆解、公式直觉、常见误区和练习题。
3. 每章读完后，自己用一句话解释核心问题：这一章到底在解决什么机器学习困难。
4. 对公式较多的章节，不要只背结论，要反复追问：这个量在优化什么、约束什么、惩罚什么、衡量什么。
