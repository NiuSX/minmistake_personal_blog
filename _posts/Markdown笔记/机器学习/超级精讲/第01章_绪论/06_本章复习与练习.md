## 2. 判断题

### 2.1 训练集准确率越高，模型一定越好。

错误。

训练集准确率高可能是过拟合。模型好不好要看泛化性能。

### 2.2 没有标签的数据不能用于机器学习。

错误。

无监督学习和半监督学习都可以利用无标签数据。

### 2.3 机器学习算法一定需要归纳偏好。

正确。

有限训练数据无法唯一决定未知样本上的预测，必须有偏好。

### 2.4 没有免费的午餐定理说明所有算法在任何实际任务上都一样。

错误。

它说的是对所有可能问题平均时没有算法永远占优；具体任务上算法差异很大。

## 3. 简答题

### 3.1 为什么说机器学习的目标是泛化，而不是记忆？

参考思路：

训练样本只是过去经验。模型最终要处理未来新样本。如果只记住训练集，遇到新样本就无法可靠判断。泛化能力才表示模型学到了可迁移规律。

### 3.2 为什么数据不能唯一决定模型？

参考思路：

训练数据有限，而可能模型无限。多个模型可能在训练集上完全一致，但对未知样本预测不同。因此必须依赖归纳偏好选择模型。

### 3.3 假设空间太大会怎样？太小又会怎样？

参考思路：

太大：表达能力强，但容易过拟合，需要更多数据。

太小：稳定简单，但可能无法表示真实规律，导致欠拟合。

## 4. 场景拆解练习

请把下面任务拆成机器学习五要素：

```text
某平台希望预测用户未来 30 天是否会流失。
```

需要回答：

1. 样本是什么？
2. 特征可能有哪些？
3. 标签如何定义？
4. 这是分类还是回归？
5. 性能指标用什么？

参考答案：

1. 样本：一个用户在某个观察时间点的记录。
2. 特征：最近登录次数、浏览时长、购买次数、投诉次数、距离上次登录天数等。
3. 标签：未来 30 天是否未登录或未购买。
4. 二分类。
5. AUC、Recall、F1、干预转化率等。

## 5. 术语配对练习

把左边和右边对应起来：

| 左边 | 右边 |
| --- | --- |
| 样本 | 一条数据记录 |
| 特征 | 描述样本的变量 |
| 标签 | 训练数据中的正确答案 |
| 模型 | 从数据中学到的规律 |
| 学习算法 | 从数据产生模型的方法 |
| 泛化 | 未见样本上的表现能力 |

## 6. 思考题

### 6.1 为什么简单模型有时比复杂模型好？

提示：

- 数据少。
- 噪声多。
- 复杂模型容易过拟合。
- 简单模型归纳偏好更强。

### 6.2 为什么复杂模型有时又必须使用？

提示：

- 真实规律复杂。
- 特征高维。
- 输入是图像、语音、自然语言。
- 简单模型欠拟合。

### 6.3 如何理解“没有免费午餐”不等于“算法研究没意义”？

提示：

算法研究的价值在于为现实中常见任务设计更合适的归纳偏好，而不是寻找对所有可能任务都最优的算法。
