# 第 3 章 线性模型：超级精讲总览

## 1. 为什么线性模型是机器学习基本功

线性模型是机器学习中最基础、最常用、也最值得反复理解的一类模型。

很多人学习机器学习时，会觉得线性模型太简单，想直接跳到神经网络、集成学习、大模型。但这是错误的。线性模型之所以重要，是因为它把机器学习中最核心的一些思想用最清楚的方式展示出来：

- 参数如何表示特征影响。
- 损失函数如何定义学习目标。
- 优化如何寻找最优参数。
- 分类边界如何产生。
- 正则化如何控制复杂度。
- 概率输出如何用于决策。

后面很多复杂模型都可以看成线性模型的扩展、组合或非线性变形。

## 3. 一句话理解线性模型

线性模型的核心是：

```text
每个特征乘以一个权重，再把所有贡献加起来。
```

数学形式：

```text
f(x) = w1*x1 + w2*x2 + ... + wd*xd + b
```

向量形式：

```text
f(x) = w^T x + b
```

其中：

- x 是特征向量。
- w 是权重向量。
- b 是偏置。
- f(x) 是模型输出。

## 4. 线性模型为什么可解释

在线性模型中，每个权重对应一个特征。

如果特征已经标准化，那么权重大小可以大致反映特征影响强弱：

- 权重大于 0：特征越大，输出越大。
- 权重小于 0：特征越大，输出越小。
- 权重接近 0：特征影响较弱。

这让线性模型比很多复杂模型更容易解释。
