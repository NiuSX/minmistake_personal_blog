# 线性回归：从直线拟合到最小二乘

## 1. 线性回归解决什么问题

线性回归用于预测连续数值。

例子：

- 根据面积、位置、楼层预测房价。
- 根据广告投放、节假日、渠道预测销量。
- 根据温度、湿度、风速预测用电量。

它假设目标值可以由特征的线性组合近似表示。

## 2. 从一维例子理解

假设只用房屋面积预测房价。

模型可以写成：

```text
房价 = w * 面积 + b
```

在二维平面中：

- 横轴是面积。
- 纵轴是房价。
- 每套房子是一个点。

线性回归要找一条直线，让它尽可能贴近这些点。

## 3. 多维线性回归

如果有多个特征：

```text
房价 = w1*面积 + w2*楼层 + w3*房龄 + w4*距离地铁 + b
```

这时不再是一条二维直线，而是高维空间中的超平面。

但思想不变：用每个特征的加权和预测目标。

## 4. 什么是残差

残差是预测值和真实值的差：

```text
残差 = 真实值 - 预测值
```

如果模型预测房价 300 万，真实成交 320 万：

```text
残差 = 20 万
```

训练模型就是希望整体残差尽量小。

## 5. 为什么用平方误差

平方误差：

```text
(真实值 - 预测值)^2
```

使用平方有几个原因：

1. 正负误差不会抵消。
2. 大误差会被更重惩罚。
3. 函数光滑，便于求导优化。
4. 如果噪声服从高斯分布，最小平方误差等价于最大似然估计。

## 6. 最小二乘法

线性回归常用最小二乘法：

```text
选择参数 w 和 b，使所有样本平方误差之和最小。
```

目标函数：

```text
min Σ(yi - f(xi))^2
```

其中：

- yi 是真实值。
- f(xi) 是预测值。

## 7. 几何直觉

在一维特征下，最小二乘就是找一条线，让所有点到线的纵向距离平方和最小。

平方会让远离直线的点影响更大。

这也是线性回归对异常值敏感的原因。

## 8. 异常值问题

假设大多数房价在 200-500 万之间，但有一条数据因为录入错误写成 5000 万。

平方误差会把这个异常点的误差放大很多，模型可能被它拉偏。

解决方法：

- 清洗异常值。
- 使用 MAE 等更稳健损失。
- 使用鲁棒回归。
- 对目标做截尾或变换。

## 9. 特征尺度影响

线性回归本身可以处理不同尺度，但如果使用正则化或梯度下降，特征尺度会影响训练。

例如：

- 面积范围 20-200。
- 距离地铁范围 0-20000 米。

尺度差异大时，参数解释和优化都会受影响。

常用标准化：

```text
新值 = (原值 - 均值) / 标准差
```

## 10. 线性回归的优点

- 简单。
- 可解释。
- 训练速度快。
- 可作为强基线。
- 适合关系近似线性的任务。

## 11. 线性回归的局限

- 只能表达线性关系。
- 对异常值敏感。
- 特征之间强相关时参数不稳定。
- 不能自然处理复杂非线性。

## 12. 如何增强线性回归

可以通过特征工程增强表达能力。

例如加入多项式特征：

```text
x, x^2, x^3
```

或交叉特征：

```text
面积 * 是否学区房
```

模型对新特征仍是线性的，但对原始变量可以表达非线性关系。

## 13. 本知识点总结

线性回归的核心是：

```text
用特征加权和预测连续值，并通过最小化平方误差学习权重。
```

它简单但非常重要，是理解损失函数、参数学习、模型解释和正则化的基础。
