# 第 4 章 决策树：超级精讲总览

## 1. 决策树为什么适合入门

决策树是机器学习中最直观的模型之一。

它像人做判断：

```text
先问一个问题；
根据答案走到下一个问题；
最后得到结论。
```

例如判断西瓜：

```text
纹理是否清晰？
  是 -> 敲声是否浊响？
       是 -> 好瓜
       否 -> 坏瓜
  否 -> 坏瓜
```

这种结构可解释、容易可视化，也方便转成业务规则。

## 3. 决策树的核心问题

构造决策树时，最关键的问题是：

```text
当前节点应该用哪个特征划分？
```

好的划分应该让子节点类别更纯。

所谓“纯”，就是一个节点中的样本尽量属于同一类别。

## 4. 决策树的优缺点

优点：

- 直观。
- 可解释。
- 能处理非线性。
- 对特征缩放不敏感。
- 能处理离散和连续特征。

缺点：

- 容易过拟合。
- 对数据扰动敏感。
- 贪心生成不保证全局最优。
- 单棵树预测能力有限。
