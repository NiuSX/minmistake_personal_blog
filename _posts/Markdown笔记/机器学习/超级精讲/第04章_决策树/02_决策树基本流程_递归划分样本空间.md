# 决策树基本流程：递归划分样本空间

## 1. 决策树长什么样

决策树由三类节点构成：

- 根节点：最开始的问题。
- 内部节点：中间判断问题。
- 叶节点：最终预测结果。

每条从根到叶的路径都对应一条规则。

## 2. 决策树如何预测

对于一个新样本，决策树从根节点开始，根据样本特征一步步往下走。

例如：

```text
纹理=清晰 -> 走左分支
敲声=浊响 -> 走左分支
叶节点=好瓜
```

最终到达哪个叶节点，就输出该叶节点的类别。

## 3. 构造决策树的递归流程

基本流程：

1. 当前节点包含一批样本。
2. 如果样本都属于同一类，生成叶节点。
3. 如果没有可用特征，生成叶节点，类别取多数类。
4. 否则选择一个最优划分特征。
5. 按该特征取值划分子节点。
6. 对每个子节点重复上述过程。

这叫递归划分。

## 4. 什么是样本空间划分

每次选择一个特征，相当于把样本空间切成几块。

例如按“色泽”划分：

- 青绿。
- 乌黑。
- 浅白。

每个分支对应一个区域。

随着树往下长，区域越来越细。

## 5. 什么时候停止划分

常见停止条件：

- 当前节点样本全属于同一类。
- 当前没有可用特征。
- 当前样本数量太少。
- 树深达到限制。
- 划分带来的收益太小。

停止太早会欠拟合；停止太晚会过拟合。

## 6. 决策树为什么容易过拟合

如果不限制深度，树可以一直划分到每个叶节点只包含极少样本。

这样训练集表现很好，但模型可能只是记住了训练样本。

例如一条过度具体的规则：

```text
色泽=青绿 且 根蒂=蜷缩 且 编号=17 -> 好瓜
```

编号对未来样本没有泛化意义。

## 7. 贪心思想

决策树通常采用贪心策略。

每一步选择当前看来最好的特征。

贪心的优点：

- 简单。
- 高效。

缺点：

- 当前最优不一定导致全局最优。

但实践中，配合剪枝和集成，决策树仍然非常有效。

## 8. 本知识点总结

决策树的核心是递归划分。

每一步都问：

```text
用哪个特征划分，能让样本类别更清楚？
```

理解这句话，就理解了决策树生成的主线。
