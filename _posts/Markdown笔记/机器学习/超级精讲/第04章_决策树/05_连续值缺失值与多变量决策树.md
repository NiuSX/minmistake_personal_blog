# 连续值、缺失值与多变量决策树

## 1. 离散特征和连续特征

离散特征取值有限，例如：

- 色泽：青绿、乌黑、浅白。
- 敲声：浊响、沉闷、清脆。

连续特征是数值，例如：

- 密度。
- 含糖率。
- 年龄。
- 收入。

决策树需要把连续值转成可划分的条件。

## 2. 连续值如何划分

常见做法是选择一个阈值 t，把样本分成两部分：

```text
特征 <= t
特征 > t
```

例如：

```text
密度 <= 0.42
密度 > 0.42
```

算法会尝试多个候选阈值，选择划分效果最好的。

## 3. 候选阈值怎么来

把连续特征取值排序。

相邻两个值的中点可以作为候选阈值。

例如取值：

```text
0.20, 0.35, 0.50
```

候选阈值：

```text
0.275, 0.425
```

然后计算每个阈值的信息增益或基尼下降。

## 4. 缺失值为什么麻烦

现实数据经常缺失。

例如某个样本没有记录密度，或者用户没有填写年龄。

简单删除缺失样本可能浪费数据。

简单填补可能引入偏差。

## 5. 缺失值处理思路

常见方法：

- 用均值、众数、中位数填补。
- 增加“是否缺失”指示特征。
- 根据其他特征预测缺失值。
- 在树分裂时按概率分配到多个分支。

缺失本身也可能有意义。

例如用户不填写收入，可能与用户隐私偏好或风险有关。

## 6. 多变量决策树

普通决策树每个节点通常只用一个特征判断：

```text
密度 > 0.5
```

多变量决策树可以在节点使用多个特征的线性组合：

```text
w1*x1 + w2*x2 > t
```

这让节点划分边界更灵活。

## 7. 单变量和多变量对比

单变量树：

- 解释简单。
- 划分轴平行。
- 可能需要很多层表达斜边界。

多变量树：

- 表达能力更强。
- 树可能更小。
- 解释性下降。

## 8. 本知识点总结

连续值通过阈值二分处理；缺失值需要结合数据机制谨慎处理；多变量决策树增强表达能力，但牺牲部分解释性。
