# 神经元模型与感知机

## 1. 人工神经元是什么

人工神经元是神经网络的基本计算单元。

它做三步：

1. 接收多个输入。
2. 对输入加权求和。
3. 经过激活函数输出。

公式：

```text
z = w1*x1 + w2*x2 + ... + wd*xd + b
y = f(z)
```

## 2. 权重和偏置

权重表示输入的重要性。

偏置表示整体阈值或偏移。

如果权重大，说明该输入对输出影响强。

如果权重为负，说明该输入增加会降低输出。

## 3. 激活函数为什么必要

如果没有激活函数，多层线性变换叠加仍然是线性变换。

也就是说：

```text
线性层 + 线性层 + 线性层 = 线性层
```

这样的网络无论多少层，都不能表达复杂非线性。

激活函数引入非线性，让神经网络能学习复杂边界。

## 4. 常见激活函数

### 阈值函数

早期感知机常用。

大于阈值输出 1，否则输出 0。

### Sigmoid

输出在 0 到 1。

早期常用，但深层网络中容易梯度消失。

### ReLU

```text
ReLU(x) = max(0, x)
```

简单高效，是深度学习常用激活函数。

## 5. 感知机

感知机是最早的神经网络模型之一。

它本质上是线性分类器。

预测过程：

```text
如果 w^T x + b > 0，判为正类；
否则判为负类。
```

## 6. 感知机如何学习

如果预测正确，不调整。

如果预测错误，就更新权重，让模型更倾向正确类别。

直觉：

- 正样本被错判为负：增加相关权重。
- 负样本被错判为正：减少相关权重。

## 7. 感知机的局限

感知机只能解决线性可分问题。

XOR 问题无法用一条直线分开，因此单层感知机解决不了。

这曾导致神经网络研究进入低谷。

## 8. 本知识点总结

神经元是加权求和加激活函数。

感知机是单层神经元分类器，能学习线性边界，但不能解决线性不可分问题。
