# 第 6 章 支持向量机：超级精讲总览

## 1. SVM 的核心思想

支持向量机 SVM 的核心不是“找一条能分开样本的线”，而是：

```text
在能分开样本的所有边界中，找间隔最大的边界。
```

它追求的是安全、稳健的分类边界。

## 2. 本章知识点

本章包括：

1. 间隔与支持向量。
2. 对偶问题。
3. 核函数。
4. 软间隔与正则化。
5. 支持向量回归。
6. 核方法。

## 3. 为什么 SVM 重要

SVM 在深度学习爆发前是非常强大的经典机器学习方法。

它适合：

- 中小规模数据。
- 高维特征。
- 分类边界需要较好泛化。
- 文本分类等稀疏高维任务。

## 4. 学习目标

学完本章，你应该能：

1. 解释什么是间隔。
2. 说明支持向量为什么关键。
3. 理解最大间隔如何提高泛化能力。
4. 理解核函数如何处理非线性。
5. 解释软间隔和 C 参数。
6. 知道 SVM 的优缺点。
