# 对偶问题与核函数：从线性到非线性

## 1. 为什么需要对偶问题

SVM 原始问题是最大化间隔。

通过数学变换，可以得到对偶问题。

对偶问题的重要价值是：

```text
模型计算只依赖样本之间的内积。
```

这一点引出了核函数。

## 2. 内积为什么重要

两个向量的内积可以衡量它们相似程度。

在线性 SVM 中，训练和预测可以写成样本内积的形式。

如果我们能替换内积，就能隐式改变特征空间。

## 3. 非线性问题

有些数据在原空间中线性不可分。

例如圆形分布：

- 正类在圆内。
- 负类在圆外。

二维平面中无法用直线分开。

但如果加入新特征：

```text
z = x1^2 + x2^2
```

在新空间中可能可以线性分开。

## 4. 映射到高维空间

一种思路是把原始特征映射到高维空间：

```text
x -> φ(x)
```

然后在高维空间中训练线性 SVM。

问题是：显式计算 φ(x) 可能很贵，甚至维度无限。

## 5. 核技巧

核函数直接计算高维空间中的内积：

```text
K(xi, xj) = φ(xi)^T φ(xj)
```

无需显式写出 φ(x)。

这就是核技巧。

## 6. 常见核函数

### 线性核

```text
K(x, z) = x^T z
```

适合线性问题和高维稀疏数据。

### 多项式核

能表达多项式特征组合。

### RBF 核

也叫高斯核，常用且强大。

能处理复杂非线性边界。

## 7. RBF 核参数直觉

RBF 核有一个控制影响范围的参数。

影响范围太小：

- 每个样本只影响很近区域。
- 边界很复杂。
- 容易过拟合。

影响范围太大：

- 边界过于平滑。
- 容易欠拟合。

## 8. 本知识点总结

对偶问题让 SVM 只依赖内积。

核函数用高维空间内积替代原始内积，使 SVM 可以处理非线性问题。

核技巧的本质是：

```text
不显式升维，却获得升维后的计算效果。
```
