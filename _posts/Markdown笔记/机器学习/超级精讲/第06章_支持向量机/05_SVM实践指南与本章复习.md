# SVM 实践指南与本章复习

## 1. SVM 优点

- 理论基础扎实。
- 最大间隔有助于泛化。
- 核函数能处理非线性。
- 高维小样本场景表现好。
- 支持向量带来稀疏解。

## 2. SVM 缺点

- 大规模数据训练慢。
- 核函数和参数选择敏感。
- 默认不直接输出概率。
- 对特征缩放敏感。
- 解释性不如线性模型和决策树直观。

## 3. 什么时候用线性核

适合：

- 样本多。
- 特征维度高。
- 文本分类。
- 数据近似线性可分。

线性核训练更快，也更容易解释。

## 4. 什么时候用 RBF 核

适合：

- 样本规模中小。
- 数据边界非线性。
- 特征维度不是特别高。

需要调 C 和 gamma。

## 5. 使用 SVM 前要做什么

1. 特征标准化。
2. 处理缺失值。
3. 检查类别不平衡。
4. 用交叉验证调参。
5. 比较线性核和 RBF 核。

## 6. 核心概念复习

请解释：

1. 间隔。
2. 最大间隔。
3. 支持向量。
4. 对偶问题。
5. 核函数。
6. 软间隔。
7. C 参数。
8. 合页损失。
9. SVR。

## 7. 判断题

### 7.1 SVM 只要求训练样本分对，不关心边界距离。

错误。SVM 追求最大间隔。

### 7.2 核函数可以隐式实现高维映射。

正确。

### 7.3 C 越大越不容易过拟合。

错误。C 大更强调训练错误，可能更容易过拟合。

### 7.4 SVM 使用前通常需要特征标准化。

正确。

## 8. 本章总结

SVM 的一句话总结：

```text
在合适的特征空间中，寻找最大间隔分类边界。
```

线性不可分时，核函数让 SVM 在隐式高维空间中寻找线性分割。

真实数据有噪声时，软间隔让模型在间隔和错误之间折中。
