# 第 7 章 贝叶斯分类器：超级精讲总览

## 1. 贝叶斯分类器的核心思想

贝叶斯分类器从概率角度做分类。

它问的问题是：

```text
看到样本 x 后，它属于每个类别的概率分别是多少？
```

然后选择后验概率最大的类别。

## 2. 本章知识点

本章包括：

1. 贝叶斯决策论。
2. 极大似然估计。
3. 朴素贝叶斯分类器。
4. 半朴素贝叶斯分类器。
5. 贝叶斯网。
6. EM 算法。

## 3. 为什么贝叶斯方法重要

贝叶斯方法提供了一种非常自然的学习视角：

```text
先有先验，再根据证据更新成后验。
```

它特别适合处理不确定性。

## 4. 学习目标

学完本章，你应该能：

1. 解释先验、似然、后验。
2. 用贝叶斯公式理解分类。
3. 理解朴素贝叶斯的条件独立假设。
4. 知道拉普拉斯平滑解决什么问题。
5. 理解贝叶斯网如何表示变量依赖。
6. 理解 EM 如何处理隐变量。
