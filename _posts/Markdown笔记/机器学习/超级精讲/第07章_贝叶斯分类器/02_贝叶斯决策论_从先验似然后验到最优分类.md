# 贝叶斯决策论：从先验、似然、后验到最优分类

## 1. 贝叶斯思维是什么

贝叶斯思维的核心是：

```text
根据新证据更新对事件的相信程度。
```

例如判断一个西瓜是不是好瓜。

在没看西瓜前，你可能知道市场上大约 60% 是好瓜。这是先验。

看到它色泽青绿、根蒂蜷缩、敲声浊响后，你会提高它是好瓜的判断。这是根据证据更新后的后验。

## 2. 贝叶斯公式

贝叶斯公式：

```text
P(c | x) = P(x | c) P(c) / P(x)
```

其中：

- P(c)：先验概率。
- P(x | c)：似然。
- P(x)：证据。
- P(c | x)：后验概率。

## 3. 分类时为什么比较后验

分类任务中，我们看到样本 x，需要判断类别 c。

如果不同错误代价相同，最合理的选择是：

```text
选择 P(c | x) 最大的类别
```

这就是贝叶斯最优分类器的思想。

## 4. P(x) 为什么可以忽略

对同一个样本 x，不同类别下 P(x) 相同。

所以比较后验时只需比较：

```text
P(x | c) P(c)
```

这让计算更简单。

## 5. 损失不同时怎么办

如果不同错误代价不同，就不能只选后验最大。

例如医疗中，把病人误判健康的代价远大于把健康人误判疑似。

这时应该最小化期望损失，而不是单纯最大化后验。

## 6. 本知识点总结

贝叶斯决策论把分类问题转成概率决策问题。

核心公式：

```text
后验 ∝ 似然 × 先验
```

核心原则：

```text
选择后验概率最大或期望损失最小的类别。
```
