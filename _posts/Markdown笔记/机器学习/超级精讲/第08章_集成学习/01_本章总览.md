# 第 8 章 集成学习：超级精讲总览

## 1. 集成学习的核心思想

集成学习的核心是：

```text
把多个学习器组合起来，获得比单个学习器更强的模型。
```

这类似多人投票或专家会诊。

单个模型可能会犯错，但如果多个模型错误不完全相同，组合后可能更稳定、更准确。

## 2. 本章知识点

本章包括：

1. 个体学习器与集成。
2. Boosting。
3. Bagging。
4. 随机森林。
5. 结合策略。
6. 多样性。

## 3. 集成学习有效的两个条件

个体学习器要：

1. 有一定准确性。
2. 有差异性。

如果每个模型都很差，组合也难好。

如果所有模型完全一样，组合没有意义。

## 4. 学习目标

学完本章，你应该能：

1. 理解为什么集成能提升性能。
2. 解释 Bagging 如何降低方差。
3. 解释 Boosting 如何逐步增强弱学习器。
4. 理解随机森林为什么比单棵树稳。
5. 掌握平均、投票、学习式结合策略。
6. 说明多样性为什么重要。
