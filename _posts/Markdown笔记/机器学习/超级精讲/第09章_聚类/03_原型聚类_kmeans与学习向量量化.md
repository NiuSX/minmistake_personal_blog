# 原型聚类：k-means 与学习向量量化

## 1. 什么是原型聚类

原型聚类用一个代表点描述一个簇。

最典型的是 k-means。

每个簇用中心点表示。

## 2. k-means 基本流程

1. 指定簇数 k。
2. 初始化 k 个中心。
3. 每个样本分配给最近中心。
4. 重新计算每个簇中心。
5. 重复直到中心稳定。

## 3. k-means 的目标

k-means 希望最小化簇内平方误差。

直觉：

```text
每个样本离自己簇中心越近越好。
```

## 4. k 的选择

k 需要提前指定。

常用方法：

- 肘部法。
- 轮廓系数。
- 业务解释。

## 5. k-means 的局限

- 对初始中心敏感。
- 需要指定 k。
- 偏好球形簇。
- 对异常值敏感。
- 不适合密度差异大或非凸形状簇。

## 6. 学习向量量化

学习向量量化 LVQ 使用带标签数据学习原型向量。

它与 k-means 相似，但利用类别标签调整原型。

## 7. 本知识点总结

k-means 简单高效，是聚类入门核心算法。

但它有强假设：簇像围绕中心的球形团块。
