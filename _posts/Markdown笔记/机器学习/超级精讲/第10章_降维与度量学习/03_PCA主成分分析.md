# PCA 主成分分析

## 1. PCA 要做什么

PCA 是最经典的线性降维方法。

它寻找一组方向，使数据投影后保留最大方差。

直觉：

```text
数据变化最大的方向，通常包含最多信息。
```

## 2. 用二维例子理解

假设二维点大致分布在一条斜线附近。

虽然数据是二维，但主要变化沿斜线方向。

PCA 会找到这条斜线，把数据投影上去，变成一维表示。

## 3. PCA 步骤

1. 数据中心化。
2. 计算协方差矩阵。
3. 求特征值和特征向量。
4. 选择最大特征值对应的前 k 个方向。
5. 投影到低维空间。

## 4. 主成分含义

第一主成分解释最大方差。

第二主成分在与第一主成分正交的方向上解释剩余最大方差。

依此类推。

## 5. PCA 优点

- 简单。
- 无监督。
- 可降噪。
- 可压缩。
- 可可视化。

## 6. PCA 局限

- 只能捕捉线性结构。
- 不看标签。
- 最大方差方向不一定最适合分类。
- 对特征尺度敏感，通常要标准化。

## 7. 本知识点总结

PCA 用最大方差原则做线性降维。

它适合压缩冗余数值特征，但不一定保留最有判别力的信息。
