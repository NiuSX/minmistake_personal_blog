# L1 正则化与稀疏表示

## 1. 什么是稀疏

稀疏表示指大部分元素为 0，只有少数非零。

例如文本向量中，词表有十万个词，但一篇文章只包含几百个词。

## 2. 稀疏的价值

稀疏带来：

- 可解释性。
- 存储高效。
- 计算高效。
- 降低过拟合。
- 突出关键因素。

## 3. L1 正则化

L1 正则化在损失中加入参数绝对值和：

```text
原损失 + λ * Σ|wi|
```

它会鼓励部分参数变成 0。

参数为 0 的特征相当于被删除。

## 4. L1 和 L2 的区别

L1：

- 倾向产生稀疏解。
- 可做特征选择。

L2：

- 倾向让参数变小。
- 通常不直接变 0。
- 提高稳定性。

## 5. 为什么 L1 能稀疏

直觉：

```text
每使用一个特征都要付费。
```

模型会只保留最值得付费的特征。

## 6. 本知识点总结

L1 正则化是特征选择和稀疏学习的重要工具。

它通过惩罚参数绝对值，让许多特征权重变成 0。
