# 贝叶斯网

贝叶斯网用有向无环图表示变量之间的条件依赖关系。每个节点是一个随机变量，边表示直接依赖。它的价值在于把一个复杂的联合概率分布拆解成多个局部条件概率，从而让建模、推断和解释变得可管理。

## 一、为什么需要贝叶斯网

如果有很多随机变量，直接写出完整联合分布非常困难。假设有 20 个二值变量，联合状态就有 2 的 20 次方种组合。贝叶斯网利用条件独立性，把联合分布分解为每个变量在其父节点条件下的概率乘积。

例如疾病诊断中，疾病会影响症状，检查结果也与疾病有关。我们不必直接枚举所有疾病、症状、检查结果的联合组合，而可以把“疾病 -> 症状”“疾病 -> 检查结果”的依赖结构画出来，再分别建模局部条件概率。

## 二、条件独立性的意义

贝叶斯网的核心不是画图好看，而是图表达了条件独立假设。如果某个节点在给定父节点后，与非后代节点条件独立，那么联合分布就可以分解。

这对学习很重要。没有条件独立假设，参数数量会爆炸；有了合理假设，少量数据也能估计局部概率表。当然，如果独立性假设错得太离谱，模型也会偏差很大。

## 三、常见结构

链式结构 A -> B -> C 表示 A 影响 B，B 影响 C。给定 B 后，A 与 C 条件独立。

分叉结构 A -> B, A -> C 表示共同原因。给定 A 后，B 与 C 条件独立。

汇合结构 A -> C <- B 表示共同结果。在不给定 C 时，A 与 B 可能独立；一旦观测 C，A 与 B 可能变得相关，这叫解释消除现象。

## 四、推断任务

贝叶斯网常见推断包括：给定症状推断疾病概率，给定部分观测补全未知变量，计算某个事件的边缘概率。精确推断在复杂图上可能非常困难，因此需要变量消去、信念传播、采样等方法。

## 五、学习任务

贝叶斯网学习分为参数学习和结构学习。结构已知时，参数学习相对直接，可以用最大似然或贝叶斯估计。结构未知时，需要从数据中搜索图结构，难度高很多，因为可能的图数量巨大，还要避免环。

## 六、学习建议

先掌握“图结构如何对应联合分布分解”，再学习推断算法。概率图模型最怕只记公式不理解依赖关系。看到一个图时，要能说出每个变量依赖谁、哪些变量在什么条件下独立、观测某个节点会改变哪些概率。
