# 数学基础速查

## 1. 线性代数

### 向量

向量是一组有序数字，可表示一个样本的特征。

例如：

```text
x = [密度, 含糖率]
```

### 矩阵

矩阵是二维数字表。机器学习中常用矩阵表示数据集。

若有 n 个样本、d 个特征，数据矩阵形状通常是：

```text
n x d
```

### 内积

两个向量的内积：

```text
w^T x = w1*x1 + ... + wd*xd
```

线性模型大量使用内积。

### 范数

范数衡量向量大小。

常见：

- L1 范数：绝对值之和。
- L2 范数：平方和开根号。

正则化常用范数限制参数大小。

## 2. 概率基础

### 条件概率

```text
P(A | B)
```

表示 B 发生条件下 A 发生的概率。

### 贝叶斯公式

```text
P(A | B) = P(B | A)P(A) / P(B)
```

用于从先验和似然计算后验。

### 期望

随机变量的平均取值。

### 方差

随机变量偏离期望的程度。

## 3. 优化基础

### 损失函数

损失函数衡量预测和真实值之间的差距。

训练模型通常就是最小化损失。

### 梯度

梯度表示函数上升最快方向。

梯度下降沿负梯度方向更新参数。

### 学习率

学习率控制每次参数更新步子多大。

过大可能震荡，过小收敛很慢。

## 4. 常见函数

### Sigmoid

```text
sigmoid(x) = 1 / (1 + e^-x)
```

把实数映射到 0 到 1。

### Softmax

把多个分数转化为概率分布。

常用于多分类输出。

### ReLU

```text
ReLU(x) = max(0, x)
```

深度学习常用激活函数。

## 5. 新手学习建议

不必先把所有数学完全学完再学机器学习。更好的方式是：

1. 先理解算法要解决什么问题。
2. 遇到公式时回到数学基础查含义。
3. 用小例子手算一次。
4. 再看代码实现。
