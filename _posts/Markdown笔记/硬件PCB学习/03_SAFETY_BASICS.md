# 03 安全基础

## 学习目标

学完本章，你应该能：

- 识别硬件学习中的主要危险。
- 知道新手应该避开哪些电路。
- 掌握第一次上电前的安全检查流程。
- 理解限流电源、保险丝、防反接和 ESD 的意义。

硬件安全不是形式。很多电路问题会导致器件烧毁、板子冒烟、电池起火，甚至人身触电。新手学习必须先建立安全边界。

## 1. 新手安全边界

建议新手从低压直流开始：

- 3.3V
- 5V
- 9V
- 12V

电流建议从小到大：

- 50mA
- 100mA
- 300mA
- 500mA
- 1A

刚开始不要直接做：

- 市电 220V / 110V
- AC-DC 开关电源原边
- 高压升压模块
- 大功率逆变器
- 大容量锂电池充放电
- 电动车控制器
- 大功率电机驱动
- 医疗或汽车安全相关硬件

## 2. 电压危险

低压不一定完全安全，但高压明显更危险。

| 电压范围 | 风险 | 新手建议 |
| :--- | :--- | :--- |
| 3.3V / 5V | 较低，主要是短路烧器件 | 适合入门 |
| 12V | 较低，但大电流会发热 | 可学习，需限流 |
| 24V | 工业常见，短路能量更高 | 有基础后再做 |
| 48V | 风险增加 | 谨慎 |
| 110V / 220V AC | 触电风险高 | 新手不要直接做 |
| 数百伏以上 | 致命风险 | 必须专业训练 |

注意：

电压危险看的是电压、电流路径、接触时间和环境。潮湿环境、金属工具、裸露焊点都会增加风险。

## 3. 电流和功率危险

很多低压系统也很危险，因为电流很大。

例如：

- 5V 10A 电源短路会产生大量热。
- 锂电池短路可能瞬间输出几十安。
- 电机启动电流可能远高于额定电流。

功率公式：

```text
P = U x I
```

12V 5A 就是 60W，已经不是“小电路”。

## 4. 短路风险

短路是最常见事故。

常见原因：

- 焊锡连锡
- PCB 走线错误
- 元件方向错
- 电容接反
- 芯片引脚焊桥
- 金属工具掉到板上
- 杜邦线插错

预防：

- 上电前测电源对地电阻。
- 使用限流电源。
- 第一次上电只焊电源部分。
- 工作台保持整洁。
- 不在通电时随意移动裸线。

## 5. 限流电源

限流电源是新手强烈建议购买的工具。

作用：

- 设置最大输出电流。
- 电路短路时限制电流。
- 避免烧毁更多元件。
- 通过电流读数判断电路状态。

第一次上电建议：

```text
目标电压：3.3V 或 5V
限流：50mA - 100mA 起步
```

如果电路正常，再逐步提高限流。

### 限流电源上电流程

1. 电源先不要接板。
2. 设置输出电压。
3. 设置限流值。
4. 关闭输出。
5. 接入电路板。
6. 打开输出。
7. 观察电流是否异常。
8. 测关键电压。

## 6. 电池安全

锂电池危险主要来自：

- 过充
- 过放
- 短路
- 过流
- 高温
- 刺穿
- 使用劣质电芯

新手如果用锂电池：

- 使用成品保护板。
- 使用正规充电模块。
- 不直接给裸电芯充电。
- 不短接电池正负极。
- 不在无人看管时长时间充电。
- 电池附近不要放易燃物。

## 7. ESD 静电防护

ESD 是 Electrostatic Discharge，静电放电。

静电可能让芯片：

- 立即损坏
- 参数变差
- 隐性损伤，过一段时间才出问题

容易受 ESD 影响：

- MCU
- 传感器
- MOSFET 栅极
- USB 芯片
- 射频芯片
- 运放输入

防护方法：

- 戴防静电手环。
- 使用防静电垫。
- 芯片放防静电袋。
- 接口加 TVS。
- 不直接摸芯片引脚。

## 8. 焊接安全

烙铁温度通常 300°C 左右。

注意：

- 烙铁不用时放烙铁架。
- 不要用手碰焊点确认温度。
- 焊接时保持通风。
- 助焊剂烟雾不要直接吸入。
- 剪下的引脚要清理，避免扎手或短路。
- 焊锡和助焊剂用完盖好。

## 9. 实验环境安全

工作台建议：

- 绝缘桌面
- 良好照明
- 通风
- 工具固定位置
- 电源线整理
- 易燃物远离烙铁和电池
- 有小灭火器更好

不要：

- 在床上、布料上焊接。
- 带电时用金属镊子乱碰。
- 通电裸板和金属杂物混放。
- 把电池、螺丝、杜邦线混在一起。

## 10. 第一次上电前检查清单

原理图：

- 电源输入范围是否正确。
- 电源极性是否正确。
- 芯片供电是否符合推荐工作条件。
- 保护器件方向是否正确。

PCB：

- 电源和地没有短路。
- 板框和连接器方向正确。
- 有极性器件方向正确。
- 丝印清楚。

焊接：

- 无明显连锡。
- 无虚焊。
- 芯片方向正确。
- 电解电容方向正确。
- 二极管方向正确。

上电：

- 电源限流已设置。
- 先测空载电源。
- 接线极性确认。
- 手不要碰裸露高温或带电区域。

## 实操练习

1. 用万用表通断档检查一块未上电电路板的 5V 和 GND 是否短路。
2. 设置限流电源为 5V / 100mA，给一个 LED 限流电路供电。
3. 故意把限流设置成 20mA，观察 LED 亮度和电源限流状态。
4. 练习识别电解电容、二极管、LED 的极性标记。

## 常见误区

- 误区：低压一定安全。
  纠正：低压大电流也会发热、烧线、烧板。

- 误区：短路一下没事。
  纠正：有些芯片毫秒级就会损坏。

- 误区：电源适配器可以直接替代实验电源。
  纠正：普通适配器没有可调限流，不适合第一次上电。

- 误区：ESD 只有工厂才需要管。
  纠正：实验室和家里也可能静电损坏芯片。

## 本章总结

硬件学习必须建立安全习惯。新手只做低压直流，第一次上电必须限流，焊接和电池都要谨慎。安全流程会让你少烧板、少损失器件，也能保护自己。
