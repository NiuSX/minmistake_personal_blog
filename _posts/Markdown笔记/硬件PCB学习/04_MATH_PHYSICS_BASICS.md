# 04 数学与物理基础

## 学习目标

学完本章，你应该能：

- 熟练进行电子单位换算。
- 使用欧姆定律和功率公式做基础计算。
- 理解频率、周期、时间常数和分贝的基本意义。
- 知道哪些数学知识是入门必须，哪些可以后续再学。

硬件设计不要求一开始掌握很深的数学，但基本计算必须扎实。很多烧板问题，本质上是没有算电流、功耗、耐压和余量。

## 1. 工程单位

电子工程中常用前缀：

| 前缀 | 符号 | 倍率 | 例子 |
| :--- | :--- | :--- | :--- |
| giga | G | 10^9 | 2.4GHz |
| mega | M | 10^6 | 16MHz、1MΩ |
| kilo | k | 10^3 | 10kΩ |
| milli | m | 10^-3 | 20mA |
| micro | u / μ | 10^-6 | 100uF |
| nano | n | 10^-9 | 100nF |
| pico | p | 10^-12 | 22pF |

注意大小写：

- `MΩ` 是兆欧。
- `mΩ` 是毫欧。
- `uF` 是微法。
- `nF` 是纳法。

## 2. 常见单位换算

电压：

```text
1V = 1000mV
1mV = 0.001V
```

电流：

```text
1A = 1000mA
1mA = 1000uA
```

电阻：

```text
1MΩ = 1000kΩ
1kΩ = 1000Ω
```

电容：

```text
1F = 1000000uF
1uF = 1000nF
1nF = 1000pF
100nF = 0.1uF
```

频率：

```text
1GHz = 1000MHz
1MHz = 1000kHz
1kHz = 1000Hz
```

## 3. 欧姆定律

核心公式：

```text
U = I x R
I = U / R
R = U / I
```

其中：

- U 是电压，单位 V。
- I 是电流，单位 A。
- R 是电阻，单位 Ω。

例子 1：5V 电源接 1kΩ 电阻，电流是多少？

```text
I = 5V / 1000Ω = 0.005A = 5mA
```

例子 2：3.3V 逻辑信号通过 330Ω 电阻点亮 LED，LED 压降 2V，电流约多少？

```text
I = (3.3V - 2V) / 330Ω = 0.00394A = 3.94mA
```

## 4. 功率计算

功率公式：

```text
P = U x I
P = I^2 x R
P = U^2 / R
```

例子：一个 10Ω 电阻流过 0.2A，功率：

```text
P = I^2 x R = 0.2 x 0.2 x 10 = 0.4W
```

这时使用 1/4W 电阻不够，至少选 1/2W，并最好留更多余量。

## 5. 余量意识

硬件设计不能只按刚好够用。

常见余量：

| 参数 | 建议 |
| :--- | :--- |
| 电容耐压 | 实际电压的 1.5-2 倍或按规范 |
| 电阻功率 | 实际功耗至少留 2 倍余量 |
| 电源电流 | 最大负载基础上留 20%-50% |
| MOSFET 耐压 | 高于最大电压并考虑尖峰 |
| TVS 选型 | 根据工作电压和保护电压匹配 |

余量不是越大越好，过大可能导致成本、体积、性能问题，但新手不能按极限设计。

## 6. 频率和周期

频率表示每秒重复次数，单位 Hz。

周期表示一次重复所需时间。

公式：

```text
T = 1 / f
f = 1 / T
```

例子：

1kHz 方波周期：

```text
T = 1 / 1000 = 0.001s = 1ms
```

16MHz 时钟周期：

```text
T = 1 / 16000000 = 62.5ns
```

## 7. RC 时间常数

电阻和电容组成 RC 电路时，有一个时间常数：

```text
τ = R x C
```

例如：

```text
R = 10kΩ
C = 100nF
τ = 10000 x 0.0000001 = 0.001s = 1ms
```

用途：

- 按键消抖
- 复位延时
- 简单滤波
- 信号平滑

## 8. RC 低通截止频率

简单 RC 低通滤波器：

```text
fc = 1 / (2πRC)
```

例子：

```text
R = 1kΩ
C = 100nF
fc ≈ 1 / (6.28 x 1000 x 0.0000001)
fc ≈ 1592Hz
```

频率高于 fc 的信号会逐渐被衰减。

## 9. 分贝 dB

dB 用来表达比例。

电压增益：

```text
dB = 20 x log10(Vout / Vin)
```

功率增益：

```text
dB = 10 x log10(Pout / Pin)
```

常见直觉：

- +6dB 约等于电压变 2 倍。
- -6dB 约等于电压变 1/2。
- +20dB 等于电压 10 倍。
- -20dB 等于电压 1/10。

入门阶段只需知道 dB 是比例单位，常用于增益、衰减、滤波和信号强度。

## 10. 需要逐步学习的数学

入门必须：

- 单位换算
- 欧姆定律
- 功率公式
- 比例和分压
- 基础频率周期

进阶再学：

- 复数和阻抗
- 微积分
- 拉普拉斯变换
- 傅里叶变换
- 控制理论
- 传输线理论

不要因为进阶数学复杂就不开始。很多入门项目只需要基础计算就能完成。

## 实操练习

1. 计算 5V 电源、LED 压降 2V、目标电流 5mA 时的限流电阻。
2. 计算 12V 到 5V LDO，输出 200mA 时的功耗。
3. 计算 10kΩ 和 100nF 的 RC 时间常数。
4. 把 0.1uF、100nF、100000pF 互相换算。
5. 计算 1MHz 信号的周期。

## 检查清单

- 我能否熟练换算 mA、uA、kΩ、uF、nF？
- 我能否用欧姆定律计算限流电阻？
- 我能否计算电阻或 LDO 的功耗？
- 我能否理解频率和周期的关系？
- 我是否知道设计要留余量？

## 常见误区

- 误区：电阻只看阻值。
  纠正：还要看功率、精度、封装和温漂。

- 误区：电容只看容值。
  纠正：还要看耐压、材质、ESR 和直流偏压。

- 误区：理论算刚好够就行。
  纠正：工程设计要留余量。

## 本章总结

数学和物理基础不是为了考试，而是为了避免设计错误。能算电流、功耗、时间常数和单位换算，就能解决大量入门硬件问题。
