# 17 模拟电路设计基础

## 学习目标

学完本章，你应该能：

- 理解模拟信号和数字信号的区别。
- 掌握运放、ADC、DAC、滤波、参考电压的基础概念。
- 知道模拟电路为什么怕噪声、布局为什么重要。
- 能设计简单传感器采集前端。

模拟电路处理连续变化的电压或电流。传感器、音频、电流采样、电池电压检测都属于模拟设计范畴。

## 1. 模拟信号是什么

模拟信号是连续变化的信号。

例子：

- 温度传感器输出电压。
- 麦克风声音信号。
- 电池电压。
- 电流采样电阻上的压降。
- 光敏传感器输出。

与数字信号不同，模拟信号的幅度细节很重要，噪声会直接影响测量结果。

## 2. 模拟电路的主要问题

常见问题：

- 噪声
- 偏置
- 失调
- 漂移
- 带宽不足
- 输入范围超限
- 参考电压不稳
- 地噪声
- ADC 采样误差

模拟电路不能只看连通，还要看精度和干扰。

## 3. 运算放大器

运放常用于：

- 放大微弱信号。
- 缓冲高阻信号。
- 滤波。
- 电压跟随。
- 电流采样放大。

### 常见运放电路

电压跟随器：

- 增益 1。
- 输入阻抗高，输出阻抗低。
- 用于缓冲。

反相放大：

```text
Gain = -Rf / Rin
```

同相放大：

```text
Gain = 1 + Rf / Rg
```

## 4. 运放选型参数

| 参数 | 含义 |
| :--- | :--- |
| 供电范围 | 支持单电源或双电源 |
| 输入共模范围 | 输入电压允许范围 |
| 输出摆幅 | 输出能否接近电源轨 |
| 增益带宽积 | 决定高频放大能力 |
| 压摆率 | 输出变化速度 |
| 输入失调电压 | 输入误差 |
| 输入偏置电流 | 输入端电流 |
| 噪声 | 对微弱信号重要 |
| 轨到轨 | 输入或输出接近电源轨能力 |

新手常错：

- 用 3.3V 单电源，却让输入或输出接近 0V 以下。
- 忽略输入共模范围。
- 忽略输出不能真正到达电源轨。

## 5. ADC

ADC 把模拟电压转换为数字值。

关键参数：

- 分辨率：如 10 位、12 位、16 位。
- 采样率。
- 输入范围。
- 参考电压。
- 输入阻抗。
- INL / DNL。
- 噪声。

### 分辨率

12 位 ADC 有：

```text
2^12 = 4096
```

如果参考电压 3.3V，每个码值约：

```text
3.3V / 4096 = 0.805mV
```

但实际精度还受噪声、参考源、布局影响。

## 6. 参考电压

ADC 测量结果依赖参考电压。

如果参考电压不稳定，测量也不稳定。

设计要点：

- 使用稳定参考源。
- 参考电压加去耦。
- 走线短。
- 远离噪声。
- 不让大电流经过参考地。

## 7. DAC

DAC 把数字值转换为模拟电压或电流。

用途：

- 输出控制电压。
- 音频。
- 波形生成。
- 校准。

DAC 输出可能需要：

- 缓冲运放。
- 低通滤波。
- 输出保护。

## 8. RC 滤波

简单低通：

```text
fc = 1 / (2πRC)
```

用途：

- ADC 输入滤波。
- 按键消抖。
- PWM 平滑。
- 传感器噪声抑制。

注意：

滤波会引入响应变慢。不能只追求更小噪声。

## 9. 电流采样

电流采样常用采样电阻。

```text
Vsense = I x Rsense
```

注意：

- 采样电阻功率。
- 阻值不能太大导致压降过高。
- 小信号需要放大。
- 使用 Kelvin 连接提高精度。

## 10. 模拟布局原则

- 模拟输入短。
- 远离 DC-DC、电机、继电器。
- 参考电压靠近 ADC。
- 模拟地路径干净。
- 运放反馈环路短。
- 高阻节点面积小。
- 必要时加屏蔽地。

## 实操练习

1. 用两个电阻分压，把电池电压缩放到 ADC 范围内。
2. 计算 12 位 ADC 在 3.3V 参考下的每码电压。
3. 设计一个 RC 低通滤波器，截止频率约 1kHz。
4. 查一个运放数据手册，找输入共模范围和输出摆幅。
5. 在 PCB 中给 ADC 输入走线远离电源开关节点。

## 检查清单

- 模拟输入是否在 ADC 范围内？
- 参考电压是否稳定？
- 运放输入和输出是否在允许范围内？
- ADC 输入是否有必要滤波？
- 模拟走线是否远离噪声源？
- 采样电阻功耗是否足够？

## 常见误区

- 误区：ADC 位数越高，实际精度越高。
  纠正：噪声、参考源和布局都会限制有效精度。

- 误区：运放是理想器件。
  纠正：真实运放有输入范围、输出范围、带宽和失调。

- 误区：模拟地必须随便切割。
  纠正：重点是控制回流路径和噪声，不是机械分割。

## 本章总结

模拟电路关注精度、噪声、参考和布局。入门阶段先掌握分压、RC 滤波、运放基础和 ADC 采样，再逐步学习更复杂的模拟前端。
