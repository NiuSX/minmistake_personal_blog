# 21 传感器与执行器

## 学习目标

学完本章，你应该能：

- 理解传感器和执行器在硬件系统中的角色。
- 知道传感器选型要看哪些参数。
- 知道不同传感器对布局和安装位置的要求。
- 理解执行器通常需要驱动和保护。

传感器负责感知世界，执行器负责改变世界。它们是嵌入式硬件从“能运行”走向“能做事”的关键。

## 1. 传感器是什么

传感器把物理量转换成电信号或数字数据。

常见物理量：

- 温度
- 湿度
- 压力
- 光照
- 距离
- 加速度
- 角速度
- 磁场
- 声音
- 电流
- 电压

输出形式：

- 模拟电压
- 模拟电流
- 电阻变化
- I2C
- SPI
- UART
- PWM
- 脉冲频率

## 2. 传感器选型参数

| 参数 | 含义 | 例子 |
| :--- | :--- | :--- |
| 测量范围 | 能测多大范围 | -40°C 到 125°C |
| 精度 | 测量值和真实值误差 | ±0.5°C |
| 分辨率 | 最小可分辨变化 | 0.01°C |
| 响应时间 | 对变化反应速度 | 1s |
| 接口 | 和 MCU 通信方式 | I2C、SPI、ADC |
| 供电 | 工作电压和电流 | 3.3V, 1mA |
| 校准 | 是否需要校准 | 出厂校准或用户校准 |
| 封装 | 安装方式 | LGA、SOT、模块 |
| 环境 | 温度、湿度、防水 | 工业环境 |

不要只看价格和接口。

## 3. 模拟传感器

模拟传感器输出连续电压或电流。

例子：

- 光敏电阻
- NTC 温度传感器
- 模拟麦克风
- 电流采样
- 模拟压力传感器

设计要点：

- 输入范围匹配 ADC。
- 必要时分压或放大。
- 加 RC 滤波。
- 参考电压稳定。
- 走线远离噪声。

## 4. 数字传感器

数字传感器内部完成采样和转换，通过数字接口输出。

常见接口：

- I2C
- SPI
- UART
- 1-Wire

优点：

- 使用简单。
- 抗模拟噪声能力较好。
- 通常有校准数据。

注意：

- 电平兼容。
- 总线地址。
- 上拉电阻。
- 布局位置。
- 供电去耦。

## 5. 温度传感器布局

温度传感器要测环境温度时：

- 远离 LDO、DC-DC、MCU、电机驱动等发热器件。
- 避免被大面积铜皮连接到热源。
- 放在空气流通位置。
- 外壳要开孔或导热设计。

如果要测板上芯片温度，则应靠近目标热源。

## 6. 光照传感器布局

要求：

- 传感器上方有开窗。
- 避免被外壳遮挡。
- 避免 LED 指示灯直射。
- 考虑透明窗口材料影响。

## 7. 气压、湿度和气体传感器

要求：

- 接触空气。
- 避免胶水、清洗剂、助焊剂污染。
- 避免放在密闭区域。
- 避免靠近热源。

湿度和气体传感器对污染非常敏感。

## 8. IMU 加速度计和陀螺仪

关注：

- 安装方向。
- 坐标轴定义。
- 靠近设备机械中心。
- 远离振动源。
- PCB 固定牢固。

布局中要在丝印或文档中标明传感器坐标方向。

## 9. 电流传感器

方式：

- 采样电阻。
- 霍尔传感器。
- 电流互感器。

采样电阻注意：

- 功率。
- 阻值精度。
- 温漂。
- Kelvin 连接。
- 放大器输入布局。

## 10. 执行器是什么

执行器把电信号转换成动作、光、声音、热等输出。

常见执行器：

- LED
- 蜂鸣器
- 继电器
- 电机
- 舵机
- 加热片
- 电磁阀
- 显示屏

执行器通常不能由 MCU GPIO 直接驱动。

## 11. 执行器驱动

驱动方式：

- 三极管
- MOSFET
- 驱动芯片
- H 桥
- 恒流源
- 继电器

设计要点：

- 负载电流。
- 启动电流。
- 感性负载反向电压。
- 散热。
- 电源容量。
- EMI。

## 12. 执行器保护

感性负载：

- 继电器
- 电机
- 电磁阀

需要：

- 续流二极管。
- TVS。
- RC 吸收。
- 驱动芯片保护。

大电流负载：

- 加宽走线。
- 留散热。
- 加保险丝。
- 做电源隔离或滤波。

## 实操练习

1. 选一个 I2C 温湿度传感器，查供电、接口、地址和布局要求。
2. 设计一个 NTC 温度采样电路，输出到 MCU ADC。
3. 设计一个 MOSFET 控制蜂鸣器或 LED 灯带电路。
4. 在 PCB 上为温度传感器选择远离热源的位置。
5. 给传感器和执行器分别留测试点。

## 检查清单

- 传感器测量范围是否满足需求？
- 精度和分辨率是否足够？
- 接口电平是否兼容？
- 模拟传感器是否匹配 ADC 输入范围？
- 传感器位置是否符合物理测量要求？
- 执行器是否需要驱动？
- 感性负载是否有保护？
- 大电流路径是否足够宽？

## 常见误区

- 误区：传感器只要接线对就能测准。
  纠正：布局、安装、热源、污染都会影响测量。

- 误区：MCU GPIO 可以直接带负载。
  纠正：多数执行器需要驱动器件。

- 误区：数字传感器不需要关心噪声。
  纠正：供电和布局仍会影响稳定性。

## 本章总结

传感器要关注测量对象和安装环境，执行器要关注驱动能力和保护。真实硬件设计不只是电气连接，还要考虑物理位置、能量、热和干扰。
