# 25 EMI / EMC 基础

## 学习目标

学完本章，你应该能：

- 理解 EMI、EMC、ESD、传导干扰、辐射干扰的区别。
- 知道 PCB 中常见干扰源。
- 掌握基础 EMI 抑制方法。
- 在设计早期减少明显的电磁兼容风险。

EMI / EMC 是硬件从“能工作”到“可靠产品”的重要门槛。入门阶段不需要马上掌握认证细节，但要建立正确设计习惯。

## 1. 基本概念

| 术语 | 含义 |
| :--- | :--- |
| EMI | Electromagnetic Interference，电磁干扰 |
| EMC | Electromagnetic Compatibility，电磁兼容 |
| ESD | Electrostatic Discharge，静电放电 |
| 辐射干扰 | 通过空间传播 |
| 传导干扰 | 通过线缆、电源传播 |
| 共模噪声 | 多根线相对地同方向噪声 |
| 差模噪声 | 两根线之间的噪声 |

EMC 包含两方面：

- 不严重干扰别人。
- 自己能承受外部干扰。

## 2. 常见干扰源

PCB 中常见：

- DC-DC 开关节点。
- 时钟。
- 高速数字边沿。
- 电机。
- 继电器。
- 长线缆。
- 大电流回路。
- 不连续地平面。
- 天线和无线模块。

## 3. 干扰三要素

干扰需要：

- 干扰源。
- 耦合路径。
- 受害者。

解决思路：

- 降低干扰源。
- 阻断耦合路径。
- 提高受害者抗扰能力。

## 4. 辐射干扰

辐射干扰常来自：

- 大回路面积。
- 高频电流。
- 长线缆像天线。
- 地平面不连续。

降低方法：

- 减小回路面积。
- 保持连续地。
- 缩短高速线。
- 降低边沿速度。
- 接口加滤波。
- 线缆屏蔽。

## 5. 传导干扰

传导干扰通过电源线、信号线传播。

常见：

- DC-DC 噪声回到输入电源。
- 电机噪声通过电源影响 MCU。
- ESD 从接口进入系统。

处理：

- 输入滤波。
- TVS。
- 共模电感。
- 磁珠。
- 保险丝。
- 良好接地。

## 6. ESD 保护

ESD 重点在接口：

- USB
- 按键
- 排针
- 传感器外接线
- 通信线
- 用户可触摸金属

TVS 放置原则：

- 靠近接口。
- 到地路径短。
- 低电容用于高速信号。
- 工作电压匹配。

## 7. DC-DC EMI

DC-DC 是常见 EMI 来源。

关键节点：

- SW 节点。
- 输入高 di/dt 回路。
- 输出电感电流回路。

布局：

- 输入电容靠近。
- 高频回路最小。
- SW 铜皮不要过大。
- 反馈线远离 SW。
- 地回路短。
- 必要时加屏蔽电感。

## 8. 电机和继电器 EMI

处理方法：

- 续流二极管。
- TVS。
- RC 吸收。
- 电机端并电容。
- 电源加大电容。
- 负载线远离信号线。
- 驱动回路短。

## 9. 接口 EMI

外部线缆容易成为天线。

接口设计：

- ESD 靠近接口。
- 共模电感视接口需要。
- 串联电阻或磁珠。
- 屏蔽层处理。
- GND 连接低阻抗。

## 10. PCB 基础习惯

降低 EMI 的基本习惯：

- 保持 GND 平面连续。
- 缩小高频和大电流回路。
- 高速线短。
- 时钟远离接口和模拟输入。
- 大电流远离敏感区域。
- 保护器件靠近接口。
- 不让线缆连接噪声节点。

## 实操练习

1. 在一个 DC-DC PCB 中标出 SW 节点和高频回路。
2. 在接口附近放置 TVS，并让到地路径最短。
3. 画一个电机驱动回路，标出噪声路径。
4. 检查一块双层板 GND 是否被切碎。
5. 给长线输入信号加 RC 滤波和 ESD。

## 检查清单

- 高频回路是否小？
- GND 平面是否连续？
- 接口是否有 ESD？
- TVS 是否靠近接口？
- 电机和继电器是否有吸收保护？
- DC-DC SW 节点是否远离敏感线？
- 长线是否有滤波或保护？

## 常见误区

- 误区：EMI 是认证阶段才考虑。
  纠正：布局阶段决定大部分 EMI 表现。

- 误区：加磁珠就能解决所有干扰。
  纠正：根本是回路、地、布局和源头控制。

- 误区：ESD 器件放哪里都一样。
  纠正：必须靠近接口且地路径短。

## 本章总结

EMI / EMC 的入门重点是控制回路面积、连续地平面、接口保护和噪声源布局。越早考虑，越容易解决；等板子做完再补救，成本很高。
