# 机器学习学习笔记：西瓜书 + 南瓜书整合版

本目录是一套围绕周志华《机器学习》（常称“西瓜书”）章节体系，并结合《机器学习公式详解》（常称“南瓜书”）中公式推导学习重点整理的原创学习笔记。

这套笔记的目标不是把算法写成公式清单，而是先把算法“讲明白”：它为什么出现、想解决什么问题、怎么一步步工作、公式中的每一项对应什么直觉。

当前版本已按“长讲义版”扩写：每章包含通俗讲解、类比、示例、公式推导、实践建议、常见误区、自测问答、扩展练习等模块。全部章节已加厚到长文讲义体量，核心章节接近万字级，后续仍可继续按专题追加案例和推导，扩展成严格单章万字版。

## 使用说明

每章尽量包含：

- 学习目标：本章应掌握什么问题、方法和判断标准。
- 先用人话理解：先解释算法到底在解决什么现实问题。
- 示例和类比：用生活类比、小数据场景帮助建立直觉。
- 知识框架：核心概念之间的关系。
- 关键公式：常见目标函数、估计式、优化式。
- 推导要点：用自己的语言展开重要公式的来龙去脉。
- 长讲义扩展：围绕算法直觉、推导、应用场景继续展开。
- 万字级补充：补充项目案例、实践 checklist、面试题和扩展练习。
- 易错点：容易混淆的概念、条件和假设。
- 练习建议：读完后应能完成的推导、证明或实现任务。

建议阅读顺序：

1. 先读“先用人话理解”和例子，确认知道算法为什么存在。
2. 再看算法流程，理解模型如何从输入得到输出。
3. 然后看公式，把公式中的每一项对应回直觉。
4. 最后看推导、易错点和练习，检查是否真正掌握。

不要一上来背公式。公式是压缩后的表达，只有先建立直觉，公式才容易记住，也更容易知道什么时候该用、什么时候不该用。

## 推荐学习顺序

第一阶段：建立机器学习全局框架

- 第 1 章：任务、假设空间、归纳偏好。
- 第 2 章：评估方法、误差估计、偏差方差。
- 第 3、4、7 章：掌握经典监督学习模型。

第二阶段：理解优化、间隔、组合思想

- 第 5 章：神经网络与反向传播。
- 第 6 章：SVM、核函数、对偶优化。
- 第 8 章：Boosting、Bagging、随机森林。

第三阶段：扩展到无监督、表示学习和高级主题

- 第 9、10、11 章：聚类、降维、特征选择和稀疏表达。
- 第 12 章：泛化边界与 PAC 学习的理论视角。
- 第 13、14、15、16 章：半监督、概率图、规则和强化学习。
