# 术语表

- 样本：一个具体实例，通常记为 `x`。
- 属性/特征：描述样本的变量或维度。
- 标签：监督学习中样本的目标输出。
- 训练集：用于拟合模型的数据。
- 验证集：用于模型选择和调参的数据。
- 测试集：用于最终估计泛化性能的数据。
- 假设空间：学习算法可选模型的集合。
- 归纳偏好：算法在多个一致假设中偏向选择某类假设的原则。
- 泛化能力：模型在未见样本上的表现。
- 过拟合：训练表现好但测试表现差。
- 欠拟合：训练和测试表现都不理想，模型表达能力或训练不足。
- 损失函数：衡量单个预测错误程度的函数。
- 风险：损失在数据分布上的期望。
- 经验风险：训练样本上的平均损失。
- 正则化：通过惩罚复杂模型降低过拟合风险。
- 似然：在参数给定下观测到当前数据的概率。
- 后验：观察数据后参数或类别的条件概率。
- 判别式模型：直接建模 `P(y|x)` 或决策边界。
- 生成式模型：建模联合分布 `P(x,y)` 或 `P(x|y)P(y)`。
- 凸优化：目标函数和可行域满足凸性，局部最优即全局最优。
- 核函数：隐式计算高维特征空间内积的函数。
- 支持向量：SVM 中决定间隔边界的关键样本。
- 熵：不确定性的度量。
- 信息增益：划分前后不确定性下降的程度。
- 集成学习：组合多个学习器提升性能。
- 聚类：无标签数据的自动分组。
- 降维：在保留主要信息的前提下降低特征维数。
- 稀疏性：参数或表示中多数分量为零。
- 马尔可夫性：未来只依赖当前状态而不依赖更早历史。
