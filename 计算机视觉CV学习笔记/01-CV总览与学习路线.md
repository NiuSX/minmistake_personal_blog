# 01-CV 总览与学习路线

## 1. 什么是计算机视觉

计算机视觉 Computer Vision，简称 CV，是让计算机从图像、视频、深度图、点云、多光谱图像等视觉数据中理解世界的技术。它的目标不是简单“看见图片”，而是从视觉数据中提取结构化信息，用于识别、检测、定位、测量、跟踪、分割、重建、理解和决策。

常见任务：

- 图像分类：判断图里是什么。
- 目标检测：找出目标位置和类别。
- 图像分割：把图像按像素划分区域。
- 实例分割：区分同类目标的不同实例。
- 关键点检测：检测人体、手、脸、物体关键点。
- OCR：检测和识别图像中的文字。
- 目标跟踪：在视频中持续追踪目标。
- 行为识别：识别视频中的动作或事件。
- 三维重建：从图像恢复三维结构。
- 缺陷检测：发现工业产品表面异常。
- 视觉测量：测量尺寸、角度、位置、面积。
- 机器人视觉：识别物体、定位、抓取、导航。

## 2. CV 学习的知识地图

完整 CV 学习可分为七层：

### 2.1 图像基础层

需要理解：

- 像素、分辨率、通道。
- RGB、BGR、HSV、灰度图。
- 图像坐标系。
- 图像直方图。
- 卷积和滤波。
- 频域和空域。
- 噪声和模糊。

### 2.2 OpenCV 工具层

需要掌握：

- 图像读取、显示、保存。
- 色彩空间转换。
- 缩放、裁剪、旋转、仿射变换。
- 阈值、边缘、轮廓、形态学。
- 模板匹配、霍夫变换。
- 相机标定、特征匹配。

### 2.3 传统算法层

需要掌握：

- 滤波降噪。
- 阈值分割。
- 边缘检测。
- 轮廓分析。
- 连通域分析。
- 形态学处理。
- 角点检测。
- 特征描述子。
- 图像配准。

传统算法优势是可解释、速度快、数据需求低；缺点是对光照、背景、姿态、噪声敏感。

### 2.4 机器学习层

需要理解：

- 特征工程。
- SVM、KNN、随机森林。
- 聚类。
- PCA 降维。
- HOG + SVM。
- Haar + AdaBoost。

这部分是传统 CV 和深度学习之间的重要过渡。

### 2.5 深度学习视觉层

需要掌握：

- CNN。
- 卷积、池化、归一化、激活函数。
- ResNet、MobileNet、EfficientNet。
- Vision Transformer。
- 迁移学习。
- 数据增强。
- 损失函数。
- 优化器。

### 2.6 任务模型层

需要掌握：

- 分类模型。
- 检测模型：YOLO、Faster R-CNN、SSD、RetinaNet。
- 分割模型：FCN、U-Net、DeepLab、Mask R-CNN、SAM。
- 关键点模型：OpenPose、HRNet、MediaPipe。
- OCR 模型：CRNN、DBNet、PaddleOCR。
- 跟踪模型：SORT、DeepSORT、ByteTrack。

### 2.7 工程落地层

需要掌握：

- 数据采集。
- 标注规范。
- 模型训练。
- 评估指标。
- 推理加速。
- ONNX、TensorRT、OpenVINO。
- 边缘部署。
- 摄像头标定。
- 线上监控。
- 误报漏报分析。

## 3. CV 常见应用场景

### 3.1 工业视觉

任务：

- 缺陷检测。
- 尺寸测量。
- OCR 读码。
- 定位引导。
- 装配检测。
- 外观检测。

特点：

- 场景可控。
- 光源和相机很重要。
- 精度和稳定性要求高。
- 误报漏报成本明确。
- 传统算法和深度学习经常混合使用。

### 3.2 自动驾驶和机器人

任务：

- 车道线检测。
- 车辆行人检测。
- 语义分割。
- 深度估计。
- SLAM。
- 目标跟踪。
- 视觉导航。

特点：

- 环境复杂。
- 实时性要求高。
- 需要多传感器融合。
- 安全性要求极高。

### 3.3 安防和视频分析

任务：

- 人脸识别。
- 行人检测。
- 越界检测。
- 人群计数。
- 行为识别。
- 目标跟踪。

特点：

- 摄像头多。
- 光照变化大。
- 遮挡多。
- 隐私和合规重要。

### 3.4 医学影像

任务：

- 病灶检测。
- 器官分割。
- 影像分类。
- 三维重建。

特点：

- 数据标注昂贵。
- 误判影响大。
- 可解释性重要。
- 需要医生参与评估。

## 4. 学习路线

### 阶段 1：图像和 OpenCV

目标：

- 能读写图像。
- 能进行预处理。
- 能用阈值、边缘、轮廓解决简单问题。

练习：

- 提取图片中的红色区域。
- 统计目标面积。
- 检测圆形零件。
- 识别二维码区域。

### 阶段 2：传统算法

目标：

- 理解滤波、边缘、轮廓、特征。
- 能用传统方法做定位和测量。

练习：

- 用 Canny 提取边缘。
- 用霍夫变换检测直线和圆。
- 用轮廓计算物体中心和角度。
- 用 ORB 做图像匹配。

### 阶段 3：深度学习

目标：

- 能训练分类和检测模型。
- 理解数据增强、损失函数、指标。

练习：

- 训练猫狗分类。
- 训练 YOLO 检测自定义目标。
- 训练 U-Net 做分割。

### 阶段 4：工程部署

目标：

- 模型能在真实环境稳定运行。
- 能分析误报漏报。
- 能优化推理速度。

练习：

- 摄像头实时检测。
- ONNX 导出。
- TensorRT 加速。
- 写一个可视化检测工具。

## 5. CV 项目基本流程

```text
需求分析 -> 图像采集 -> 数据标注 -> 数据清洗 -> 基线算法 -> 模型训练 -> 指标评估 -> 误差分析 -> 部署优化 -> 现场迭代
```

关键原则：

- 先明确业务指标，再选择算法。
- 数据质量优先于模型复杂度。
- 先做简单基线，再上复杂模型。
- 评估指标要和实际场景一致。
- 现场光照、相机、安装位置会决定很多算法成败。
