# 04-传统图像处理算法精讲

## 1. 传统图像处理的价值

深度学习很强，但传统图像处理仍然重要，尤其在以下场景：

- 目标形状规则。
- 光照和背景可控。
- 数据量少。
- 需要高可解释性。
- 需要精确测量。
- 算力有限。
- 工业视觉项目快速交付。

传统算法常用流程：

```text
读图 -> 预处理 -> 阈值/边缘 -> 形态学 -> 轮廓/连通域 -> 特征计算 -> 规则判断
```

## 2. 阈值分割

阈值分割根据像素灰度或颜色把目标和背景分开。

适用条件：

- 目标和背景差异明显。
- 光照稳定。
- 图像噪声较少。

常见方法：

- 固定阈值。
- Otsu。
- 自适应阈值。
- HSV 颜色阈值。

阈值分割优点：

- 简单。
- 快速。
- 可解释。

缺点：

- 对光照敏感。
- 对阴影、反光敏感。
- 对背景复杂场景效果差。

## 3. 边缘检测

边缘是图像灰度变化剧烈的位置。

### 3.1 Sobel

Sobel 计算水平和垂直方向梯度。

```python
gx = cv2.Sobel(gray, cv2.CV_64F, 1, 0, ksize=3)
gy = cv2.Sobel(gray, cv2.CV_64F, 0, 1, ksize=3)
```

适合检测方向边缘。

### 3.2 Laplacian

二阶导数边缘检测，对噪声敏感。

```python
edge = cv2.Laplacian(gray, cv2.CV_64F)
```

### 3.3 Canny

Canny 是经典边缘检测算法。

流程：

1. 高斯滤波去噪。
2. 计算梯度幅值和方向。
3. 非极大值抑制。
4. 双阈值筛选。
5. 边缘连接。

```python
edges = cv2.Canny(gray, 50, 150)
```

参数：

- 低阈值。
- 高阈值。

经验：

- 高阈值控制强边缘。
- 低阈值控制边缘连接。
- 阈值太低会有很多噪声。
- 阈值太高会断边。

## 4. 轮廓检测

轮廓是二值图中连续目标边界。

```python
contours, hierarchy = cv2.findContours(binary, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
```

常用特征：

- 面积。
- 周长。
- 外接矩形。
- 最小外接矩形。
- 最小外接圆。
- 质心。
- 方向角。
- 凸包。
- 轮廓近似。

示例：

```python
area = cv2.contourArea(cnt)
x, y, w, h = cv2.boundingRect(cnt)
M = cv2.moments(cnt)
cx = int(M["m10"] / M["m00"])
cy = int(M["m01"] / M["m00"])
```

应用：

- 计数。
- 定位。
- 尺寸测量。
- 形状判断。
- 缺陷区域提取。

## 5. 连通域分析

连通域分析用于找二值图中的独立区域。

```python
num_labels, labels, stats, centroids = cv2.connectedComponentsWithStats(binary)
```

stats 包含：

- x。
- y。
- width。
- height。
- area。

连通域适合做目标计数和小区域过滤。

## 6. 形态学算法

形态学基于结构元素处理二值图或灰度图。

### 6.1 腐蚀

腐蚀会缩小白色区域。

用途：

- 去除小白噪声。
- 分离细连接。
- 缩小目标。

### 6.2 膨胀

膨胀会扩大白色区域。

用途：

- 填补小缺口。
- 连接断裂区域。
- 扩大目标。

### 6.3 开运算

先腐蚀后膨胀。

用途：

- 去除孤立噪点。

### 6.4 闭运算

先膨胀后腐蚀。

用途：

- 填补小孔洞。
- 连接近邻区域。

## 7. 霍夫变换

### 7.1 直线检测

霍夫直线变换把图像空间的点映射到参数空间，寻找共线点。

```python
lines = cv2.HoughLinesP(edges, 1, np.pi / 180, threshold=80,
                        minLineLength=50, maxLineGap=10)
```

参数：

- threshold：投票阈值。
- minLineLength：最短直线长度。
- maxLineGap：允许断裂间隔。

应用：

- 车道线。
- 边框检测。
- 工件直线边缘。
- 表格线检测。

### 7.2 圆检测

```python
circles = cv2.HoughCircles(gray, cv2.HOUGH_GRADIENT, dp=1.2,
                           minDist=30, param1=100, param2=30,
                           minRadius=10, maxRadius=100)
```

应用：

- 圆孔。
- 螺丝。
- 瓶盖。
- 圆形零件。

## 8. 模板匹配

模板匹配在大图中寻找与模板相似的区域。

```python
result = cv2.matchTemplate(img, template, cv2.TM_CCOEFF_NORMED)
_, max_val, _, max_loc = cv2.minMaxLoc(result)
```

适用：

- 目标外观固定。
- 尺度和旋转变化小。
- 背景可控。

缺点：

- 对尺度变化敏感。
- 对旋转敏感。
- 对光照变化敏感。

改进：

- 多尺度模板匹配。
- 旋转模板。
- 边缘模板。
- 特征匹配。

## 9. 传统算法项目案例：检测圆形零件

流程：

```text
灰度化 -> 高斯滤波 -> Canny -> 霍夫圆 -> 圆心和半径输出 -> 规则过滤
```

需要考虑：

- 光照是否均匀。
- 圆边缘是否清晰。
- 是否存在反光。
- 是否有遮挡。
- 圆半径范围是否已知。

## 10. 传统算法失效原因

- 光照变化。
- 背景复杂。
- 目标姿态变化大。
- 目标外观不一致。
- 噪声严重。
- 反光和阴影。
- 参数对场景过拟合。

工程建议：

- 能控制硬件就先控制硬件。
- 固定光源、背景、相机位置。
- 使用 ROI 降低干扰。
- 给参数留调节入口。
- 用可视化工具显示每一步中间结果。
