# 08-图像分类、目标检测与实例分割

## 1. 图像分类

图像分类输出整张图的类别。

示例：

```text
输入：一张图片
输出：cat / dog / car
```

适用：

- 图片内容识别。
- 缺陷有无判断。
- 医学影像分类。
- 商品分类。

局限：

- 不给出目标位置。
- 多目标场景不适合只用分类。

## 2. 分类模型

常见：

- ResNet。
- MobileNet。
- EfficientNet。
- ConvNeXt。
- ViT。

评价指标：

- Accuracy。
- Precision。
- Recall。
- F1。
- Confusion Matrix。

类别不平衡时，不要只看 Accuracy。

## 3. 目标检测

目标检测输出：

- 类别。
- 边界框。
- 置信度。

```text
[class, score, x1, y1, x2, y2]
```

适用：

- 行人车辆检测。
- 工业缺陷定位。
- 商品检测。
- 安防目标检测。
- 机器人抓取定位。

## 4. 检测模型分类

### 4.1 Two-stage

代表：

- Faster R-CNN。
- Mask R-CNN。

特点：

- 先生成候选框。
- 再分类和回归。
- 精度高，速度相对慢。

### 4.2 One-stage

代表：

- YOLO。
- SSD。
- RetinaNet。

特点：

- 直接预测类别和框。
- 速度快。
- 工程落地常用。

## 5. YOLO 思想

YOLO 把检测看成一次前向预测问题。

核心：

- 输入固定尺寸图像。
- 网络输出多个尺度预测。
- 每个预测包含类别、置信度、边框。
- NMS 去除重复框。

YOLO 适合实时检测和工业部署。

## 6. Anchor 和 Anchor-Free

Anchor-based：

- 使用预设框。
- 模型预测相对偏移。

Anchor-free：

- 直接预测目标中心点或边界。
- 参数更少，设计更简洁。

## 7. IoU

IoU 是预测框和真实框的交并比。

```text
IoU = 交集面积 / 并集面积
```

用途：

- 判断预测是否命中。
- NMS。
- 损失函数。
- mAP 计算。

## 8. NMS

NMS 非极大值抑制用于删除重复检测框。

流程：

1. 按置信度排序。
2. 选择最高分框。
3. 删除与它 IoU 过高的框。
4. 重复。

问题：

- 密集目标可能误删。
- 可用 Soft-NMS 或 Weighted Boxes Fusion 改进。

## 9. mAP

mAP 是检测常用指标。

相关概念：

- TP：正确检测。
- FP：误检。
- FN：漏检。
- Precision：预测为正中有多少是真的。
- Recall：真实目标中有多少被找到。
- AP：单类别 PR 曲线面积。
- mAP：多类别 AP 平均。

注意：

- mAP 高不一定业务好。
- 工业场景可能更关注漏检率或误报率。

## 10. 小目标检测

小目标难点：

- 特征少。
- 缩放后容易消失。
- 定位误差影响大。

改进：

- 提高输入分辨率。
- 使用更浅层特征。
- 切图检测。
- 数据增强保留小目标。
- 调整 anchor 或检测头。

## 11. 实例分割

实例分割不仅检测目标框，还输出每个目标的像素级 mask。

代表：

- Mask R-CNN。
- YOLACT。
- SOLO。
- YOLO-seg。
- SAM。

应用：

- 精确面积计算。
- 缺陷区域提取。
- 医学病灶。
- 机器人抓取轮廓。

## 12. 检测项目流程

```text
定义类别 -> 采集数据 -> 标注框/mask -> 划分训练验证 -> 训练模型 -> 查看 PR 曲线 -> 误差分析 -> 调参 -> 导出部署
```

误差分析重点：

- 哪些类别混淆。
- 哪些场景漏检。
- 哪些背景误检。
- 小目标是否不足。
- 标注是否一致。
- 置信度阈值是否合理。
